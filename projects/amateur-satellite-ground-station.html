Amateur Satellite Ground Station

Build a full satellite ground station: track orbital positions from Two-Line Element (TLE) data with Python sgp4, auto-point a motorised Yagi antenna via two servos, receive APT weather imagery from NOAA 15/18/19 and AX.25 telemetry from amateur CubeSats using an RTL-SDR and GNU Radio on Raspberry Pi 4.

📖 Introduction

The NOAA weather satellites (NOAA-15, NOAA-18, NOAA-19) orbit at 850km altitude and continuously broadcast Automatic Picture Transmission (APT) images at 137 MHz — no decryption, free to receive. With an RTL-SDR USB dongle ($25), a V-dipole or Yagi antenna, and free software (GNU Radio + WxtoImg), you can receive actual satellite weather images from orbit. This advanced version adds: TLE-based orbital tracking with sgp4, two servo-driven auto-tracking antenna rotator, and AX.25 CubeSat telemetry decoding. A complete ground station for £60.

🧩 Components Required

  • 📡
    RTL-SDR V3 Dongle + 137MHz Antenna — RTL2832U + R820T2 chipset; 137MHz V-dipole (free DIY: 2× 53.4cm wires at 120°) or 3-element Yagi for better gain
  • 🔄
    2× Standard Servo (SG90 elevation + azimuth) — Azimuth: continuous rotation servo; elevation: standard 180° servo; 3D-printed rotator mount; pigpio PWM control
  • 🧠
    Raspberry Pi 4 — GNU Radio Companion; SoapySDR + rtlsdr driver; pyephem/sgp4 for orbital mechanics; pigpio for servo control
  • 🧲
    MPU-6050 IMU (Antenna Feedback) — Mount on Yagi to measure actual antenna elevation/azimuth; closed-loop corrections compensate for wind/settling

⚙️ How It Works

  1. 1

    TLE Orbital Prediction with sgp4

    Download current TLE from CelesTrak: curl -o noaa.tle "https://celestrak.org/SOCRATES/query.php?IDENT=NOAA-18". Parse and propagate: from sgp4.api import Satrec, jday; sat = Satrec.twoline2rv(line1, line2); e, r, v = sat.sgp4(jd, fr). Convert ECI→AzEl: subtract observer's ECEF position, rotate to local topocentric frame. Compute azimuth, elevation, range. Schedule next pass: find next time elevation > 5°, duration, maximum elevation. Print pass table for the next 24 hours.

  2. 2

    Auto-Tracking Antenna Rotator

    During pass: update antenna az/el every 1 second. Azimuth: continuous rotation servo — PD controller: duty = 7.5 + Kp*(az_error) + Kd*(az_error-prev_err); clamp 5–10% (1000–2000μs). Elevation: standard servo mapped 0–90°: pulse = 1000 + el/90*1000. IMU feedback for closed-loop: read MPU-6050 pitch (elevation), compass (azimuth) to verify pointing. Doppler correction: compute range rate, frequency offset: df = -137.912e6 * v_radial/c; apply to RTL-SDR centre frequency via SoapySDR API.

  3. 3

    NOAA APT Image Decoding

    Install WXtoImg or use Python apt-decoder. GNU Radio flowgraph: RTL-SDR source → narrow FM demod (bandwidth 34kHz) → audio sink at 11025 Hz. Record audio during pass as WAV. Decode APT: WXtoImg auto-processes WAV to produce false-colour infrared + visible channel image. Python alternative: pip install apt-decoder; apt-decoder.decode("pass.wav", "output.png"). Result: actual satellite weather image ~ 2000km wide strip of Earth.

  4. 4

    CubeSat AX.25 Telemetry Decoding

    Many amateur CubeSats (FUNcube, OSCAR series) transmit AX.25 packet telemetry at 145.825 MHz (ISS APRS) or satellite-specific frequencies. GNU Radio + gr-ax25 plugin demodulates AFSK1200 packets. Python: import kiss; frame = kiss.decode(raw_bytes); callsign = frame[:7]; data = frame[16:]. Submit decoded packets to amateur satellite networks (SatNOGS). Visualise: battery voltage, temperatures, orbital data from satellite's own sensors. Search satnogs.org for your satellite's beacon frequency.

💻 Core Code Snippet

from sgp4.api import Satrec, jday
from datetime import datetime, timezone
import math

def eci_to_azel(r_eci, obs_lat, obs_lon, obs_alt, dt):
    """Convert ECI position to topocentric Az/El."""
    lat, lon = math.radians(obs_lat), math.radians(obs_lon)
    jd, fr = jday(dt.year, dt.month, dt.day, dt.hour, dt.minute, dt.second)
    # GMST
    t = (jd - 2451545.0) / 36525.0
    gmst = math.radians((360.98564724 * (jd - 2451545.0)) % 360)
    # Observer ECEF
    R_E = 6378.137; h = obs_alt/1000
    N = R_E / math.sqrt(1 - 0.00669437999**2 * math.sin(lat)**2)
    ox = (N + h)*math.cos(lat)*math.cos(lon + gmst)
    oy = (N + h)*math.cos(lat)*math.sin(lon + gmst)
    oz = (N*(1-0.00669437999**2)+h)*math.sin(lat)
    # Range vector
    dx, dy, dz = r_eci[0]-ox, r_eci[1]-oy, r_eci[2]-oz
    rng = math.sqrt(dx**2+dy**2+dz**2)
    # SEZ transform
    S = math.sin(lat)*math.cos(lon+gmst)*dx + math.sin(lat)*math.sin(lon+gmst)*dy - math.cos(lat)*dz
    E = -math.sin(lon+gmst)*dx + math.cos(lon+gmst)*dy
    Z = math.cos(lat)*math.cos(lon+gmst)*dx + math.cos(lat)*math.sin(lon+gmst)*dy + math.sin(lat)*dz
    az = (math.degrees(math.atan2(-E, S)) + 360) % 360
    el = math.degrees(math.asin(Z/rng))
    return az, el, rng
