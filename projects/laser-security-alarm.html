Laser Security Alarm

Aim a laser module at an LDR — when someone breaks the beam an alarm triggers instantly. Learn optical sensing and interrupt-based alarming.

Home › Projects › Laser Security Alarm
KY-008
LDR

Beam breaks → LDR voltage drops → alarm fires

▶ Watch on YouTube

📖 Introduction

A laser tripwire alarm works on simple photoresistor physics: an LDR (Light Dependent Resistor) has very low resistance (~1 kΩ) when illuminated by the laser and very high resistance (~1 MΩ) in darkness. Pair it in a voltage divider with a fixed resistor and the Arduino ADC reads a high value when the beam is intact — the moment the beam breaks, the voltage drops sharply and you trigger the alarm.

At under $6 total, this is one of the most cost-effective security demonstrations you can build. It also teaches a universally useful principle: detecting a threshold change in an analog signal, which applies to all sensor-triggered systems.

⚙️ How It Works

  1. 1
    Build the voltage divider — Wire: 5 V → 10 kΩ → junction → LDR → GND. Connect the junction to A0. When laser hits LDR: A0 reads ~900–1000. When beam breaks: A0 reads ~50–200. Run the sketch once first to calibrate your actual threshold value.
  2. 2
    Set the threshold — Define const int THRESHOLD = 500;. In loop(): int val = analogRead(A0);. If val < THRESHOLD, the beam is broken.
  3. 3
    Trigger the alarm — On beam break, turn on the buzzer pin and red LED. Use an edge-detect boolean (wasArmed) to avoid repeated triggering. Latch the alarm on — require it to be manually reset with a button press.
  4. 4
    Align the laser — Mount KY-008 and LDR on opposite sides of a doorway, hallway, or box. For longer ranges (1–5 m) use a small lens or mount both in cardboard tubes to narrow the cone. Avoid direct sunlight on the LDR which can blind it.
