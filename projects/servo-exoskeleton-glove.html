Servo Exoskeleton Glove

Wear a flex-sensor glove on your right hand and a 3D-printed servo exoskeleton on your left hand. Five flex sensors read finger bend angles via Arduino Mega ADC. A PCA9685 16-channel PWM driver positions five MG996R servos accordingly — your exoskeleton hand mirrors every movement in real-time. Optional HC-05 Bluetooth wireless mode.

📖 Introduction

Robotic exoskeleton hands have applications in rehabilitation medicine, hazardous environment manipulation, and prosthetics research. This beginner-to-advanced transition project builds both the sensing glove and the actuated exoskeleton, teaching the full sensor-to-actuator pipeline. Flex sensors are simple variable resistors that change resistance proportional to bend angle — combined with a fixed resistor voltage divider, they produce an analogue voltage readable by any ADC. The PCA9685 I²C PWM driver offloads PWM generation for all 5 servos from the Arduino, freeing the microcontroller for higher-level logic like gesture recognition.

🧩 Components Required

  • 🌡️
    5× Flex Sensors (2.2" or 4.5") — Spectra Symbol FS-L-0095; 25kΩ flat, 125kΩ bent; 10kΩ pull-down resistor voltage divider per sensor; glued to knuckle joints of fabric glove
  • ⚙️
    5× MG996R Metal Gear Servos — 9.4 kg·cm torque; 3D-printed finger linkage brackets; Bowden-cable or direct link to exo finger segments; powered 6V 3A rail (NOT Arduino 5V pin)
  • 🔌
    PCA9685 16-Channel PWM Driver — I²C address 0x40; 12-bit 4096-step PWM; 50Hz servo frequency: pca.set_pwm_freq(50); controls all 5 servos with just 2 wires to Arduino
  • 💻
    Arduino Mega + HC-05 BT Module — 5× ADC inputs (A0–A4); I²C to PCA9685; optional HC-05 UART BT for wireless operation; 6V 3A BEC or regulated supply for servos

⚙️ How It Works

  1. 1

    Flex Sensor Calibration

    Each sensor needs calibration: record ADC value straight (OPEN) and fully bent (CLOSED). Store in EEPROM: int flex_min[5] = {340, 350, 330, 355, 345}; int flex_max[5] = {700, 720, 690, 710, 700}. On startup: press calibration button, hold hand open for 3s (auto-record raw values), then close fist for 3s (record bent values). Map: int angle = map(analogRead(A0), flex_min[0], flex_max[0], 0, 180); angle = constrain(angle, 0, 180). Recalibrate if environment temperature changes significantly (flex sensors are temperature-sensitive).

  2. 2

    PCA9685 PWM Setup

    Install: Arduino Adafruit_PWMServoDriver library. Init: Adafruit_PWMServoDriver pca = Adafruit_PWMServoDriver(0x40); pca.begin(); pca.setPWMFreq(50);. Calculate pulse for angle: int pulse = map(angle, 0, 180, SERVO_MIN, SERVO_MAX) where SERVO_MIN=150 (1.0ms) and SERVO_MAX=600 (2.0ms) in 4096-tick units at 50Hz. Set servo: pca.setPWM(channel, 0, pulse). Each of 5 channels controls one finger. Smooth interpolation: move 5° per 20ms loop towards target to prevent servo jerks (lowpass filtering on angle command).

  3. 3

    3D Printed Exoskeleton Design

    Design in Fusion 360 or Tinkercad. Key parts: finger bracket links (proximal+distal phalanx), palm mounting plate, servo horn connection brackets. Print in PETG (flexible-tough) at 30% infill. Finger actuation: servo → wire tendon → fingertip pulls finger closed. Return spring (elastic band) opens the finger. Thumb: two DOF — abduction by one servo (channel 4), flexion by another. STL files: search Thingiverse "servo exoskeleton hand" for remixable base designs. Scale to hand measurements using finger circumference as reference.

  4. 4

    Gesture Recognition Mode

    Log 5-dimensional flex readings across 8 gestures (fist, open, point, OK, peace, thumbs-up, gun, flat). 100 samples each. Train k-NN classifier on PC: from sklearn.neighbors import KNeighborsClassifier; knn = KNeighborsClassifier(k=5); knn.fit(X, y). Export to Arduino as lookup table or small decision tree. On Arduino: every 200ms, classify current flex values, output gesture label via Serial and actuate corresponding pre-programmed servo position set. Enables "play back" gesture sequences for teleoperation.

💻 Core Code Snippet

#include 
#include 

Adafruit_PWMServoDriver pca = Adafruit_PWMServoDriver(0x40);
#define SERVO_MIN 150   // ~1ms at 50Hz
#define SERVO_MAX 600   // ~2ms at 50Hz
const int flex_pins[5]  = {A0, A1, A2, A3, A4};
const int flex_min[5]   = {340, 350, 330, 355, 345};
const int flex_max[5]   = {700, 720, 690, 710, 700};
float servo_angles[5]   = {90,90,90,90,90};
float target_angles[5]  = {90,90,90,90,90};

void setServoAngle(uint8_t ch, float deg) {
  int pulse = map((int)deg, 0, 180, SERVO_MIN, SERVO_MAX);
  pca.setPWM(ch, 0, constrain(pulse, SERVO_MIN, SERVO_MAX));
}

void setup() { Wire.begin(); pca.begin(); pca.setPWMFreq(50); Serial.begin(9600); }

void loop() {
  for (int i = 0; i < 5; i++) {
    int raw = analogRead(flex_pins[i]);
    target_angles[i] = map(raw, flex_min[i], flex_max[i], 0, 180);
    target_angles[i] = constrain(target_angles[i], 0, 180);
    // Smooth interpolation
    servo_angles[i] += (target_angles[i] - servo_angles[i]) * 0.15f;
    setServoAngle(i, servo_angles[i]);
    Serial.printf("F%d:%3d  ", i, (int)servo_angles[i]);
  }
  Serial.println(); delay(20);  // 50Hz
}
