📖 Introduction

MEMS accelerometers like the MPU6050 can detect the subtle ground motion that precedes or accompanies an earthquake. While they are no substitute for professional seismic equipment, a well-calibrated Arduino-based detector can reliably detect local vibrations above ≈ M2 if mounted rigidly to a concrete floor or table and isolated from building-HVAC noise.

This project samples X/Y/Z acceleration at 100 Hz, maintains a baseline of "quiet" readings, and triggers when a sustained deviation exceeds tiered thresholds (minor > 0.05g, moderate > 0.15g, major > 0.5g). A simplified magnitude approximation (M ≈ log10(a / 0.001)) maps peak ground acceleration to a Richter-like value for display. All events are logged to a microSD card with timestamps from a DS3231 RTC module.

🧩 Components Required

  • 🎛️
    Arduino Uno / Nano — Controller for sensor reading and alert logic
  • 📐
    MPU6050 IMU (GY-521) — 6-axis IMU; I2C @ 0x68; ±2g / ±4g / ±8g / ±16g ranges
  • 🕐
    DS3231 RTC Module — Accurate timekeeping for event log timestamps
  • 💾
    MicroSD SPI Module + 4GB Card — Stores timestamped CSV event files
  • 📟
    16×2 LCD with I2C backpack — Live intensity and alert display
  • 🔔
    Passive Buzzer + 3× LEDs (G/Y/R) — Audible + visual alert tiers

⚙️ How It Works

  1. 1

    Calibrate Baseline

    On startup, sample 500 readings over 5 seconds and calculate average X/Y/Z acceleration values (ax0, ay0, az0). Subtract these offsets from every subsequent reading to get Δg values. This removes the constant 1g Earth gravity component and any sensor offset, leaving only dynamic motion.

  2. 2

    Calculate Peak Ground Acceleration

    Compute vector magnitude: a = sqrt(dAx² + dAy² + dAz²). Maintain a running max over a 0.5-second window. Convert raw ADC values using g = raw * 2.0 / 32768.0 at ±2g scale (most sensitive). Filter with a running average over 5 samples to reject single-sample noise spikes from footsteps.

  3. 3

    Tiered Alert System

    Minor (>0.05g): green LED blinks, M ≈ 2-3. Moderate (>0.15g): yellow LED + 2 buzzer beeps, M ≈ 3-4. Major (>0.5g): red LED + continuous siren pattern, M ≈ 4+. Display alert level on LCD. All tiers also log to SD. Alerts auto-clear after 10 seconds of quiet below threshold.

  4. 4

    SD Card Event Logging

    On each trigger, write a CSV row: timestamp, mag_estimate, peak_g_x, peak_g_y, peak_g_z, duration_ms, alert_level. Log file is named SEISLOG.CSV. On boot, append to existing file so history persists. Use the SD library's append mode; flush after each write to prevent corruption on power loss.

💻 Core Code Snippet

#include <MPU6050_light.h>
#include <SD.h>
#include <RTClib.h>
#include <LiquidCrystal_I2C.h>

MPU6050 mpu(Wire);
RTC_DS3231 rtc;
LiquidCrystal_I2C lcd(0x27, 16, 2);

const int CS_PIN = 10;
const float MINOR = 0.05, MODERATE = 0.15, MAJOR = 0.5; // g thresholds

void setup() {
  Wire.begin(); mpu.begin(); mpu.calcOffsets(true, true);
  rtc.begin(); lcd.init(); lcd.backlight();
  SD.begin(CS_PIN);
}

void loop() {
  mpu.update();
  float ax = mpu.getAccX(), ay = mpu.getAccY(), az = mpu.getAccZ() - 1.0;
  float pga = sqrt(ax*ax + ay*ay + az*az);  // Peak Ground Acceleration

  // Richter-like: M ≈ log10(pga / 0.001)
  float mag = (pga > 0.001) ? log10(pga / 0.001) : 0;

  lcd.setCursor(0, 0);
  lcd.print("PGA:" + String(pga, 3) + "g M:" + String(mag, 1) + "  ");

  if (pga > MAJOR) {
    tone(8, 880); digitalWrite(5, HIGH); // RED LED pin 5
    lcd.setCursor(0, 1); lcd.print("!! MAJOR QUAKE !! ");
    logEvent(pga, mag, "MAJOR");
  } else if (pga > MODERATE) {
    tone(8, 440); digitalWrite(4, HIGH); // YELLOW LED pin 4
    lcd.setCursor(0, 1); lcd.print("MODERATE event    ");
    logEvent(pga, mag, "MODERATE");
  } else if (pga > MINOR) {
    digitalWrite(3, HIGH); // GREEN LED pin 3
    lcd.setCursor(0, 1); lcd.print("Minor vibration   ");
  } else {
    noTone(8); digitalWrite(3,LOW); digitalWrite(4,LOW); digitalWrite(5,LOW);
    lcd.setCursor(0, 1); lcd.print("Quiet             ");
  }
  delay(10);
}

void logEvent(float pga, float mag, String level) {
  DateTime now = rtc.now();
  File f = SD.open("SEISLOG.CSV", FILE_WRITE);
  if (f) {
    f.println(String(now.unixtime()) + "," + String(pga,4) + "," + String(mag,2) + "," + level);
    f.close();
  }
}
