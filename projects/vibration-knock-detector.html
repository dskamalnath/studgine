Vibration Knock Detector

Mount a piezoelectric disk on a surface to detect knocks and taps, triggering LEDs or a buzzer in response. An accessible intro to piezoelectric sensing and analog debouncing.

📖 Introduction

A piezoelectric disk converts mechanical deformation (vibration, tapping, pressure) into a small electrical voltage — and vice versa. When someone knocks on the surface it's attached to, the disk flexes, generating a voltage spike detectable by Arduino's analog-to-digital converter.

The challenge is analog debouncing: a single knock produces a damped oscillating waveform (ring-down) that can trigger multiple "readings" as the piezo vibrates. We must implement a lockout window after the first detection — the same concept as button software debouncing but applied to analog signal peaks.

⚙️ How It Works

  1. 1
    Wire piezo to analog pin — Connect one lead of the piezo disk to A0 and the other to GND. Add a 1 MΩ bleed resistor between A0 and GND — this bleeds off stored charge slowly (time constant ~1 ms for typical piezo) preventing the ADC from seeing a prolonged voltage. Without it, a single tap might show as a square-wave plateau lasting 100+ ms.
  2. 2
    Read and threshold — Continuously read: int val = analogRead(A0);. A resting piezo reads ~0–5. A light knock produces 50–200; a hard knock 300–700+. Set a threshold: const int THRESHOLD = 60;. if (val > THRESHOLD) trigger knock actions and record lastKnockTime = millis();
  3. 3
    Analog debounce lockout — After detecting a knock, ignore all further readings for a lockout window: if (millis() - lastKnockTime < 200) return;. This 200 ms window absorbs the ring-down. Without it a single knock fires 5–50+ times depending on mounting surface and material stiffness.
  4. 4
    Knock pattern recognition — Track knock timing to recognise patterns: store the last 5 inter-knock intervals. Compare them to a stored pattern (e.g. "knock-knock-pause-knock knock" = 2, 3, 1 beat pattern for a door code). If times match within ±30% tolerance, trigger a special action — turning on a relay, unlocking a servo, or sending a Serial message.
