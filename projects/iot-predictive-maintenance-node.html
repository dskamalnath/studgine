IoT Predictive Maintenance Node

Mount an ADXL345 vibration accelerometer on a DC motor bearing, collect normal and fault-state vibration data, train a one-class SVM anomaly detector on Raspberry Pi, and publish real-time health scores to an MQTT + Grafana monitoring dashboard — a genuine Industry 4.0 prototype.

📖 Introduction

Predictive maintenance (PdM) uses continuous sensor monitoring to detect early signs of equipment failure — preventing costly unplanned downtime. Bearing defects produce characteristic vibration signatures: increased RMS amplitude, elevated kurtosis (impulsive spikes), and spectral components at bearing defect frequencies (BPFI, BPFO, BSF). This project trains a scikit-learn Isolation Forest (one-class novelty detector) on normal operating vibration to define a healthy baseline. Any deviation triggers a Grafana alert before failure occurs.

🧩 Components Required

  • 📳
    ADXL345 3-Axis MEMS Accelerometer — ±16g range; SPI/I²C; 3200Hz max ODR; FIFO buffer; interrupt on activity; I²C address 0x53
  • ⚙️
    DC Motor with Bearing + Mount — ADXL345 epoxy-mounted on bearing housing; connect motor to regulated 5–12V supply at constant RPM
  • 🧠
    Raspberry Pi 4 (2 GB) — I²C read at 400kHz; Python data pipeline; scikit-learn inference; Paho MQTT publisher
  • 📊
    MQTT + InfluxDB + Grafana Stack — Mosquitto broker; InfluxDB 2 time-series storage; Grafana alert rules + dashboard panels

⚙️ How It Works

  1. 1

    High-Rate Vibration Data Capture

    Configure ADXL345 for maximum output rate: write 0x0C to BW_RATE register (3200Hz). Read FIFO burst: i2c.read_i2c_block_data(0x53, 0x32, 6) → 6 bytes = X_low, X_high, Y_low, Y_high, Z_low, Z_high. Scale: g_value = (int16_t / 256) for ±2g range. Collect 1024 samples per measurement window (≈0.32s). Store in numpy array for feature extraction.

  2. 2

    Feature Engineering

    Time-domain features: RMS = sqrt(mean(x²+y²+z²)), peak = max(|a|), crest factor = peak/RMS, kurtosis = scipy.stats.kurtosis(samples) (healthy bearing ≈3, defective >5). Frequency-domain (FFT): compute freqs, psd = scipy.signal.welch(samples, fs=3200). Power in bearing defect frequency bands (BPFO ≈ 0.4×RPM×ball_count). Assemble feature vector of 12 values per window.

  3. 3

    Isolation Forest Training + Inference

    Collect 1000 windows of normal operation → assemble 1000×12 feature matrix. Train: from sklearn.ensemble import IsolationForest; model = IsolationForest(contamination=0.01).fit(X_normal). Save: joblib.dump(model, 'bearing_model.pkl'). Runtime: score = model.score_samples(features)[0]. Scores near 0 = normal; strongly negative = anomaly. Convert to health % = min(100, max(0, (score + 0.5)*200)).

  4. 4

    Alerting + Dashboard

    Publish JSON via MQTT every 5s: {"health":95.2,"rms_g":0.12,"kurtosis":3.2,"status":"NORMAL"}. InfluxDB Telegraf consumes → time-series storage. Grafana: health score gauge (red <70%, yellow 70–90%, green >90%), kurtosis trend chart, RMS alert when >2× baseline. Email/Telegram alert when health < 70% for 3 consecutive minutes → schedule maintenance before catastrophic failure.

💻 Core Code Snippet

import smbus2, numpy as np, scipy.stats, joblib, json
import paho.mqtt.client as mqtt

bus = smbus2.SMBus(1)
ADDR = 0x53
# Set 3200Hz data rate, measurement mode
bus.write_byte_data(ADDR, 0x2C, 0x0C)  # BW_RATE
bus.write_byte_data(ADDR, 0x31, 0x00)  # DATA_FORMAT ±2g
bus.write_byte_data(ADDR, 0x2D, 0x08)  # POWER_CTL measure

model = joblib.load('bearing_model.pkl')
client = mqtt.Client(); client.connect("localhost")

def read_accel():
    data = bus.read_i2c_block_data(ADDR, 0x32, 6)
    x = np.int16((data[1]<<8)|data[0])
    y = np.int16((data[3]<<8)|data[2])
    z = np.int16((data[5]<<8)|data[4])
    return np.array([x,y,z]) / 256.0  # convert to g

import time
while True:
    samples = np.array([read_accel() for _ in range(256)])
    mag = np.sqrt((samples**2).sum(axis=1))
    rms = float(np.sqrt(np.mean(mag**2)))
    kurt = float(scipy.stats.kurtosis(mag))
    peak = float(np.max(np.abs(mag)))
    
    features = [[rms, peak, peak/max(rms,0.001), kurt,
                 0,0,0,0,0,0,0,0]]
    score = model.score_samples(features)[0]
    health = min(100, max(0, (score+0.5)*200))
    
    payload = json.dumps({"health":round(health,1),
                          "rms_g":round(rms,3),
                          "kurtosis":round(kurt,2),
                          "status":"NORMAL" if health>70 else "ALERT"})
    client.publish("factory/bearing/node1", payload)
    time.sleep(5)
