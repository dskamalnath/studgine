📖 Introduction

YOLO (You Only Look Once) processes the entire image in a single neural network pass, making it extremely fast. The YOLOv8n (nano) variant has only 3.2M parameters — small enough to run meaningfully on Raspberry Pi 5 (with ARM NEON optimisations) or at full speed on a Google Coral USB Accelerator (4 TOPS of INT8 inference).

This project covers the complete edge deployment pipeline: download pretrained COCO weights → export to TFLite INT8 → benchmark on Pi → overlay bounding boxes on live camera feed → trigger GPIO outputs based on detections (e.g., buzzer when person detected).

🧩 Components Required

  • 🧠
    Raspberry Pi 5 (4GB) or Pi 4 — Pi 5: ~12 FPS YOLOv8n at 320×320. Pi 4: ~4-6 FPS. Pi 5 has 2× CPU speed; use 64-bit OS for best performance
  • ⚡
    Google Coral USB Accelerator (optional) — 4 TOPS INT8 inference; bumps detection to 25+ FPS; requires Edge TPU TFLite model export
  • 📷
    Pi Camera v3 or USB Webcam — Capture at 640×480; resize to 320×320 for inference; full 1920×1080 only if Coral accelerator used

⚙️ How It Works

  1. 1

    Export YOLOv8n to TFLite

    On PC: pip install ultralytics. Export: from ultralytics import YOLO; model = YOLO('yolov8n.pt'); model.export(format='tflite', int8=True, imgsz=320). This produces yolov8n_int8.tflite (~3MB). Copy to Pi. For Coral: export with format='edgetpu' (requires Google Edge TPU compiler installed).

  2. 2

    TFLite Inference on Pi

    Load model: interpreter = tflite.Interpreter("yolov8n_int8.tflite", num_threads=4). Preprocess: resize to 320×320, normalise to [0,1], add batch dim. Run: interpreter.set_tensor(in_idx, img); interpreter.invoke(); out = interpreter.get_tensor(out_idx). YOLOv8 output shape: (1, 84, 2100) — 84 = 4 box coords + 80 class scores, 2100 candidate boxes. Apply NMS (Non-Max Suppression) with confidence threshold 0.45, IoU threshold 0.5. Use ultralytics.utils.ops.non_max_suppression() or implement manually.

  3. 3

    Draw Detections + Actions

    For each detection: cv2.rectangle(frame, (x1,y1), (x2,y2), color, 2) and cv2.putText(frame, f"{class_name} {conf:.2f}", ...). Map class IDs to COCO names (80-class list). Trigger actions: loop through detections, if class_id==0 (person) and confidence>0.7: GPIO.output(buzzer_pin, HIGH). Calculate FPS: fps = 1/(time.time()-t_prev). Display on HDMI with cv2.imshow() or stream via Flask MJPEG.

  4. 4

    Performance Optimisation

    Skip frames: run inference every 2nd or 3rd frame; show previous boxes on skipped frames. Reduce input size: 256×256 instead of 320×320 gives 40% speedup with small accuracy loss. Multi-threading: capture thread + inference thread + display thread run independently via Queue. For Pi 5: compile OpenCV with NEON + VFPV3 flags for 30% speedup. Monitor CPU temperature: thermal throttling at 80°C; add heatsink/fan.

💻 Core Code Snippet

import cv2, numpy as np, time
import tflite_runtime.interpreter as tflite

COCO_NAMES = ["person","bicycle","car","motorcycle","airplane","bus","train"]  # ... 80 total
CONF_THRESH = 0.45; IOU_THRESH = 0.5

interpreter = tflite.Interpreter("yolov8n_int8.tflite", num_threads=4)
interpreter.allocate_tensors()
in_idx = interpreter.get_input_details()[0]['index']
out_idx = interpreter.get_output_details()[0]['index']

def preprocess(frame, size=320):
    img = cv2.resize(frame, (size, size))
    img = cv2.cvtColor(img, cv2.COLOR_BGR2RGB)
    return (img.astype(np.float32)/255.0)[np.newaxis]

def infer(frame):
    img = preprocess(frame)
    interpreter.set_tensor(in_idx, img)
    interpreter.invoke()
    return interpreter.get_tensor(out_idx)[0]  # shape (84, 2100)

cap = cv2.VideoCapture(0); t0 = time.time()
while True:
    ret, frame = cap.read()
    if not ret: break
    output = infer(frame)
    # output[0:4] = cx,cy,w,h (normalised); output[4:84] = class scores
    scores = output[4:].max(axis=0); mask = scores > CONF_THRESH
    boxes = output[:4, mask].T; class_ids = output[4:, mask].argmax(axis=0)
    confs = scores[mask]
    # Scale boxes to frame
    h, w = frame.shape[:2]
    for (cx,cy,bw,bh), cls, conf in zip(boxes, class_ids, confs):
        x1,y1 = int((cx-bw/2)*w), int((cy-bh/2)*h)
        x2,y2 = int((cx+bw/2)*w), int((cy+bh/2)*h)
        cv2.rectangle(frame,(x1,y1),(x2,y2),(0,255,0),2)
        cv2.putText(frame,f"{COCO_NAMES[cls]} {conf:.2f}",(x1,y1-5),cv2.FONT_HERSHEY_SIMPLEX,0.4,(0,255,0),1)
    fps = 1/(time.time()-t0); t0=time.time()
    cv2.putText(frame,f"FPS:{fps:.1f}",(5,20),cv2.FONT_HERSHEY_SIMPLEX,0.5,(255,255,0),1)
    cv2.imshow("YOLO",frame)
    if cv2.waitKey(1)&0xFF==ord('q'): break
