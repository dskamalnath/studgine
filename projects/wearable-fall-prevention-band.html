Wearable Fall Prevention Band

A 3D-printed wristband with Arduino Nano BLE Sense, MPU-6050 IMU, and small haptic motor. Samples wrist acceleration at 200Hz, extracts real-time features (CoM acceleration variance, jerk), and runs a TFLite Micro classifier that detects pre-fall postural instability 200ms before a loss-of-balance event — vibrating a warning before the fall.

📖 Introduction

Falls are the leading cause of injury death in adults over 65. A preventive wearable that detects the characteristic instability pattern 200ms before a fall — enough time for a muscle reflex-triggering haptic cue — could prevent millions of injuries. The key insight: falls are preceded by a characteristic increase in body sway (centre of mass jerk) before the actual rapid acceleration of falling. This project trains a 1D CNN on labelled accelerometer windows (stable walk/stand, pre-fall sway, actual fall), quantises it to TFLite Micro, and runs on the Arduino Nano BLE Sense — 36kB flash, 256kB RAM.

🧩 Components Required

  • 💻
    Arduino Nano BLE Sense 33 — nRF52840 Cortex-M4F; built-in LSM9DS1 IMU; 256kB RAM; BLE for data logging; TFLite Micro support
  • 📳
    Vibration Motor (ERM or LRA) + DRV2605L Driver — LRA haptic actuator 3V; DRV2605L I²C driver with 123 waveform library; distinct pre-warning buzz pattern
  • 🔋
    LiPo 100mAh + MCP73831 Charger — 3.7V 100mAh flat cell; micro-USB charging; ~8hr runtime; housed in 3D-printed PETG wristband enclosure
  • 🧩
    Training Hardware (PC + Dataset) — Collect data via BLE serial logger; label segments; train Keras 1D CNN; quantise to TFLite INT8; convert to C array

⚙️ How It Works

  1. 1

    Data Collection + Feature Engineering

    Wear device during normal activities (walking, sitting, standing) and simulated near-falls (controlled leans, catching balance). Log 200Hz accel data via BLE Serial to Python. Label windows: STABLE (0), PRE_FALL (1), FALL (2). Feature extraction per 50-sample (250ms) window: jerk = np.diff(accel_magnitude) * 200; svm_var = np.var(np.sqrt(ax**2+ay**2+az**2)); max_jerk = np.max(np.abs(jerk)). Dataset: ~5000 labelled windows. Split 80/10/10.

  2. 2

    1D CNN Model Training

    Input: (50, 3) float32 window (ax, ay, az). Architecture: Conv1D(32, 5, relu) → MaxPool1D(2) → Conv1D(64, 3, relu) → MaxPool1D(2) → Flatten → Dense(32, relu) → Dense(3, softmax). Train 100 epochs, Adam lr=0.001, batch=32. Test accuracy ~92%. Quantise to INT8: converter = tf.lite.TFLiteConverter.from_keras_model(model); converter.optimizations=[tf.lite.Optimize.DEFAULT]; tflite_model = converter.convert(). Convert to C array: xxd -i model.tflite > model_data.cc for Arduino.

  3. 3

    TFLite Micro Inference on Arduino

    Arduino sketch: include TensorFlowLite_ESP32 or Arduino_TensorFlowLite library. Load model: tflite::MicroInterpreter interpreter(model_data, resolver, tensor_arena, TENSOR_ARENA_SIZE, &error_reporter). Sliding window buffer: collect 50 accel samples at 200Hz, then infer. Copy to input tensor, invoke, read output probabilities. If output[1] > 0.8 (pre-fall risk), activate haptic: drv.setWaveform(0, 14); drv.setWaveform(1, 0); drv.go(). Inference time: ~4ms on M4F at 64MHz.

  4. 4

    BLE Logging + Caregiver Alert

    BLE service: transmit fall risk score + event log to smartphone app. Use ArduinoBLE library: BLECharacteristic riskChar("UUID-HERE", BLERead|BLENotify, 4); riskChar.writeFloat(risk_score). Phone app (MIT App Inventor or React Native): display real-time risk score, vibrate phone on fall detected. Send SMS alert to caregiver via Twilio API when actual fall confirmed (rapid deceleration after high acceleration). Battery indicator: voltage divider → ADC → BLE battery service advertise charge level.

💻 Core Code Snippet

// Arduino Nano BLE — TFLite Micro sliding-window fall classifier
#include 
#include "model_data.h"
#include 

namespace {
  tflite::MicroErrorReporter error_reporter;
  const tflite::Model* model;
  tflite::MicroInterpreter* interpreter;
  TfLiteTensor* input; TfLiteTensor* output;
  constexpr int ARENA_SIZE = 30*1024;
  uint8_t tensor_arena[ARENA_SIZE];
  float window[50][3];  int win_idx = 0;
}

void setup() {
  IMU.begin();
  model = tflite::GetModel(model_data);
  static tflite::MicroMutableOpResolver<5> resolver;
  resolver.AddConv2D(); resolver.AddPooling2D();
  resolver.AddFullyConnected(); resolver.AddSoftmax(); resolver.AddReshape();
  static tflite::MicroInterpreter static_interp(model, resolver, tensor_arena, ARENA_SIZE, &error_reporter);
  interpreter = &static_interp; interpreter->AllocateTensors();
  input = interpreter->input(0); output = interpreter->output(0);
}

void loop() {
  float ax, ay, az; IMU.readAcceleration(ax, ay, az);
  window[win_idx][0] = ax; window[win_idx][1] = ay; window[win_idx][2] = az;
  if (++win_idx >= 50) {
    win_idx = 0;
    memcpy(input->data.f, window, 50*3*sizeof(float));
    interpreter->Invoke();
    float pre_fall_prob = output->data.f[1];
    if (pre_fall_prob > 0.80) {
      digitalWrite(LED_BUILTIN, HIGH);  // Trigger haptic motor
    }
  }
  delayMicroseconds(5000);  // 200Hz
}
