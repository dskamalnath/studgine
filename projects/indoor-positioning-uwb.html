Indoor Positioning System (UWB)

Four DWM1000 Ultra-Wideband ranging modules placed at room corners act as anchors. A mobile tag module performs Two-Way Ranging (TWR) with each anchor; a Raspberry Pi aggregates the four distances and trilateration solves the tag's X,Y position to ±10cm accuracy — enabling real-time indoor asset tracking on a live floor-plan map.

📖 Introduction

GPS does not work indoors. Ultra-Wideband (UWB) radio measures Time of Flight (ToF) of short RF pulses with nanosecond precision — translating to centimetre-level range measurements. The DecaWave DWM1000 module provides ±10cm ranging accuracy over 10m, costs ~$15 each, and communicates via SPI. This project places 4 anchors at known positions (room corners), and a mobile tag ranges to all four simultaneously. The Least-Squares trilateration algorithm solves the over-determined system of 4 range equations to find the tag's X,Y position. Applications: asset tracking, robot localisation, and elderly/patient location in hospitals.

🧩 Components Required

  • 📡

  • 🔌
    5× Arduino Pro Mini 3.3V — One per DWM1000 node; runs DecaWave TWR timing firmware; anchors report ranges to Pi via UART; tag mobile
  • 🧠
    Raspberry Pi 4 (Network Hub) — Receives range data from 4 anchors via USB-UART adaptors; runs trilateration solver; Flask web map display
  • 🗺️
    Floor-Plan Map (image file) — Scaled PNG of room; overlaid with tag position dot; matplotlib or Flask/Leaflet.js interactive display

⚙️ How It Works

  1. 1

    Two-Way Ranging Protocol

    Each anchor-tag pair performs DecaWave DS-TWR (Double-Sided Two-Way Ranging) to cancel clock drift. Anchor initiates: sends POLL, records T_TX. Tag receives, timestamps T_RX, replies RESPONSE, records T_TX2. Anchor receives, timestamps T_RX2, sends FINAL with all timestamps. Tag computes distance: Tround1 = T_RX2 - T_TX; Treply2 = T_TX2 - T_RX; d = c*(Tround1 - Treply2)/2. Achieves ±10cm @ 10Hz. Anchors serial-print range CSV: "A1:2.81\n".

  2. 2

    Trilateration Algorithm

    Known anchor positions: A1=(0,0), A2=(W,0), A3=(W,L), A4=(0,L) (room corners). Given ranges r1,r2,r3,r4. Set up least-squares: d_i² = (x-xi)²+(y-yi)². Linearise: subtract equation 1: 2(x1-x_i)*x + 2(y1-y_i)*y = r_i²-r1²-xi²-yi². Build A matrix [3×2], b vector [3×1]. Solve: pos = np.linalg.lstsq(A, b)[0]. Output: [x, y] in metres. Kalman filter for smoothing: process noise σ=0.1m, measurement noise σ=0.15m.

  3. 3

    Flask Web Map Display

    Flask server on Pi returns index.html with Leaflet.js map using the floor-plan PNG as tile layer. Position endpoint: @app.route('/position'); return jsonify({'x':tx,'y':ty}). JavaScript polls every 100ms and updates marker position. Draw range circles from each anchor (radius = range). Visualise uncertainty ellipse (from Kalman covariance). Display position history trail (last 50 positions as dots). Optional: store position history to PostgreSQL + display playback timeline.

  4. 4

    Multi-Tag + Zone Detection

    Add more tags (each with unique ID). TDMA slot assignment: anchor polls tags round-robin to avoid radio collisions. Pi tracks multiple tags simultaneously, each as distinct marker. Define zones: if 1.0 < tx < 3.0 and 0.5 < ty < 2.5: zone='Meeting Room'. Zone entry/exit events logged. Applications: hospital patient tracking, museum visitor analytics, warehouse forklift safety zones (alarm if near pedestrian <1m).

💻 Core Code Snippet

import numpy as np, serial

# Anchor positions in metres (room 6m × 5m)
ANCHORS = np.array([[0.0,0.0],[6.0,0.0],[6.0,5.0],[0.0,5.0]])

def trilaterate(ranges):
    """Least-squares trilateration from 4 anchors."""
    A = []
    b = []
    x0,y0 = ANCHORS[0]; r0 = ranges[0]
    for i in range(1, 4):
        xi,yi = ANCHORS[i]; ri = ranges[i]
        A.append([2*(x0-xi), 2*(y0-yi)])
        b.append(r0**2 - ri**2 - x0**2 - y0**2 + xi**2 + yi**2)
    A, b = np.array(A), np.array(b)
    pos, _, _, _ = np.linalg.lstsq(A, b, rcond=None)
    return pos  # [x, y] in metres

# Read from 4 anchor serial streams
ports = [serial.Serial(f'/dev/ttyUSB{i}', 115200) for i in range(4)]

while True:
    ranges = []
    for i, port in enumerate(ports):
        line = port.readline().decode().strip()
        d = float(line.split(':')[1])  # "A1:2.81"
        ranges.append(d)
    pos = trilaterate(ranges)
    print(f"Tag position: X={pos[0]:.2f}m, Y={pos[1]:.2f}m")
