📖 Introduction

An electrocardiogram (ECG) measures the electrical potential difference between body-surface electrodes as the heart's sinoatrial node fires a depolarisation wave through the myocardium. The characteristic PQRST waveform — small P wave (atrial depolarisation), tall QRS complex (ventricular depolarisation), and T wave (ventricular repolarisation) — carries enormous diagnostic information.

The AD8232 is a dedicated cardiac monitoring front-end IC from Analog Devices. It combines an instrumentation amplifier, bandpass filter (0.5 Hz – 40 Hz), right-leg drive circuit, and leads-off detection — all in one 4×4 mm package. It amplifies the typical 1 mV ECG signal to the 0–3.3 V input range of a microcontroller ADC, enabling a complete single-lead ECG analyser with just two components.

🧩 Components Required

  • ❤️
    AD8232 ECG Sensor Module — 3.5 mm jack or pin headers; OUTPUT → Arduino A0; LO+/LO− → digital pins for leads-off detect
  • 🧠
    Arduino Uno / Nano — ADC at 10-bit; use TimerOne at 500 Hz for accurate sampling; USB serial to PC
  • 🩹
    Disposable ECG Electrodes (×10) — Ag/AgCl gel electrodes; 3-lead standard (RA right wrist, LA left wrist, RL right ankle/leg)
  • 🔋
    9V Battery (isolated) — CRITICAL: never use USB power while electrodes are on body; use battery or isolation transformer
  • 🖥️
    PC Running Processing / Python — Real-time graph with PySerial + matplotlib or Arduino Serial Plotter (built-in)

⚙️ How It Works

  1. 1

    AD8232 Signal Chain

    The AD8232 instrumentation amplifier measures the differential voltage between the RA (+IN) and LA (−IN) electrodes with CMRR >80 dB, rejecting common-mode 50/60 Hz mains interference. The right-leg drive (RLD) circuit feeds back a processed common-mode signal through the RL electrode, further cancelling interference. The internal bandpass filter removes baseline wander (<0.5 Hz due to breathing movement) and high-frequency muscle artefacts (>40 Hz). Output pin voltage = 1.65 V (midpoint) ± ECG amplitude, suitable for 3.3 V ADC. The Arduino maps 0–1023 ADC values to a signed millivolts value for plotting.

  2. 2

    500 Hz Timer-Interrupt Sampling

    Use TimerOne library to fire an ISR every 2 ms (500 Hz). In the ISR: check LO+ and LO− pins — if either is HIGH, the corresponding electrode is detached; send a "LEADS OFF" marker instead of ADC data. Otherwise, read analogRead(A0) and store in a circular sample buffer. In the main loop, send buffered samples over Serial at 115200 baud as ASCII integers. Each line is one sample: 516\n. The Serial Plotter reads this automatically and scrolls the waveform in real time.

  3. 3

    Python / Processing Visualisation

    For a richer display, write a Python script using pyserial + matplotlib.animation. Open serial port at 115200 baud, read integer lines, append to a deque of 500 samples (1 second of data), redraw with FuncAnimation every 20 ms. Add a threshold line at 700 ADC (R-peak detection) and compute BPM from R-peak intervals using scipy.signal.find_peaks. Alternatively, use Processing 3.x with the Serial library for a compact graphical sketch.

  4. 4

    Safety Considerations (Critical)

    1. Electrical isolation: Power the Arduino ONLY from a battery (9V) when electrodes are attached to the body — never USB while wired to mains-powered PC. If needed, use a USB isolator (e.g. ADUM3160). 2. Electrode placement: Clean skin with alcohol swab, ensure gel contact. 3. Scope of use: This is an educational prototype only — not a medical device. Do not use for clinical diagnosis. Do not apply to the chest near the heart (limb Lead I only for safety). Add a 1 MΩ resistor in series with each electrode lead as a current-limiting safety measure.

💻 Core Code Snippet

#include <TimerOne.h>

const int ECG_OUT = A0;
const int LO_PLUS = 10, LO_MINUS = 11;
volatile bool newSample = false;
volatile int ecgValue;

void sampleISR() {
  if(digitalRead(LO_PLUS)==HIGH || digitalRead(LO_MINUS)==HIGH) {
    ecgValue = -1; // leads-off marker
  } else {
    ecgValue = analogRead(ECG_OUT);
  }
  newSample = true;
}

void setup() {
  Serial.begin(115200);
  pinMode(LO_PLUS, INPUT); pinMode(LO_MINUS, INPUT);
  Timer1.initialize(2000); // 2ms = 500 Hz
  Timer1.attachInterrupt(sampleISR);
}

void loop() {
  if(newSample) {
    newSample = false;
    if(ecgValue == -1) Serial.println("!"); // leads off
    else Serial.println(ecgValue);
  }
}
