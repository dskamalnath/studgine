RT Language Translator Device

A fully offline real-time language translator: speak English, hear Spanish (or any of 200 languages). Whisper tiny speech-to-text, Meta's NLLB-200-distilled translation model, and Piper neural TTS all run on a Raspberry Pi 4 — no cloud API needed. Complete audio pipeline with hardware push-to-talk button and 2.8" touchscreen language selector.

📖 Introduction

Commercial real-time translators (Pocketalk, ili) cost $200–$300 and require internet. This project builds a comparable device for $55 in hardware. The AI pipeline: (1) Whisper tiny (39M parameters, 77MB) transcribes speech to text in ~1.5s on Pi 4. (2) NLLB-200-distilled (600M parameters) translates between 200 languages in ~1.5s. (3) Piper TTS speaks the translation through a small speaker in ~0.5s. Total latency ~3.5s — fast enough for conversational use. Pushed-to-talk button; 2.8" capacitive touchscreen for language pair selection; runs from a USB power bank.

🧩 Components Required

  • 🎤
    USB Microphone (Samson Go or ReSpeaker 2-Mic) — Cardioid pattern; 44.1kHz; close-range speech capture; ReSpeaker HAT optional for better SNR and LED feedback ring
  • 🔊
    Speaker + MAX98357A I²S Amp — 3W 8Ω speaker; MAX98357A I²S amplifier HAT; or USB audio dongle + 3.5mm speaker; 16kHz TTS output
  • 🖥️
    2.8" SPI Touchscreen (ILI9341) — 320×240 capacitive; language pair selector UI; Pygame on Pi; show source/target text live during translation
  • 🔘
    Push-to-Talk Button + Raspberry Pi 4 — Momentary button GPIO; press-record-release-translate pattern; optional LED ring for recording/processing/speaking states

⚙️ How It Works

  1. 1

    Speech-to-Text with Whisper

    Install: pip install openai-whisper; pip install faster-whisper. Use faster-whisper (CTranslate2 optimised): from faster_whisper import WhisperModel; model = WhisperModel("tiny", device="cpu", compute_type="int8"). Record audio on button press: import sounddevice as sd; audio = sd.rec(int(5*44100), samplerate=44100, channels=1); sd.wait(). Transcribe: segments, _ = model.transcribe(audio.flatten(), language="en"). Whisper tiny: ~1.5s for 5s speech on Pi 4 with INT8 quantisation.

  2. 2

    Neural Machine Translation with NLLB

    Meta NLLB-200 (No Language Left Behind) translates 200 language pairs with state-of-the-art accuracy. Install: pip install transformers; from transformers import AutoTokenizer, AutoModelForSeq2SeqLM. Load distilled 600M model: model = AutoModelForSeq2SeqLM.from_pretrained("facebook/nllb-200-distilled-600M"). Translate: inputs = tokenizer(text, return_tensors="pt", src_lang="eng_Latn"); translated = model.generate(**inputs, forced_bos_token_id=tokenizer.lang_code_to_id["spa_Latn"]); result = tokenizer.batch_decode(translated, skip_special_tokens=True)[0].

  3. 3

    Text-to-Speech with Piper TTS

    Piper TTS is a fast, high-quality offline neural TTS: pip install piper-tts. Download voice model (Spanish es_MX, ~56MB): piper --model es_MX-claude-high --download-dir /models. Synthesise: import subprocess; subprocess.run(["piper", "--model", "/models/es_MX-claude-high.onnx", "--output_file", "/tmp/speech.wav"], input=translated_text.encode(), check=True). Play: import pygame; pygame.mixer.music.load("/tmp/speech.wav"); pygame.mixer.music.play(). Piper is ~300ms for typical sentence on Pi 4.

  4. 4

    Touchscreen UI + Language Selector

    Pygame UI on ILI9341 320×240 display. Language grid: display 8 target language buttons. On selection, update NLLB target language code and Piper voice model. State machine: IDLE (show flag icons) → RECORDING (red LED + waveform) → PROCESSING (spinner) → SPEAKING (text scroll + green LED). Show source + translated text on screen. Save translations to log file for session review. Optional: QR code display of translated text for reading by the other party.

💻 Core Code Snippet

import sounddevice as sd, numpy as np, subprocess
from faster_whisper import WhisperModel
from transformers import AutoTokenizer, AutoModelForSeq2SeqLM

# Load models once at startup (~30s on Pi 4)
stt = WhisperModel("tiny", device="cpu", compute_type="int8")
nllb_name = "facebook/nllb-200-distilled-600M"
tokenizer = AutoTokenizer.from_pretrained(nllb_name)
nmt = AutoModelForSeq2SeqLM.from_pretrained(nllb_name)

LANG_PAIRS = {"Spanish": "spa_Latn", "French": "fra_Latn", "Arabic": "arb_Arab"}
target_lang = "spa_Latn"

def translate_speech(duration=5, src_lang="eng_Latn"):
    # 1. Record
    print("Recording..."); audio = sd.rec(int(duration*16000), samplerate=16000, channels=1, dtype='float32'); sd.wait()
    # 2. STT
    segments, _ = stt.transcribe(audio.flatten(), language="en")
    text = " ".join(s.text for s in segments).strip()
    print(f"STT: {text}")
    # 3. NMT
    inputs = tokenizer(text, return_tensors="pt", src_lang=src_lang)
    out = nmt.generate(**inputs, forced_bos_token_id=tokenizer.lang_code_to_id[target_lang], max_new_tokens=256)
    translated = tokenizer.batch_decode(out, skip_special_tokens=True)[0]
    print(f"Translation: {translated}")
    # 4. TTS
    subprocess.run(["piper","--model","models/es_MX.onnx","--output_file","/tmp/out.wav"], input=translated.encode())
    subprocess.run(["aplay","/tmp/out.wav"])

translate_speech()
