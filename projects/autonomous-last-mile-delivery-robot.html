Autonomous Last-Mile Delivery Robot

A wheeled delivery robot that navigates GPS waypoints, stops for pedestrians detected by YOLOv8n, scans QR codes with pyzbar to verify delivery address, opens a servo compartment to release the parcel, and publishes a delivery confirmation via MQTT. Runs entirely on Raspberry Pi 4 with a front-facing Pi Camera.

📖 Introduction

Last-mile delivery robots like Starship Technologies' units are becoming common on university campuses. This educational replica combines three advanced subsystems: (1) GPS waypoint navigation with haversine distance + bearing PD controller, reusing the lawnmower navigation stack. (2) Computer vision pedestrian detection — YOLOv8n runs at 4–6fps on Pi 4; if a "person" class is detected within 1.5m depth estimate, the robot stops. (3) QR code delivery verification — the recipient shows a QR code; pyzbar reads the delivery address; if it matches the order, the servo compartment opens. A complete autonomous logistics platform in one project.

🧩 Components Required

  • 📍
    U-blox NEO-M8N GPS Module — 3.5m CEP accuracy; 10Hz update rate; UART → PySerial → pynmea2 parsing; onboard patch antenna; L80 or BN-880 alternatives
  • 📷
    Raspberry Pi Camera v2 (wide-angle) — 160° FOV lens mod preferred; YOLOv8n pedestrian detection; pyzbar QR scanning; both running on same frame pipeline
  • 🤖
    Wheeled Chassis + L298N Motor Driver — 4WD platform or differential drive 2WD; 12V DC motors; L298N H-bridge; encoder feedback optional (odometry for short-range dead-reckoning)
  • 📦
    Servo Compartment Lock — MG996R or DS3115 metal gear servo; 3D-printed hatch latch; SG90 alternative for lightweight build; pulls locking rod to open lid on delivery verify

⚙️ How It Works

  1. 1

    GPS Waypoint Navigation

    Load delivery waypoints as (lat, lon) list from JSON. Parse GPS NMEA: import serial, pynmea2; msg = pynmea2.parse(line); lat, lon = msg.latitude, msg.longitude. Haversine distance to next waypoint: d = 2*R*arcsin(sqrt(sin²(Δlat/2)+cos(lat1)*cos(lat2)*sin²(Δlon/2))). Bearing: θ = atan2(sin(Δlon)*cos(lat2), cos(lat1)*sin(lat2)-sin(lat1)*cos(lat2)*cos(Δlon)). Heading error (IMU compass vs bearing target) → PD motor command: left_speed = base - Kp*err - Kd*derr; right_speed = base + Kp*err + Kd*derr. Waypoint reached at distance <1m → advance to next.

  2. 2

    YOLOv8n Pedestrian Detection

    Install: pip install ultralytics. Load nano model (6MB): from ultralytics import YOLO; model = YOLO("yolov8n.pt"). Run on every 3rd frame (balance fps vs latency): results = model(frame, classes=[0], conf=0.5). Class 0 = "person". Estimate distance from bounding box height: detected person bounding box height h_px at known distance d_ref: d_est = d_ref * h_ref_px / h_px. If any person closer than 1.5m → pause motors, wait for clear path. Beep warning tone. Resume after 3 seconds of no detection.

  3. 3

    QR Code Delivery Verification

    On arrival at delivery waypoint: stop motors, activate "DELIVERY MODE". Announce via speaker: "Please show QR code". Continuously scan camera frame with pyzbar: from pyzbar.pyzbar import decode; codes = decode(frame). Parse decoded data: delivery_data = json.loads(codes[0].data.decode()). Compare: if delivery_data['order_id'] == current_order: verified = True. On match: open compartment servo: servo.ChangeDutyCycle(7.5) (90° open). Wait 10s, close: servo.ChangeDutyCycle(5.0). Log mismatch attempts (security).

  4. 4

    MQTT Delivery Confirmation

    On successful delivery: publish to broker: import paho.mqtt.client as mqtt; client.publish("delivery/confirmed", json.dumps({"order_id": oid, "lat": lat, "lon": lon, "timestamp": time.time()})). Also publish telemetry every 5s: current position, battery voltage, speed, obstacle status, next waypoint. Receive return-to-base command: subscribe to "robot/command" topic. Process: "RETURN" → load depot GPS as final waypoint. "STOP" → emergency halt. Web dashboard: Grafana + InfluxDB with MQTT bridge shows live robot position on a map.

💻 Core Code Snippet

import serial, pynmea2, math, time
from ultralytics import YOLO
from pyzbar.pyzbar import decode
from picamera2 import Picamera2
import RPi.GPIO as GPIO, json

# Setup
model = YOLO("yolov8n.pt"); cam = Picamera2(); cam.start()
gps_ser = serial.Serial('/dev/ttyAMA0', 9600)
WAYPOINTS = [{"lat": 51.5074, "lon": -0.1278, "order_id": "DEL001"}]

def haversine(lat1, lon1, lat2, lon2):
    R = 6371000
    φ = math.radians(lat2-lat1); λ = math.radians(lon2-lon1)
    a = math.sin(φ/2)**2 + math.cos(math.radians(lat1))*math.cos(math.radians(lat2))*math.sin(λ/2)**2
    return 2*R*math.asin(math.sqrt(a))

wp_idx = 0
while wp_idx < len(WAYPOINTS):
    wp = WAYPOINTS[wp_idx]
    # Parse GPS
    line = gps_ser.readline().decode(errors='ignore')
    if 'GPGGA' in line:
        try:
            msg = pynmea2.parse(line)
            dist = haversine(msg.latitude, msg.longitude, wp['lat'], wp['lon'])
            print(f"Distance to waypoint: {dist:.1f}m")
            if dist < 1.0:
                # At waypoint — verify QR
                frame = cam.capture_array()
                codes = decode(frame)
                if codes and json.loads(codes[0].data)['order_id'] == wp['order_id']:
                    GPIO.output(SERVO_PIN, GPIO.HIGH)  # Open compartment
                    print(f"Delivered order {wp['order_id']} ✓")
                    wp_idx += 1
        except: pass
    # Check pedestrians
    frame = cam.capture_array()
    results = model(frame, classes=[0], conf=0.5, verbose=False)
    if results[0].boxes and len(results[0].boxes) > 0:
