📖 Introduction

The MAX30100 integrates a red LED, infrared LED, and two photodetectors in a single 5.6 mm package. It measures photoplethysmography (PPG) — the variation in light absorption caused by blood volume changes in each heartbeat. By comparing red vs. IR absorption, it calculates SpO₂ (peripheral oxygen saturation) using the Beer-Lambert law.

This project wires the MAX30100 to an Arduino Nano via I2C, processes the PPG signal using a beat-detection algorithm, and displays BPM and SpO₂ on a compact SSD1306 OLED. The whole assembly fits on a wristwatch-size PCB powered by a 3.7 V Li-Po, teaching you biosignal ADC, I2C peripherals, and embedded signal filtering.

⚙️ How It Works

  1. 1

    I2C Initialisation

    The MAX30100 uses I2C address 0x57. Call pox.begin() — if it fails, check SDA/SCL pull-ups (4.7 kΩ to 3.3 V; the module may need the 1 kΩ SMD resistors DNP'd depending on version). Set LED current with pox.setIRLedCurrent(MAX30100_LED_CURR_50MA) for stable readings through skin.

  2. 2

    Continuous PPG Sampling

    Call pox.update() as frequently as possible inside loop() — this feeds the internal FIFO. The MAX30100 samples at up to 100 Hz. Attach a beat callback with pox.setOnBeatDetectedCallback(onBeat) — it fires once per detected R-wave, giving you instantaneous BPM via inter-beat intervals.

  3. 3

    Read BPM and SpO₂

    Every 500 ms, call pox.getHeartRate() and pox.getSpO2() for rolling-average values. Discard readings where BPM <40 or >200 — these indicate no finger contact. SpO₂ values below 94% should display a warning (clinical hypoxemia threshold).

  4. 4

    Render on OLED

    Use U8g2 or Adafruit_SSD1306 to draw a large BPM number, an SpO₂ value, and a scrolling PPG waveform. Animate a small heart icon on beat callbacks for visual feedback. If 10 s passes with no beat, display "No Finger" with the sensor powered down to save battery.

💻 Core Code Snippet

#include <Wire.h>
#include <MAX30100_PulseOximeter.h>
#include <Adafruit_SSD1306.h>

PulseOximeter pox;
Adafruit_SSD1306 oled(128, 64, &Wire, -1);
uint32_t lastReport = 0;
bool beatFlag = false;

void onBeatDetected() { beatFlag = true; }

void setup() {
  oled.begin(SSD1306_SWITCHCAPVCC, 0x3C);
  oled.clearDisplay(); oled.display();
  if (!pox.begin()) {
    oled.print("MAX30100 FAIL"); oled.display();
    while(1);
  }
  pox.setIRLedCurrent(MAX30100_LED_CURR_50MA);
  pox.setOnBeatDetectedCallback(onBeatDetected);
}

void loop() {
  pox.update();
  if (millis() - lastReport > 500) {
    lastReport = millis();
    float bpm  = pox.getHeartRate();
    float spo2 = pox.getSpO2();
    oled.clearDisplay();
    oled.setTextSize(2); oled.setCursor(0,0);
    if (bpm < 40 || bpm > 200) {
      oled.print("No Finger");
    } else {
      oled.print(String((int)bpm) + " BPM");
      oled.setTextSize(1); oled.setCursor(0,32);
      oled.print("SpO2: ");
      oled.print(String((int)spo2) + "%");
      if (spo2 < 94 && spo2 > 0) {
        oled.setCursor(0,48); oled.print("!! LOW OXYGEN !!");
      }
    }
    oled.display();
    beatFlag = false;
  }
}
