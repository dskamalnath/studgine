📖 Introduction

The Automated Greenhouse Controller is a complete closed-loop IoT project. Four sensors continuously feed data to an ESP32, which makes autonomous decisions to maintain perfect growing conditions: fans spin when temperature exceeds limits, grow lights activate at dusk, and water pumps run when soil dries — all without human intervention.

You'll learn multi-sensor data fusion, relay logic safety design, rule-based automation, and IoT cloud monitoring. The skills directly map to industrial building-management and SCADA systems.

🧩 Components Required

  • 🎛️
    ESP32 DevKit V1 — Main MCU with Wi-Fi
  • 🌡️
    DHT22 Temperature / Humidity Sensor — ±0.5°C accuracy
  • 🌱
    Capacitive Soil Moisture Sensor ×2 — Analog output, corrosion-resistant
  • ☀️
    BH1750 Light Sensor — I2C lux measurement
  • 🔌
    4-Channel Relay Module — Fan, grow light, pump, heater
  • 💨
    12 V Fan + LED Grow Light Strip — Controlled via relays
  • 💧
    5 V Mini Water Pump + Tubing — Drip irrigation
  • 🖥️
    0.96″ OLED (optional) — Local status display

⚙️ How It Works

  1. 1

    Sensor Polling

    Poll the DHT22 every 2 s (non-blocking), soil sensors every 5 s, and BH1750 lux every 10 s. Store rolling averages (window = 5 readings) to avoid reacting to transient spikes.

  2. 2

    Rule-Based Actuator Control

    Fan ON if temp > 30°C; fan OFF if temp < 26°C. Pump ON if soil < 30% and moisture was stable for 60 s. Grow light ON if lux < 200 and hour is between 6:00–22:00 from NTP clock.

  3. 3

    Safety Guards

    Implement per-actuator maximum run times (pump max 60 s, fan max 10 min). Log actuator toggle events to SPIFFS with timestamps. Alert via Telegram if any sensor is stuck (unchanged for 5 min).

  4. 4

    Cloud Dashboard

    Push all sensor values to ThingSpeak fields every 30 s using HTTPClient. Use ThingSpeak React and ThingTweet to send email alerts when temperature exceeds critical thresholds.

💻 Core Code Snippet

#include <DHT.h>
#include <BH1750.h>
#include <Wire.h>
#include <WiFi.h>
#include <HTTPClient.h>

DHT dht(4, DHT22);
BH1750 lightMeter;

#define SOIL_PIN   A0
#define RELAY_FAN  26
#define RELAY_PUMP 27
#define RELAY_LIGHT 14

const float TEMP_HIGH  = 30.0;
const float TEMP_LOW   = 26.0;
const int   SOIL_DRY   = 30;   // % moisture threshold
const float LUX_LOW    = 200.0;

void loop() {
  float temp   = dht.readTemperature();
  float hum    = dht.readHumidity();
  float lux    = lightMeter.readLightLevel();
  int   soilPct = map(analogRead(SOIL_PIN), 2800, 1200, 0, 100);

  // Fan control
  if (temp > TEMP_HIGH) digitalWrite(RELAY_FAN, LOW);   // ON
  if (temp < TEMP_LOW)  digitalWrite(RELAY_FAN, HIGH);  // OFF

  // Pump control
  static unsigned long pumpStart = 0;
  if (soilPct < SOIL_DRY && digitalRead(RELAY_PUMP) == HIGH) {
    digitalWrite(RELAY_PUMP, LOW);
    pumpStart = millis();
  }
  if (millis() - pumpStart > 60000) digitalWrite(RELAY_PUMP, HIGH); // safety off

  // Grow light
  if (lux < LUX_LOW) digitalWrite(RELAY_LIGHT, LOW);
  else                digitalWrite(RELAY_LIGHT, HIGH);

  delay(2000);
}
