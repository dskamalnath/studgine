📖 Introduction

Remotely Operated Vehicles (ROVs) for underwater exploration require waterproof electronics, thrust-vectored propulsion, and a reliable control link. This project uses Schedule 80 PVC pipe fittings waterproofed with O-rings and silicone to house a Raspberry Pi Zero 2W. Three T200 brushless thrusters (or bilge pump motors) provide forward/backward, lateral, and vertical movement. A 100m Ethernet CAT5e tether carries power from a surface 12V supply and full gigabit control/video bandwidth — far more reliable than RF underwater.

🧩 Components Required

  • 💧
    Waterproof Enclosure (PVC / Acrylic) — Schedule 40 PVC tube 100mm diameter; acrylic end caps; O-ring grooves; cable penetrators (Blue Robotics FXTI)
  • 🌀
    3× Brushless Thrusters — Blue Robotics T200 or DIY bilge pump conversions; 2× horizontal (fwd/lateral), 1× vertical; 30A ESC each
  • 📷
    Raspberry Pi Zero 2W + Pi Camera v3 — Picamera2 MJPEG stream on port 8000; wide-angle 120° lens; 30fps @ 1080p via H.264 encoding
  • 🔗
    100m CAT5e Ethernet Tether — Power Over Ethernet (PoE) injector 12V 5A; converts 48V PoE to 12V via DC-DC on ROV; gigabit bandwidth
  • 🕹️
    Xbox Controller + Surface Laptop — pygame joystick on surface PC; sends thruster commands to ROV over TCP socket; displays MJPEG stream

⚙️ How It Works

  1. 1

    Waterproofing Design

    All cable penetrations use Blue Robotics M10 cable penetrators or DIY epoxy-potted PG7 glands. O-ring grooves machined into acrylic end caps (shore 70A O-ring, 2× backup). Pressure test to 3 bar (30m equivalent) with bicycle pump before water entry. Electronics mounted on foam inside tube: RPi Zero 2W, 3× ESC, 5V BEC, MPU-6050 IMU, MS5837 depth sensor (I²C, 300m rated). Desiccant packet absorbs condensation.

  2. 2

    TCP Thruster Control Socket

    ROV runs Python TCP server on port 5000: receives JSON commands from surface. {"fwd":0.7, "lat":0.0, "vrt":0.3}. Maps to ESC PWM: fwd_pw = 1500 + int(cmd['fwd']*400) (1100–1900µs). Drives ESCs via pigpio servo pulsewidth. Surface client: read Xbox joystick axes via pygame, build command JSON, send every 50ms. Watchdog: ROV stops all thrusters if no command received for 500ms (tether loss safety stop).

  3. 3

    Video Streaming

    Picamera2 encodes H.264 on Pi Zero 2W hardware encoder: encoder = H264Encoder(bitrate=2000000). Stream via RTSP or MJPEG over HTTP. Surface: display with VLC (RTSP) or OpenCV MJPEG: cap = cv2.VideoCapture('http://rov.local:8000/stream.mjpg'). Overlay telemetry (depth, pitch, temp) onto video frame using cv2.putText before display. Record directly to MP4 on surface laptop.

  4. 4

    Depth + IMU Telemetry

    MS5837 I²C depth/pressure sensor: sensor.read(); depth_m = sensor.depth(). MPU-6050: compute pitch/roll from accelerometer + gyro complementary filter for orientation display on surface HUD. Telemetry JSON sent back from ROV via second TCP port every 200ms: depth, temp, pitch, roll, battery voltage. Surface HUD rendered as overlay on video feed with artificial horizon indicator.

💻 Core Code Snippet

# ROV onboard server (runs on Raspberry Pi Zero 2W)
import socket, json, pigpio, threading, time

pi = pigpio.pi()
ESC_FWD, ESC_LAT, ESC_VRT = 18, 19, 20  # GPIO pins

def arm_escs():
    for pin in [ESC_FWD, ESC_LAT, ESC_VRT]:
        pi.set_servo_pulsewidth(pin, 1500)  # neutral
    time.sleep(3)

def set_thrust(fwd=0, lat=0, vrt=0):
    for pin, val in [(ESC_FWD, fwd), (ESC_LAT, lat), (ESC_VRT, vrt)]:
        pw = max(1100, min(1900, 1500 + int(val*400)))
        pi.set_servo_pulsewidth(pin, pw)

arm_escs()
last_cmd = time.time()

def watchdog():
    while True:
        if time.time() - last_cmd > 0.5:
            set_thrust(0, 0, 0)  # safety stop
        time.sleep(0.1)
threading.Thread(target=watchdog, daemon=True).start()

server = socket.socket(); server.bind(('0.0.0.0', 5000)); server.listen(1)
conn, _ = server.accept()
while True:
    data = conn.recv(1024).decode()
    if data:
        cmd = json.loads(data)
        set_thrust(cmd.get('fwd',0), cmd.get('lat',0), cmd.get('vrt',0))
        last_cmd = time.time()
