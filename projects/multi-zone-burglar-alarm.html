📖 Introduction

Real-world alarm systems use the concept of zones — independently monitored detector circuits with different behaviours. An entry zone (front door) triggers a countdown entry delay giving the homeowner time to disarm, while an instant zone (window) trips the siren immediately.

This project implements 4 zones using HC-SR501 PIR sensors and normally-closed magnetic door contacts wired in series per zone (professional standard). A 4×4 matrix keypad on a 20×4 LCD lets you arm/disarm with a PIN code, configure bypass, and review event history. A SIM800L GSM module adds SMS alerts so you're notified wherever you are.

  • 🚪

  • 📟

  • ⌨️

⚙️ How It Works

  1. 1

    Zone Wiring (Normally-Closed Loop)

    Wire each zone as a normally-closed loop: pull-up resistor → door contact → PIR tamper output → Arduino pin. When the circuit is broken (door opens or wire cut), the pin goes HIGH → zone violated. This fail-safe design triggers the alarm if tampered with.

  2. 2

    System States: Armed / Disarmed / Entry Delay / Alarm

    Implement a finite state machine. In ARMED state, any zone violation triggers ENTRY_DELAY (30 s countdown displayed on LCD) for entry zones, or ALARM immediately for instant zones. Incorrect PIN or timeout → ALARM state (siren on, SMS sent).

  3. 3

    Keypad PIN Authentication

    Use the Keypad library to read 4-digit PIN entries. Store the hash of the PIN in EEPROM (not plain text). On correct PIN in ALARM/ARMED state → disarm. Wrong PIN three times → lockout + "Tamper Alert" SMS. Separate engineer PIN for configuration mode.

  4. 4

    GSM SMS on Alarm

    When alarm triggers, use the SIM800L AT command set: AT+CMGF=1 (text mode), then AT+CMGS="+441234567890" followed by the message text and Ctrl+Z to send. Include zone name, time, and "ALARM TRIGGERED" in the SMS body.

💻 Core Code Snippet

#include <Keypad.h>
#include <LiquidCrystal_I2C.h>
#include <SoftwareSerial.h>

// Zone pins (normally-closed → HIGH when violated)
const int ZONE_PINS[4] = {2, 3, 4, 5};
const bool ZONE_INSTANT[4] = {false, false, true, true};  // Z3,Z4 = instant
const String ZONE_NAMES[4] = {"Front Door","Living Room","Kitchen","Back Door"};
const int SIREN_PIN = 6;
int entryDelay = 30;  // seconds
String correctPin = "1234";

enum AlarmState { DISARMED, ARMED, ENTRY_DELAY, ALARM };
AlarmState state = DISARMED;

void triggerAlarm(int zoneNum) {
  state = ALARM;
  digitalWrite(SIREN_PIN, HIGH);
  sendSMS("!! ALARM: " + ZONE_NAMES[zoneNum] + " triggered!");
  lcd.clear(); lcd.print("ALARM! Z" + String(zoneNum+1));
}

void checkZones() {
  for (int z = 0; z < 4; z++) {
    if (digitalRead(ZONE_PINS[z]) == HIGH) {  // zone violated
      if (ZONE_INSTANT[z] || state == ENTRY_DELAY) triggerAlarm(z);
      else if (state == ARMED) { state = ENTRY_DELAY; startCountdown(z); }
    }
  }
}

void sendSMS(String msg) {
  gsm.println("AT+CMGF=1");   delay(200);
  gsm.print("AT+CMGS=\"+441234567890\"\r"); delay(200);
  gsm.print(msg);
  gsm.write(26);  // Ctrl+Z = send
  delay(3000);
}

void loop() {
  if (state == ARMED || state == ENTRY_DELAY) checkZones();
  // ... keypad reading and PIN verification
}
