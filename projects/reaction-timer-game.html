📖 Introduction

The average human reaction time is 200–300 ms. This project builds a simple game to measure yours. A green LED lights up after a random delay (2–5 seconds); the moment it turns on you press a button. The Arduino measures the precise interval between the LED turning on and the button press using millis(), then displays the result on a 16×2 LCD.

It's a fun project because it's immediately playable — you can compete with friends. It also teaches a core Arduino skill: non-blocking timing with millis() instead of delay(), which is essential for any multi-task sketch.

⚙️ How It Works

  1. 1
    Randomise the wait — At the start of each round, seed the RNG: randomSeed(analogRead(A0));, then calculate: waitTime = random(2000, 5000);. Record roundStart = millis(). Display "WAIT…" on the LCD.
  2. 2
    Check for early press (cheating) — While waiting, poll the button. If pressed before the LED turns on, display "TOO EARLY!" and restart the round after 2 seconds. This prevents holding the button down.
  3. 3
    Flash the GO signal — Once millis() - roundStart >= waitTime, turn on the LED, record ledOn = millis(), and display "GO!". Now wait for the button press in a tight loop.
  4. 4
    Calculate and display — On button press: reactionTime = millis() - ledOn;. Turn off LED. Display something like "Time: 247 ms". If it is a new best, save to a variable and show "NEW BEST!" Pause 3 s then loop.
