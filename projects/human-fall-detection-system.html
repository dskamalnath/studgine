Human Fall Detection System

A MoveNet Lightning pose-estimation model running on Raspberry Pi analyses live camera frames, detects sudden falls from keypoint velocity and body orientation changes, and instantly dispatches an SMS alert via the Twilio API — a practical elderly care safety device.

📖 Introduction

Falls are the leading cause of injury in people over 65. This project creates a camera-based fall detector running entirely on-device (no cloud required for inference). TensorFlow Lite MoveNet Lightning — a single-person pose estimator that runs at ~30fps on Pi 4 — outputs 17 body keypoints (nose, ears, shoulders, hips, knees, ankles). The fall detection algorithm analyses hip height, body aspect ratio, and keypoint vertical velocity to classify events as STANDING, SITTING, or FALLEN.

On FALLEN detection sustained for 2+ seconds, a Twilio SMS is sent to a carer's phone, including the timestamp and optionally a snapshot JPEG.

🧩 Components Required

  • 📷
    Raspberry Pi Camera Module v2 — 30fps @ 640×480; picamera2 library; camera mounted at 1.5–2m height for full-body view
  • 🧠
    Raspberry Pi 4 (4 GB) — TFLite MoveNet Lightning INT8; ~30fps inference; no TPU needed; connected to home WiFi for Twilio
  • 📱
    Twilio SMS API Account — Free tier: 1 SMS phone number + 15 credits; twilio Python library; sends alert with timestamp to carer number
  • 🔊
    Buzzer + LED Alert (optional) — Audible + visual local alert on GPIO in addition to SMS; 5V buzzer + red LED on GPIO 17

⚙️ How It Works

  1. 1

    MoveNet Pose Inference

    Load INT8 TFLite model: interpreter = tflite.Interpreter('movenet_lightning_int8.tflite'). Capture 192×192 frame from picamera2, preprocess: input_image = cv2.resize(frame, (192,192))[np.newaxis,...].astype(np.int8). Run: interpreter.invoke(). Extract output: keypoints = interpreter.get_tensor(output_details[0]['index'])[0,0] → shape [17,3] (y,x,confidence). 17 keypoints: 0=nose, 5/6=shoulders, 11/12=hips, 13/14=knees, 15/16=ankles.

  2. 2

    Fall Detection Features

    Extract key metrics each frame: hip_y = (kp[11][0] + kp[12][0]) / 2 (normalised 0=top, 1=bottom of frame). shoulder_y = (kp[5][0] + kp[6][0]) / 2. Body height ratio: ratio = (shoulder_y - hip_y) / max((hip_y - ankle_y), 0.01). In standing: ratio ≈ 1.8; in fallen: ratio ≈ 0.3 (horizontal). Hip vertical velocity: vel_y = abs(hip_y - prev_hip_y). Fast downward velocity + low ratio = fall event.

  3. 3

    Fall State Machine

    Three states: STANDING (ratio>1.2), TRANSITIONING (<0.8 for <2s), FALLEN (<0.8 for >2s sustained). Use a counter: if ratio < 0.8 and vel_y > 0.1: fall_counter += 1; else fall_counter = 0. Alert when fall_counter > 30 (≈2s at 15fps). One-shot alert (don't re-alert until person detected standing again). False positive reduction: require BOTH low height ratio AND high hip velocity in event window.

  4. 4

    Twilio SMS Alert

    from twilio.rest import Client. Credentials from environment variables (never hardcode). On fall confirmed: save snapshot JPEG (cv2.imwrite('fall_snapshot.jpg', frame)). Send SMS: client.messages.create(body=f"FALL DETECTED at {timestamp} — location: Living Room", from_='+15551234567', to=carer_number). Limit to 1 alert per 5 minutes to prevent spam. Log all events to JSON file with timestamp + confidence.

💻 Core Code Snippet

import numpy as np, cv2, time
import tflite_runtime.interpreter as tflite
from picamera2 import Picamera2
from twilio.rest import Client
import os

interp = tflite.Interpreter('movenet_lightning_int8.tflite')
interp.allocate_tensors()
in_d = interp.get_input_details(); out_d = interp.get_output_details()
cam = Picamera2(); cam.start()

twilio = Client(os.environ['TWILIO_SID'], os.environ['TWILIO_TOKEN'])
fall_counter = 0; last_alert = 0; prev_hip_y = 0.5

def get_keypoints(frame):
    img = cv2.resize(frame, (192,192))[np.newaxis,...].astype(np.int8)
    interp.set_tensor(in_d[0]['index'], img)
    interp.invoke()
    return interp.get_tensor(out_d[0]['index'])[0,0]  # [17,3]

while True:
    frame = cam.capture_array()
    kp = get_keypoints(frame)
    hip_y  = (kp[11][0] + kp[12][0]) / 2
    shl_y  = (kp[5][0]  + kp[6][0])  / 2
    ank_y  = (kp[15][0] + kp[16][0]) / 2
    ratio  = abs(shl_y - hip_y) / max(abs(hip_y - ank_y), 0.01)
    vel_y  = abs(hip_y - prev_hip_y)
    prev_hip_y = hip_y

    if ratio < 0.7 and vel_y > 0.05:
        fall_counter += 1
    else:
        fall_counter = max(0, fall_counter - 2)

    if fall_counter > 20 and time.time() - last_alert > 300:
        twilio.messages.create(
            body=f"⚠️ FALL DETECTED @ {time.strftime('%H:%M:%S')}",
            from_='+15551234567', to=os.environ['CARER_PHONE'])
        last_alert = time.time(); fall_counter = 0
    time.sleep(0.067)  # ~15fps
