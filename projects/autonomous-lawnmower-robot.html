Autonomous Lawnmower Robot

A GPS-guided Raspberry Pi robot navigates a pre-mapped lawn boundary using RTK-GNSS waypoints, 5× ultrasonic collision avoidance sensors, a rain detector for auto-dock return, and a brushless cutting motor — a full autonomous outdoor robotics engineering challenge with real utility.

📖 Introduction

Robotic lawnmowers represent a £2B+ market dominated by Husqvarna Automower and Bosch Indego. This DIY version demonstrates all the core engineering: GPS waypoint navigation, obstacle avoidance, boundary enforcement (virtual perimeter wire or GPS polygon), and intelligent behaviours (low battery → dock, rain → dock). Built on a differential-drive robot chassis with two 100W brushed DC motors and a brushless cutting deck, controlled by a Raspberry Pi 4 with GPS module, 5× HC-SR04 range sensors, IMU compass, and a rain sensor. Coverage algorithm: boustrophedon (back-and-forth stripes).

🧩 Components Required

  • 🛞
    Differential Drive Chassis + 2× 100W DC Motors — Steel outdoor chassis; rubber wheels; L298N H-bridge or BTS7960 motor driver; 12V 18Ah SLA battery
  • 📡
    U-blox NEO-M8N GPS Module — 2.5m CEP accuracy; 10Hz update rate; NMEA UART; optional RTK for 2cm accuracy with base station
  • 🧠
    Raspberry Pi 4 + Arduino Mega — Pi runs navigation + state machine; Arduino handles motor PWM + sensor polling at 50Hz; UART bridge
  • 🌿
    Brushless Cutting Deck + ESC — 30cm aluminium blade disc; 1400KV 2826 brushless motor; 30A ESC; emergency blade-stop relay GPIO

⚙️ How It Works

  1. 1

    GPS Boundary Mapping + Waypoint Generation

    Walk the lawn perimeter once with GPS logger to record polygon vertices. Save as GeoJSON. Compute boustrophedon coverage path inside polygon: generate parallel stripes spaced 25cm (cutting width) apart using Shapely: from shapely.geometry import Polygon; path = shapely_boustrophedon(lawn_polygon, stripe_width=0.25). Store waypoints list. At runtime, navigate to each waypoint in sequence: heading error → PD controller → differential drive.

  2. 2

    GPS Navigation PD Controller

    Current GPS position + IMU heading. Compute bearing to next waypoint: bearing = atan2(sin(Δlon)*cos(lat2), cos(lat1)*sin(lat2)-sin(lat1)*cos(lat2)*cos(Δlon)). Heading error: err = bearing - current_heading. PD controller: turn = Kp*err + Kd*(err-prev_err). Motor commands: left_pwm = base_speed - turn; right_pwm = base_speed + turn. Waypoint reached when distance < 0.15m. Advance to next waypoint.

  3. 3

    Obstacle Avoidance + Safety

    5× HC-SR04 sonar (front, 45°L, 45°R, rear-L, rear-R) polled at 50Hz by Arduino. Emergency stop: any forward sensor < 20cm. Avoidance: back up 30cm, rotate 90°, re-plan path. Blade-stop relay on any sonar < 10cm (safety for pets/children). Rain sensor: if rain_digital_pin == HIGH: blade_stop(); dock_return(). Tilt sensor (MPU-6050): stop blade if pitch >20° (lifted). Battery voltage divider → ADC → auto-dock when < 11.5V.

  4. 4

    Auto-Dock Return + Charging

    Dock location stored as GPS coordinate. When docking triggered (battery/rain/schedule): navigate to dock GPS point. Final approach: use IR beacon homing — dock transmits 38kHz IR, robot has front IR receiver, steers toward signal. Dock: slide onto spring-loaded charging contacts (12V 2A). Log session: area covered (m²), mowing time, battery usage, obstacles encountered. Send MQTT status message to home automation (Home Assistant, Node-RED).

💻 Core Code Snippet

import math, serial, time
from gps import *

def haversine(lat1,lon1,lat2,lon2):
    R=6371000
    φ1,φ2=math.radians(lat1),math.radians(lat2)
    Δφ=math.radians(lat2-lat1); Δλ=math.radians(lon2-lon1)
    a=math.sin(Δφ/2)**2+math.cos(φ1)*math.cos(φ2)*math.sin(Δλ/2)**2
    return R*2*math.atan2(math.sqrt(a),math.sqrt(1-a))

def bearing_to(lat1,lon1,lat2,lon2):
    Δlon=math.radians(lon2-lon1)
    lat1,lat2=math.radians(lat1),math.radians(lat2)
    x=math.sin(Δlon)*math.cos(lat2)
    y=math.cos(lat1)*math.sin(lat2)-math.sin(lat1)*math.cos(lat2)*math.cos(Δlon)
    return math.degrees(math.atan2(x,y))

Kp, Kd = 1.5, 0.3
BASE_SPEED = 60  # PWM 0-255
arduino = serial.Serial('/dev/ttyACM0', 115200)
gpsd = gps(mode=WATCH_ENABLE)

def navigate_to(target_lat, target_lon):
    prev_err = 0
    while True:
        gpsd.next(); report = gpsd.fix
        dist = haversine(report.latitude, report.longitude, target_lat, target_lon)
        if dist < 0.15: return  # waypoint reached
        bearing = bearing_to(report.latitude, report.longitude, target_lat, target_lon)
        heading = float(arduino.readline().split(b':')[1])  # compass
        err = (bearing - heading + 180) % 360 - 180
        turn = Kp*err + Kd*(err-prev_err); prev_err = err
        l = int(max(0, min(255, BASE_SPEED - turn)))
        r = int(max(0, min(255, BASE_SPEED + turn)))
        arduino.write(f"M:{l},{r}\n".encode())
