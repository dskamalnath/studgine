Alcohol-Interlocked Ignition

An MQ-3 alcohol sensor must detect safe breath levels before a relay allows an engine ignition signal. A road-safety demo project combining sensor thresholds and relay interlock logic.

📖 Introduction

Alcohol-interlocked ignition systems are mandated by law in several countries for repeat DUI offenders. This project builds an educational prototype demonstrating how an MQ-3 gas sensor — a tin-oxide chemiresistive device whose resistance decreases in the presence of ethanol vapour — can gate a relay that controls an ignition circuit.

The key engineering challenges are sensor warm-up (the heater coil must reach operating temperature before readings are valid), calibration against a known ethanol concentration reference, and designing the interlock state machine so the relay only closes in the SAFE state and requires re-measurement if the driver's door is opened again (preventing the "blow by proxy" trick).

🧩 Components Required

  • 🍷
    MQ-3 Alcohol Gas Sensor — Chemiresistive; R₀ calibration in clean air; heater draws ~180 mA, needs 5 V
  • 🧠
    Arduino Uno — ADC reads sensor voltage; controls relay and display; runs FSM
  • ⚡
    5 V Relay Module (1-channel) — Normally open (NO); closed only when BAC below safe threshold
  • 🖥️
    16×2 LCD or OLED — Shows current BAC % reading, state (WARM UP / READY / LOCKED), countdown timer
  • 🔔
    Passive Buzzer — Warning tones: double beep for READY, continuous for LOCKED state
  • 💡
    3× LEDs (R/Y/G) — Visual BAC level indicator bar (SAFE / CAUTION / LOCKOUT)

⚙️ How It Works

  1. 1

    Sensor Warm-Up & Calibration

    After power-on, the MQ-3 heater must warm for 30–60 seconds. During warm-up, display a countdown and disable the relay. To calibrate R₀: in clean air, read 200 ADC samples over 5 seconds, average them, convert to sensor resistance using Rs = (5.0 - Vout) / Vout × RL, then set R₀ = Rs / 60.0 (datasheet curve constant for clean air). Store R₀ in EEPROM so it survives power cycles.

  2. 2

    BAC Percentage Calculation

    Compute ratio = Rs / R₀. Use the MQ-3 datasheet sensitivity curve (log-log linear): ppm = A × ratio^B with constants A ≈ 0.3934, B ≈ -1.504. Convert ethanol ppm to approximate BAC% by multiplying by a breath-to-blood partition factor (~2100:1). Validate against a commercial breathalyser during calibration. Sample continuously at 10 Hz, use a 5-reading median filter to reduce noise from sensor shot noise and ambient breath disturbances.

  3. 3

    FSM: WARMING_UP → READY → SAFE / LOCKED

    Four states: WARMING_UP (30s timer; relay OFF; buzzer quiet), READY (two-beep prompt; user blows for 3 seconds), MEASURING (peak BAC captured over 3s blow window), RESULT: if BAC<0.05% → SAFE (relay ON; green LED; engine enabled 10 minutes), else LOCKED (relay OFF; buzzer 5 beeps; 20-minute lockout timer before retry). AUTO-RELOCK if vibration sensor (SW-420) detects car door slam.

  4. 4

    Relay Interlock Wiring

    Wire the relay's Normally Open (NO) contacts in series with the ignition key signal wire. The relay coil is driven by a BC547 transistor (Arduino pin → 1 kΩ base resistor → transistor base → collector drives relay coil → flyback diode across coil). The relay only ever closes when the firmware explicitly sets the output HIGH in the SAFE state — fail-safe means any firmware crash or power glitch defaults to engine-disabled (open circuit).

💻 Core Code Snippet

#include <EEPROM.h>
#include <LiquidCrystal_I2C.h>

const int MQ3_PIN = A0, RELAY_PIN = 8, BUZZ = 9;
float R0;

enum State { WARMING, READY, MEASURING, SAFE, LOCKED } state = WARMING;
unsigned long stateTimer;

float readBAC() {
  long sum=0; for(int i=0;i<50;i++){sum+=analogRead(MQ3_PIN);delay(10);}
  float Vout = (sum/50.0) * 5.0 / 1023.0;
  float Rs = (5.0-Vout)/Vout * 10.0; // RL=10kΩ
  float ratio = Rs / R0;
  float ppm = 0.3934 * pow(ratio, -1.504);
  return ppm / 2100.0; // approx BAC%
}

void setup() {
  pinMode(RELAY_PIN, OUTPUT); digitalWrite(RELAY_PIN, LOW);
  EEPROM.get(0, R0); if(R0 < 0.1 || R0 > 200) R0 = 9.8; // default
  stateTimer = millis();
}

void loop() {
  switch(state) {
    case WARMING:
      if(millis()-stateTimer > 45000) { tone(BUZZ,1000,200); state=READY; stateTimer=millis(); }
      break;
    case READY:
      if(millis()-stateTimer > 3000) { state=MEASURING; stateTimer=millis(); }
      break;
    case MEASURING: {
      float bac = readBAC();
      if(bac < 0.05) { digitalWrite(RELAY_PIN,HIGH); state=SAFE; }
      else { tone(BUZZ,500,1000); state=LOCKED; }
      stateTimer=millis();
    } break;
    case SAFE:
      if(millis()-stateTimer > 600000) { digitalWrite(RELAY_PIN,LOW); state=READY; }
      break;
    case LOCKED:
      if(millis()-stateTimer > 1200000) state=READY;
      break;
  }
}
