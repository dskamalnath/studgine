Facial Recognition Door Lock

Secure a door using face recognition — a Raspberry Pi 4 with Pi Camera continuously detects faces, compares encodings against a known-persons database, and actuates a servo bolt for authorised individuals.

📖 Introduction

The face_recognition Python library (built on dlib's ResNet) makes facial recognition accessible: it generates a 128-dimension face encoding from any photo, and compares new encodings with Euclidean distance (threshold ≈ 0.6). On Raspberry Pi 4, face detection runs at ~3–5 FPS with a 640×480 frame — fast enough for a door lock application.

You'll learn how computer vision pipelines are structured: capture → detect faces → extract encodings → compare → decide → actuate. You'll also handle the real-world complications: varying lighting, faces at angles, and preventing spoofing with printed photos.

🧩 Components Required

  • 🧠
    Raspberry Pi 4 (4GB RAM) — 4GB recommended; dlib encoding is CPU-intensive; enable OpenCV ARM NEON optimisations
  • 📷
    Raspberry Pi Camera v2 (or v3) — Pi Camera v3 has better low-light; connect via CSI ribbon; use picamera2 library for capture
  • 🔒
    MG996R Servo + Door Bolt — 0° = locked (bolt extended), 90° = unlocked (bolt retracted); 3D-print bolt bracket; servo powered from 5V 3A supply, NOT Pi GPIO
  • 🔊
    Active Buzzer + RGB LED — Green LED + short beep = access granted; Red LED + long beep = denied; visual/audio feedback
  • 💡
    IR LED Strip (850nm) — Illuminate face in low light for consistent recognition; Pi NoIR Camera v2 pairs perfectly with IR illumination

⚙️ How It Works

  1. 1

    Build the Known-Faces Database

    For each authorised person: take 5–10 photos under varied lighting and angles. Compute encoding for each: enc = face_recognition.face_encodings(image)[0]. Average the 128-dim vectors or store all individually. Pickle the list: pickle.dump({"name": name, "encodings": enc_list}, f). Load at startup. Adding a new person: run an enrolment mode that captures 10 photos live and adds to the pickle file.

  2. 2

    Live Detection Pipeline

    Capture frame with picamera2: frame = picam2.capture_array(). Resize to 320×240 for speed. Detect face locations: face_locs = face_recognition.face_locations(small_frame, model="hog"). Extract encodings: face_encs = face_recognition.face_encodings(small_frame, face_locs). For each detected face: compare with all known encodings using face_recognition.compare_faces(known_encs, face_enc, tolerance=0.55) and face_recognition.face_distance(known_encs, face_enc). Best match = lowest distance below threshold.

  3. 3

    Servo Bolt Actuation

    Use RPi.GPIO PWM: servo = GPIO.PWM(18, 50). Locked position: 2.5% duty cycle (0°, bolt extended). Unlocked: 7.5% duty (90°, bolt retracted). Stop PWM after movement to prevent servo jitter: servo.ChangeDutyCycle(0) 0.5s after move. Auto-relock after 5 seconds: use threading.Timer for non-blocking delay. Log all access events to SQLite: timestamp, name, confidence, granted/denied.

  4. 4

    Anti-Spoofing (Liveness Detection)

    Basic: require two consecutive positive matches 1 second apart (photo won't change between frames, real face will have micro-movement). Better: blink detection using facial landmarks — face_recognition.face_landmarks() returns eye points; compute eye aspect ratio (EAR); require EAR change (blink) within 3-second window for access. Advanced: use a depth camera (Intel RealSense) or structured light sensor to reject 2D photos.

💻 Core Code Snippet

import face_recognition, pickle, cv2
from picamera2 import Picamera2
import RPi.GPIO as GPIO, time, threading

# Load known faces
with open("encodings.pkl","rb") as f: db = pickle.load(f)
known_encs = [e for person in db for e in person["encodings"]]
known_names = [person["name"] for person in db for _ in person["encodings"]]

GPIO.setmode(GPIO.BCM); GPIO.setup(18, GPIO.OUT)
servo = GPIO.PWM(18, 50); servo.start(2.5)  # locked

def unlock():
    servo.ChangeDutyCycle(7.5); time.sleep(0.3)
    servo.ChangeDutyCycle(0)
    threading.Timer(5.0, lock).start()

def lock():
    servo.ChangeDutyCycle(2.5); time.sleep(0.3)
    servo.ChangeDutyCycle(0)

picam2 = Picamera2()
picam2.configure(picam2.create_preview_configuration(main={"size":(640,480)}))
picam2.start()

while True:
    frame = picam2.capture_array()
    small = cv2.resize(frame, (0,0), fx=0.5, fy=0.5)
    rgb = cv2.cvtColor(small, cv2.COLOR_BGR2RGB)
    locs = face_recognition.face_locations(rgb, model="hog")
    encs = face_recognition.face_encodings(rgb, locs)
    for enc in encs:
        dists = face_recognition.face_distance(known_encs, enc)
        best = dists.argmin()
        if dists[best] < 0.55:
            print(f"ACCESS GRANTED: {known_names[best]} ({dists[best]:.3f})")
            unlock()
        else:
            print(f"ACCESS DENIED ({dists[best]:.3f})")
    time.sleep(0.3)
