Brain-Computer Interface (EEG)

Stream 8-channel EEG from an OpenBCI Ganglion board. Collect motor imagery trials (imagine left/right hand movement). Apply Common Spatial Patterns (CSP) to extract discriminative features from mu/beta band power. Train a scikit-learn SVM to achieve 70–80% classification accuracy. Send FORWARD/LEFT/RIGHT commands to a serial robot over BLE in real-time.

📖 Introduction

Brain-computer interfaces allow direct control of devices through thought alone — originally developed for paralysed patients, but now accessible to students with affordable EEG hardware. Motor imagery BCI exploits Event-Related Desynchronisation (ERD): when you imagine moving your left hand, neural oscillations in the 8–30Hz mu/beta band decrease over the right motor cortex (and vice versa). Common Spatial Patterns (CSP) is a supervised spatial filter that maximises the variance ratio between two classes, making this signal difference large and easily classified by a linear SVM. OpenBCI Ganglion ($200) provides medical-grade EEG at 250Hz — enough for reliable motor imagery classification.

🧩 Components Required

  • 🧠
    OpenBCI Ganglion Board (4 or 8 ch) — ADS1299-based 24-bit EEG ADC; 250Hz; BLE / USB dongle; 4 active electrodes + driven-right-leg circuit; companion EEG headset optional
  • 🪖
    EEG Electrode Cap (10-20 positions) — Place C3 (left motor), C4 (right motor), Cz (reference), Oz (ground); conductive gel or sintered dry electrodes; headband mount
  • 💻
    PC / Raspberry Pi (Python + scikit-learn) — Stream via pyOpenBCI; NumPy CSP; scikit-learn SVM; real-time 1-second window classification; serial output commands
  • 🤖
    Arduino Robot or Servo Pan-Tilt — Serial command receiver: L/R/F/B/S; BLE-to-serial bridge optional; visual feedback display to close the human-in-the-loop

⚙️ How It Works

  1. 1

    EEG Data Collection — Motor Imagery Paradigm

    Install pyOpenBCI: pip install pyOpenBCI. Stream: board = OpenBCIGanglion(); board.start_stream(callback). Training protocol: display visual cue (arrow left/right on screen) every 4 seconds; subject imagines the corresponding hand movement for 3 seconds; label this trial. Collect 100+ trials per class (left hand / right hand). Epoch: extract 0.5–3.5s post-cue window at 250Hz = 750 samples/channel. Store as: X.shape = (n_trials, n_channels, n_samples); y = [0,1,0,1,...].

  2. 2

    Preprocessing + Band Power Features

    Bandpass filter to mu+beta band (8–30Hz): from scipy.signal import butter, sosfilt; sos = butter(4, [8,30], 'bandpass', fs=250, output='sos'); X_filt = sosfilt(sos, X, axis=-1). Common Average Reference (CAR): subtract mean across channels. Epoch variance as initial feature (for visualisation): band_power = np.var(X_filt, axis=-1). Motor cortex channels C3/C4 show the clearest ERD with left-right hand imagery.

  3. 3

    Common Spatial Patterns (CSP)

    CSP finds W such that W^T cov(class_0) W is diagonal and W^T cov(class_1) W = I. Implementation: from mne.decoding import CSP; csp = CSP(n_components=6, reg='ledoit_wolf', log=True); X_csp = csp.fit_transform(X_filt, y). Alternatively: Sigma1 = np.cov(X1.mean(0)); Sigma2 = np.cov(X2.mean(0)); W = generalised_eigvec(Sigma1, Sigma1+Sigma2). Use top+bottom 3 components. Resulting features: log-variance of projected components per trial — highly discriminative for left vs right imagery.

  4. 4

    SVM Classification + Robot Control

    Train: from sklearn.svm import SVC; from sklearn.pipeline import Pipeline; clf = Pipeline([('csp',CSP(6,'ledoit_wolf',True)),('svm',SVC(kernel='rbf',C=1.0,gamma='auto'))]); clf.fit(X, y). Cross-validation: from sklearn.model_selection import cross_val_score; scores = cross_val_score(clf, X, y, cv=5); print(f"Accuracy: {scores.mean():.1%}") — typically 70–80%. Real-time: sliding 1-second windows, predict every 500ms. 3 consecutive same prediction → send serial command: import serial; ser.write(b'L').

💻 Core Code Snippet

import numpy as np
from scipy.signal import butter, sosfilt
from sklearn.svm import SVC
from sklearn.pipeline import Pipeline
from mne.decoding import CSP

# --- Training ---
# X: (n_trials, n_channels, n_samples)  y: 0=left, 1=right
sos = butter(4, [8,30], 'bandpass', fs=250, output='sos')
X_filt = sosfilt(sos, X, axis=-1)

clf = Pipeline([
    ('csp', CSP(n_components=6, reg='ledoit_wolf', log=True)),
    ('svm', SVC(kernel='rbf', C=1.0, probability=True))
])
clf.fit(X_filt, y)
print(f"Train acc: {clf.score(X_filt, y):.1%}")

# --- Real-time streaming ---
import pyOpenBCI, serial
robot = serial.Serial('/dev/ttyACM0', 9600)
buf = np.zeros((8, 250*2))   # 2-second rolling buffer

def handle_sample(sample):
    global buf
    buf = np.roll(buf, -1, axis=1)
    buf[:, -1] = sample.channels_data[:8]
    epoch = buf[:, -250:].reshape(1, 8, 250)  # last 1 second
    epoch_f = sosfilt(sos, epoch, axis=-1)
    pred = clf.predict(epoch_f)[0]
    cmd = b'L' if pred == 0 else b'R'
    robot.write(cmd)

board = pyOpenBCI.OpenBCIGanglion()
board.start_stream(handle_sample)
