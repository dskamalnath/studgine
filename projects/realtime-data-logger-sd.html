Real-Time Data Logger (SD Card)

Capture timestamped sensor readings — temperature, humidity, voltage, and more — to a microSD card as a CSV file using Arduino and a DS3231 RTC. Import directly into Excel or Python for lab analysis and reports.

📖 Introduction

Field engineers and researchers often need hours or days of sensor data recorded offline — no internet, no laptop, just a battery-powered logger that quietly saves readings to a card. This project produces exactly that: a self-contained logging unit that writes a new row to a CSV file on microSD every 10 seconds, with an accurate timestamp from a battery-backed DS3231 real-time clock.

The CSV format is universally importable — just open in Excel, Google Sheets, or Python pandas. The DS3231's ± 2 ppm accuracy means timestamps drift less than one minute per year. Configurable logging intervals (1 s to 60 min) via a hardware dial make the same logger suitable for rapid electrical measurements or week-long climate studies.

🧩 Components Required

  • 🔧
    Arduino Uno / Nano — Central controller; Nano preferred for compact builds
  • 💾
    microSD Card Module (SPI) — Use Class 10 card 2–16 GB; FAT32 formatted
  • ⏰
    DS3231 RTC Module — I2C real-time clock; battery-backed so time survives power cycles
  • 🌡️
    DHT22 (Temp + Humidity) — ±0.5°C and ±2.5% RH accuracy; single-wire data pin
  • ⚡
    INA219 Current/Voltage Module — I2C power monitoring; measures supply voltage + current draw
  • 🖥️
    16×2 LCD or 0.96″ OLED — Live readout of last logged values + SD write status

⚙️ How It Works

  1. 1

    SD Card Initialisation

    Use the Arduino SD library with hardware SPI (CS pin 10 on Uno, pin 4 on Ethernet shield). Call SD.begin(CS_PIN) at startup. Check SD.exists("DATA.CSV") — if the file is new, write the header row first: file.println("datetime,temp_c,hum_pct,vbat_v,ibat_ma");. Open the file in APPEND mode (FILE_WRITE) to avoid overwriting previous sessions.

  2. 2

    Accurate Timestamping with DS3231

    Initialize DS3231 with RTClib: RTC.begin(); DateTime now = RTC.now();. Format timestamp as ISO 8601 string: now.year(), now.month(), now.day(), now.hour(), now.minute(), now.second() with zero-padding via sprintf. If RTC has lost power (first run), set time from compile timestamp: RTC.adjust(DateTime(F(__DATE__), F(__TIME__)));

  3. 3

    Multi-Sensor Reading

    Read DHT22 with dht.readTemperature() and dht.readHumidity(). Validate with isnan() check — DHT22 needs 2 s between reads. Read INA219 with ina219.getBusVoltage_V() and ina219.getCurrent_mA(). Read any analog sensor (light, soil moisture, etc.) via ADC: analogRead(A0) * (5.0 / 1023.0). All in a single data-collection function called every logging interval.

  4. 4

    Write CSV Row and Safe Close

    Build the CSV line with comma-separators and limited decimal places (2 d.p. for temperature, 1 for voltage). Always close the file after each write: file.close();. This is critical — if power is lost mid-session, an unclosed FAT32 file will be 0 bytes and all data lost. Use a LED write-indicator that blinks during SD access. Increment a row counter and display on LCD so the user can see logging is active.

💻 Core Code Snippet

#include <SD.h>
#include <RTClib.h>
#include <DHT.h>
#include <Adafruit_INA219.h>

RTC_DS3231 rtc; DHT dht(7, DHT22); Adafruit_INA219 ina;
File logFile;
const int CS_PIN = 10;
unsigned long lastLog = 0, LOG_INTERVAL = 10000;

void setup() {
  SD.begin(CS_PIN); rtc.begin(); dht.begin(); ina.begin();
  if(!SD.exists("DATA.CSV")) {
    logFile = SD.open("DATA.CSV", FILE_WRITE);
    logFile.println("datetime,temp_c,hum_pct,vbus_v,cur_ma");
    logFile.close();
  }
}

void loop() {
  if(millis() - lastLog >= LOG_INTERVAL) {
    DateTime now = rtc.now();
    float t = dht.readTemperature();
    float h = dht.readHumidity();
    float v = ina.getBusVoltage_V();
    float c = ina.getCurrent_mA();
    char ts[20];
    sprintf(ts, "%04d-%02d-%02d %02d:%02d:%02d",
      now.year(), now.month(), now.day(),
      now.hour(), now.minute(), now.second());
    logFile = SD.open("DATA.CSV", FILE_WRITE);
    logFile.print(ts); logFile.print(",");
    logFile.print(t,2); logFile.print(",");
    logFile.print(h,1); logFile.print(",");
    logFile.print(v,3); logFile.print(",");
    logFile.println(c,1);
    logFile.close();      // always close!
    lastLog = millis();
  }
}
