Digital Stethoscope with ML

A high-sensitivity MEMS microphone captures heart and lung sounds amplified through a custom chest piece. A log-mel spectrogram CNN trained on the PhysioNet dataset classifies normal vs. abnormal cardiac audio patterns in real time on Raspberry Pi — a genuinely medical-grade AI demonstration.

📖 Introduction

Auscultation — listening to body sounds — is a fundamental diagnostic skill requiring years of training. This project digitises the process: a MEMS microphone (ICS-43434 or SPH0645) pressed against the chest captures the S1 (lub) and S2 (dub) heart sounds at high fidelity. A log-mel spectrogram is computed on each 3-second audio window and fed to a lightweight CNN trained on the PhysioNet 2016 Heart Sound Challenge dataset (3153 recordings across 5 classes: normal, murmur, extra sounds, extrasystole, artefact). The model achieves ~88% test accuracy, making this a genuinely impressive medical AI showcase.

🧩 Components Required

  • 🎤
    MEMS I²S Microphone (SPH0645 or ICS-43434) — 48kHz I²S digital output; connects to Pi GPIO 18/19/20; SNR 65dB; captures 20Hz–20kHz cardiac sounds
  • 🩺
    Acoustic Chest Piece + Tubing — 3D-print or adaptor bell chamber; silicone tubing to MEMS capsule; standard stethoscope diaphragm for acoustic coupling
  • 🧠
    Raspberry Pi 4 (4 GB) — TFLite CNN inference; sounddevice/I²S capture at 4kHz (cardiac band); pyaudio; 3.5mm output for audio playback
  • 📊
    7" Touchscreen or HDMI Display — Show waveform + spectrogram + classification result; tkinter or pygame GUI; LED: green=normal, red=abnormal

⚙️ How It Works

  1. 1

    I²S MEMS Audio Capture

    Configure I²S on Raspberry Pi: add dtoverlay=i2s-mems-mic to /boot/config.txt. Enable snd_soc_hifiberry_dac module. Record with sounddevice: audio = sd.rec(int(3*4000), samplerate=4000, channels=1, dtype='float32'). Sample at 4kHz — sufficient for 20–1000Hz cardiac band. Apply bandpass filter: scipy.signal.butter(4, [20/2000, 1000/2000], btype='band') to isolate heart sounds from noise.

  2. 2

    Log-Mel Spectrogram Feature Extraction

    Compute mel spectrogram: import librosa; S = librosa.feature.melspectrogram(y=audio, sr=4000, n_mels=64, hop_length=64, n_fft=512). Convert to log scale: S_db = librosa.power_to_db(S, ref=np.max). Normalise: zero-mean, unit-variance over each mel band. Resize to fixed [64×64] tensor. This 2D image captures both frequency content and temporal structure of the cardiac cycle — exactly what a CNN can learn to classify.

  3. 3

    CNN Training on PhysioNet Dataset

    Download PhysioNet 2016 dataset (3153 WAV files + labels). Augment: time-stretch ×0.9/1.1, add white noise, random volume. Build CNN: Conv2D(32)→BN→ReLU→MaxPool → Conv2D(64)→BN→ReLU→MaxPool → GlobalAvgPool → Dense(64) → Softmax(5). Train with Adam, categorical cross-entropy. 5-fold CV: ~88% balanced accuracy. Export to TFLite INT8. Inference time on Pi 4: ~15ms per 3-second window.

  4. 4

    BPM Detection + Real-Time Display

    S1/S2 peak detection with scipy: peaks, _ = scipy.signal.find_peaks(envelope, height=0.3, distance=int(0.3*sr)). BPM: 60 / np.mean(np.diff(peaks)/sr). GUI: scrolling waveform plot + real-time spectrogram + classification bar (green/red). Record 30-second sessions to WAV for later review. Optional: Bluetooth LE audio streaming to smartphone app for remote monitoring.

💻 Core Code Snippet

import numpy as np, sounddevice as sd, librosa
import tflite_runtime.interpreter as tflite
import scipy.signal

SAMPLE_RATE = 4000; DURATION = 3
CLASSES = ['Normal','Murmur','Extra','Extrasystole','Artefact']

interp = tflite.Interpreter('heart_sound_cnn.tflite')
interp.allocate_tensors()
in_d = interp.get_input_details(); out_d = interp.get_output_details()

def classify_heart_sound():
    audio = sd.rec(int(DURATION*SAMPLE_RATE), samplerate=SAMPLE_RATE,
                   channels=1, dtype='float32')
    sd.wait()
    y = audio[:,0]
    # Bandpass filter 20-500Hz
    sos = scipy.signal.butter(4,[20/(SAMPLE_RATE/2),500/(SAMPLE_RATE/2)],
                              btype='band', output='sos')
    y = scipy.signal.sosfilt(sos, y)
    # Log-mel spectrogram
    S = librosa.feature.melspectrogram(y=y, sr=SAMPLE_RATE,
                                        n_mels=64, hop_length=64, n_fft=512)
    S_db = librosa.power_to_db(S, ref=np.max)
    S_norm = (S_db - S_db.mean()) / (S_db.std() + 1e-6)
    x = np.resize(S_norm, (64,64))[np.newaxis,...,np.newaxis].astype(np.float32)
    interp.set_tensor(in_d[0]['index'], x)
    interp.invoke()
    probs = interp.get_tensor(out_d[0]['index'])[0]
    pred = CLASSES[np.argmax(probs)]
    return pred, float(np.max(probs))

while True:
    label, conf = classify_heart_sound()
    print(f"Classification: {label} ({conf*100:.1f}%)")
