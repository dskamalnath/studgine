Digital Stopwatch

Three push-buttons start, stop, and reset a millisecond-accurate stopwatch displayed on a 4-digit 7-segment display using Arduino's millis().

📖 Introduction

The Arduino millis() function returns the number of milliseconds since the board last reset — it never blocks your code. By recording a start timestamp when the Start button is pressed and computing elapsed = millis() - startTime, you get a non-blocking timer that keeps running while everything else works simultaneously.

The TM1637 4-digit LED display module reduces wiring to just two wires (CLK + DIO). Its Arduino library handles segment encoding, so you simply call display.showNumberDecEx() to show minutes and seconds with a blinking colon.

⚙️ How It Works

  1. 1
    Connect the display — TM1637 CLK → D2, DIO → D3, VCC → 5 V, GND → GND. Buttons to D4 (Start), D5 (Stop), D6 (Reset) with pull-down resistors. Enable INPUT_PULLUP in code if you wire buttons to GND instead.
  2. 2
    State machine — Track enum State { IDLE, RUNNING, STOPPED };. On Start press: record startTime = millis(); and set state to RUNNING. On Stop: record elapsedSnapshot and set STOPPED. On Reset: clear all variables and return to IDLE.
  3. 3
    Calculate MM:SS — unsigned long elapsed = millis() - startTime;. Then: int secs = (elapsed/1000)%60; int mins = (elapsed/60000)%60;. Pack into display: int displayVal = mins*100 + secs; and call display.showNumberDecEx(displayVal, 0x40, true); to show colon.
  4. 4
    Debounce buttons — Check button state only every 50 ms using a lastDebounce millis() check. Or use a simple flag: read the button, only act when state changes from HIGH to LOW (on release). This prevents multiple triggers from a single press.
