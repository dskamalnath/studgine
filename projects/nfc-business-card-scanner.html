NFC Business Card Scanner

Tap an NFC card on a PN532 reader to display contact info on an OLED and log the record to SD card or send via Bluetooth. Program new NFC tags with contact vCard data.

📖 Introduction

Near Field Communication (NFC) enables contactless data exchange at 13.56 MHz within a few centimetres. The PN532 is the most versatile NFC controller available in the hobbyist market — it handles ISO14443A/B, NTAG21x, Mifare Classic, and FeliCa standards that cover the vast majority of business cards and smart labels.

NDEF (NFC Data Exchange Format) is the standard way data is stored on tags. A vCard record encodes name, phone, email, and company fields in a compact binary-text format. This project parses incoming NDEF messages, extracts contact fields, renders them on an OLED screen, and optionally programs blank NFC tags with new contact data — making it a complete two-way NFC workstation.

🧩 Components Required

  • 📶
    PN532 NFC Module — HSU/I2C/SPI selectable; up to 10 cm range; 3.3 V logic
  • 🧠
    Arduino Uno / Nano — Software or hardware SPI to PN532; 5 V tolerant
  • 🖥️
    0.96″ SSD1306 OLED — 128×64 I2C display for contact details
  • 💾
    MicroSD + SPI Module — Logs contact name, UID, and timestamp as CSV rows
  • 📡
    HC-05 Bluetooth (optional) — Stream vCard as serial text to phone contacts app
  • 📇
    NTAG213 NFC Stickers / Cards — 144 bytes NDEF capacity; ISO14443A compliant

⚙️ How It Works

  1. 1

    PN532 SPI Initialisation

    Set the PN532 module's DIP switches to SPI mode. Initialise with nfc.begin() and confirm the firmware version with nfc.getFirmwareVersion(). Set passive activation retries to 10 and start polling with nfc.readPassiveTargetID(PN532_MIFARE_ISO14443A, uid, &uidLength). The tag UID is a 4–7 byte identifier unique per chip.

  2. 2

    NDEF Record Parsing

    Use the Adafruit PN532 NDEF library. Read all NDEF records from the tag memory. A vCard NDEF record carries type "text/vcard" with payload: BEGIN:VCARD\nVERSION:3.0\nFN:Alex Kim\nTEL:+1-555\nEMAIL:a@b.com\nEND:VCARD. Parse newline-delimited fields with strtok() to extract name/phone/email into separate char arrays.

  3. 3

    OLED Display & SD Logging

    Render contact details on the SSD1306 in 4 rows: Name (row 0), Phone (row 1), Email (row 2 — truncate to 21 chars), company (row 3). Log to SD as CONTACTS.CSV with columns: timestamp, uid_hex, name, phone, email. Detect duplicate UIDs by searching the last 50 lines of the CSV before writing again (prevents re-tap duplication).

  4. 4

    NFC Tag Writing Mode

    Hold a push button to enter WRITE mode. Via USB serial, send a JSON string {"name":"Alex","tel":"555","email":"a@b.com"}. The firmware formats it as vCard, encodes an NDEF message, and writes pages starting at page 4 on an NTAG213 tag. Each page is 4 bytes; NTAG213 has pages 4–39 available (136 bytes), enough for a compact vCard with 130-char contacts.

💻 Core Code Snippet

#include <Wire.h>
#include <Adafruit_PN532.h>
#include <Adafruit_SSD1306.h>
#include <SD.h>

#define PN532_SS 10
Adafruit_PN532 nfc(PN532_SS);
Adafruit_SSD1306 display(128, 64, &Wire);

void setup() {
  nfc.begin();
  nfc.SAMConfig();
  display.begin(SSD1306_SWITCHCAPVCC, 0x3C);
  SD.begin(9); // SD CS on pin 9
}

void loop() {
  uint8_t uid[7]; uint8_t uidLen;
  if(nfc.readPassiveTargetID(PN532_MIFARE_ISO14443A, uid, &uidLen)) {
    String uidHex = "";
    for(int i=0;i<uidLen;i++) { if(uid[i]<16) uidHex+="0"; uidHex+=String(uid[i],HEX); }
    // Read NDEF message pages from NTAG213 (pages 4-39)
    uint8_t page[4]; char vcard[180] = "";
    for(int pg=4; pg<=39; pg++) {
      if(!nfc.ntag2xx_ReadPage(pg, page)) break;
      for(int b=0;b<4;b++) if(page[b]) strncat(vcard,(char*)&page[b],1);
    }
    // Parse and display
    display.clearDisplay();
    display.setTextSize(1); display.setTextColor(SSD1306_WHITE);
    display.setCursor(0,0); display.print("UID: "); display.println(uidHex);
    // Log to SD
    File f = SD.open("contacts.csv", FILE_WRITE);
    if(f) { f.print(millis()); f.print(","); f.print(uidHex); f.print(","); f.println(vcard); f.close(); }
    display.display(); delay(3000);
  }
}
