📖 Introduction

Air quality monitoring is one of the most impactful IoT applications you can build as a student. This project goes beyond a single sensor node — you'll deploy a small mesh of ESP32 boards, each measuring particulate matter (PM2.5) and toxic gas concentrations, then aggregate all readings into a time-series InfluxDB database and visualise them with Grafana dashboards.

It teaches the complete end-to-end IoT pipeline: sensor calibration, firmware development, network protocols (MQTT or ESP-NOW mesh), database ingestion, and real-time dashboarding — every skill valued by IoT employers and research labs.

🧩 Components Required

  • 🎛️
    ESP32 DevKit × 2–3 — One per monitoring node
  • 🌫️
    MQ-135 Gas Sensor — CO₂, NH₃, benzene, smoke detection
  • 💨
    PM2.5 Dust Sensor (GP2Y1010) — Measures fine particulate matter
  • 🖥️
    0.96″ OLED Display — Local AQI readout on each node
  • 🔋
    18650 Battery + TP4056 Charger — Portable, rechargeable power
  • 🖥️
    Raspberry Pi / PC (server) — Runs InfluxDB + Grafana stack

⚙️ How It Works

  1. 1

    Calibrate & Wire Sensors

    The MQ-135 requires a 24-hour burn-in and a Rs/R0 calibration in clean air. The GP2Y1010 uses an LED pulse circuit — wire the 150 Ω resistor and 220 µF capacitor per the datasheet, then read the analog output on an ADC pin.

  2. 2

    Set Up Mesh Network

    Use ESP-NOW (peer-to-peer, no router) to relay readings from remote nodes to a gateway node. Alternatively, connect each ESP32 to Wi-Fi and publish directly to an MQTT broker on the Raspberry Pi.

  3. 3

    Ingest into InfluxDB

    A Node-RED flow (or a small Python script) subscribes to the MQTT topics and writes each reading as an InfluxDB time-series measurement: air_quality,node=N1 ppm=412,pm25=18 <timestamp>.

  4. 4

    Build Grafana Dashboard

    Connect Grafana to InfluxDB as a data source. Create time-series panels for each metric, add threshold bands for WHO AQI levels, and configure alert rules that send Telegram notifications when PM2.5 exceeds safe limits.
