Fan Speed PWM Controller

📖 Introduction

Pulse Width Modulation (PWM) is how a microcontroller "pretends" to output an analog voltage using a digital pin. By toggling the pin on and off very rapidly, the average voltage seen by the load is proportional to the duty cycle — 50% duty = ~2.5 V equivalent, 100% duty = 5 V. Fans, LEDs, and motors all respond smoothly to PWM control.

A 10 kΩ potentiometer gives a 0–1023 analog reading on the ADC. The Arduino's built-in map() function re-scales that to 0–255, which is then written to a PWM pin. The result: a fully analog-feel fan speed knob built with digital hardware.

⚙️ How It Works

  1. 1
    Read the pot — Connect the pot wiper to A0. In loop(): int potVal = analogRead(A0); — returns 0–1023, representing 0–5 V.
  2. 2
    Map to PWM range — int pwmVal = map(potVal, 0, 1023, 0, 255); — scales linearly. Most fans need a minimum ~80 PWM to spin; you can clamp: if(pwmVal < 80 && pwmVal > 0) pwmVal = 80;
  3. 3
    Drive the fan transistor — Connect Arduino pin 9 (PWM) through a 1 kΩ resistor to the base of an NPN transistor (2N2222 or TIP120). Fan positive → 5 V rail; fan negative → transistor collector; emitter → GND. Place a 1N4007 flyback diode across the fan terminals.
  4. 4
    Write PWM — analogWrite(9, pwmVal); sets the duty cycle. Add a delay(20); to reduce ADC noise jitter. Optionally print to Serial Monitor: Serial.println(pwmVal); to visualise the mapping live.
