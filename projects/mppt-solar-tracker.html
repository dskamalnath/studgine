MPPT Solar Tracker

Combine dual-axis LDR sun-following with a Perturb & Observe Maximum Power Point Tracking algorithm. Servo motors tilt the panel toward the sun while the MPPT code extracts the maximum available power from any irradiance condition.

📖 Introduction

A fixed solar panel produces up to 40% less energy than one that tracks the sun across the sky. This project adds two layers of intelligence: a physical tracking layer using quad-LDR differential sensing that continuously tilts the panel toward the sun, and a power-point tracking layer using the Perturb & Observe (P&O) algorithm that adjusts the PWM duty cycle on a DC-DC converter to extract the maximum power at the panel's operating point.

Together these techniques can increase daily energy harvest by 35–55% compared to fixed-mount systems, making the project highly relevant to renewable energy engineering coursework.

🧩 Components Required

  • ☀️
    5 V / 1 W Mini Solar Panel — Primary power source for demonstration; 138 × 60 mm typical
  • 📡
    4× LDR (GL5528) + 10 kΩ Resistors — Voltage dividers in quad arrangement: top-left, top-right, bot-left, bot-right
  • 🤖
    2× MG995 Servo Motors — Azimuth (±90°) and elevation (0–90°) axes
  • 🔧
    Arduino Uno / Nano — Runs both tracking loop and P&O MPPT algorithm
  • ⚡
    INA219 Voltage/Current Module — Measures Vpanel and Ipanel for P&O power calculation
  • 🔋
    Li-Po 3.7 V Battery + TP4056 — Stores harvested energy; TP4056 handles safe charging

⚙️ How It Works

  1. 1

    Quad-LDR Sun Tracking

    Mount four LDRs at the panel corners separated by a cross-shaped cardboard divider. Read ADC values for all four. Compute: topDiff = (TL+TR) - (BL+BR) for elevation; leftDiff = (TL+BL) - (TR+BR) for azimuth. If |diff| exceeds a threshold (deadband), increment the respective servo angle by 1°. This prevents hunting when sun is already centred. Limit servo range to 10°–170° to protect mechanism.

  2. 2

    Perturb & Observe MPPT

    Every 100 ms: read Vpanel and Ipanel from INA219. Compute P_new = V * I. If P_new > P_old: panel is moving toward MPP — continue in same direction (increment duty cycle). If P_new < P_old: overshot — reverse direction. Update P_old = P_new. Output the duty cycle to a PWM-controlled MOSFET driving a DC-DC buck converter that feeds the load.

  3. 3

    Night-Return and Cloud Handling

    Monitor total LDR average. If ambient light drops below threshold (e.g., LDR average > 900 on 10-bit scale): enter sleep mode. Park both servos at home position (panel flat, facing east for morning sun). Wake from sleep mode when LDR average drops below 600 — indicating sunrise. Handle passing clouds by pausing MPPT for 5 s to avoid erratic hunting during rapid irradiance changes.

  4. 4

    OLED Dashboard & Logging

    Display on 0.96″ OLED: panel voltage, current, power (W), daily energy Wh counter (accumulated), efficiency % vs fixed mount (estimated). Log every 5 min to SD card for post-session chart. Optionally upload via ESP8266 to ThingSpeak for web-based monitoring. A 4-digit 7-segment display showing live watts is also a popular output for live demos.

💻 Core Code Snippet (P&O MPPT)

float V_old = 0, P_old = 0;
int dutyCycle = 128;  // 0–255 PWM
int perturbStep = 3;

void mpptStep() {
  float V = ina219.getBusVoltage_V();
  float I = ina219.getCurrent_mA() / 1000.0;
  float P = V * I;

  if(P > P_old) {
    // still climbing toward MPP — keep direction
    if(V > V_old) dutyCycle += perturbStep;
    else dutyCycle -= perturbStep;
  } else {
    // overshot — reverse direction
    if(V > V_old) dutyCycle -= perturbStep;
    else dutyCycle += perturbStep;
  }
  dutyCycle = constrain(dutyCycle, 50, 230);
  analogWrite(PWM_PIN, dutyCycle);
  V_old = V; P_old = P;
}

void trackSun() {
  int TL = analogRead(A0), TR = analogRead(A1);
  int BL = analogRead(A2), BR = analogRead(A3);
  int errElev = (TL+TR) - (BL+BR);
  int errAzi = (TL+BL) - (TR+BR);
  if(abs(errElev) > 50) elevAngle += (errElev>0)?1:-1;
  if(abs(errAzi) > 50) aziAngle += (errAzi>0)?1:-1;
  elevServo.write(constrain(elevAngle,10,170));
  aziServo.write(constrain(aziAngle,10,170));
}
