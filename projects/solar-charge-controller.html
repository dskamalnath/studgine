Solar Charge Controller

📖 Introduction

A solar charge controller sits between the panel and the battery, regulating current flow to prevent overcharging (which damages cells) and deep discharge (which destroys lead-acid capacity). Commercial PWM controllers do this with a shunt switch — when battery voltage hits the bulk/absorption threshold, they reduce charge current by chopping the panel connection with a PWM signal.

This Arduino project measures panel and battery voltage through potential dividers, calculates charge current using an ACS712 Hall-effect current sensor, and drives an IRF3205 N-channel MOSFET with PWM to regulate charging. An LCD shows real-time panel wattage, battery state of charge, load current, and daily Ah throughput — a surprisingly complete energy management system for around $20.

🧩 Components Required

  • 🎛️
    Arduino Uno — PWM output, analog ADC for voltage/current
  • ⚡
    IRF3205 N-Channel MOSFET — Main PWM switch; R_DS(on) = 8 mΩ at 10 V gate
  • 🔌
    ACS712-20A Current Sensor — Hall-effect, 100 mV/A, measures charge current
  • 📏
    Voltage Dividers (10kΩ + 3.3kΩ) — Scale 20 V panel down to 5 V ADC range
  • 📟
    16×2 LCD with I2C — Energy dashboard display
  • 🔋
    12 V 7 Ah SLA Battery + 20 W Panel — Sealed lead-acid and small solar panel for testing

⚙️ How It Works

  1. 1

    Measure Voltages and Current

    Panel voltage: read A0, multiply by (R1+R2)/R2 = (10k+3.3k)/3.3k = 4.03 scaling factor. Battery voltage: read A1 similarly. Current (ACS712): read A2; voltage midpoint at 2.5 V = 0 A; amps = (analogRead(A2) * 5.0/1023 - 2.5) / 0.1. Average 50 readings to reduce noise.

  2. 2

    PWM Charging Algorithm (3-Stage)

    Implement bulk (MOSFET 100% ON until 14.4 V), absorption (PWM regulated to hold 14.4 V for 2 hours), and float (PWM pulse at low duty to maintain 13.6 V). Use PID control on battery voltage → PWM duty cycle for smooth regulation instead of bang-bang switching.

  3. 3

    Overcharge and Deep-Discharge Protection

    If battery voltage exceeds 14.8 V, cut the PWM signal entirely (MOSFET OFF). If battery drops below 11.0 V (≈50% SoC for SLA), relay-disconnect the load output to prevent deep discharge damage. Display "BATT LOW – LOAD OFF" and sound buzzer. Reconnect load above 12.0 V.

  4. 4

    Daily Energy Accounting

    Every second, integrate Ah += current × (1/3600). At midnight (DS3231 RTC check), save daily kWh to EEPROM and reset. Display today's panel harvest and load consumption on the LCD row 2, giving a complete picture of the energy budget.

💻 Core Code Snippet

#include <LiquidCrystal_I2C.h>
LiquidCrystal_I2C lcd(0x27, 16, 2);

#define MOSFET_PWM  9
#define ACS_PIN     A2
#define VPANEL_PIN  A0
#define VBATT_PIN   A1
#define LOAD_RELAY  7

const float V_SCALE = 4.03;     // voltage divider ratio
const float BULK_V  = 14.4;
const float FLOAT_V = 13.6;
const float LVD_V   = 11.0;     // low-voltage disconnect

int pwmDuty = 255;

float readVoltage(int pin) {
  long sum = 0;
  for(int i=0;i<50;i++) sum += analogRead(pin);
  return (sum / 50.0) * (5.0 / 1023.0) * V_SCALE;
}

float readCurrent() {
  long sum = 0;
  for(int i=0;i<50;i++) sum += analogRead(ACS_PIN);
  return ((sum / 50.0) * (5.0/1023.0) - 2.5) / 0.1;
}

void setup() {
  pinMode(MOSFET_PWM, OUTPUT); pinMode(LOAD_RELAY, OUTPUT);
  lcd.init(); lcd.backlight();
}

void loop() {
  float vPanel = readVoltage(VPANEL_PIN);
  float vBatt  = readVoltage(VBATT_PIN);
  float iCharge = readCurrent();
  // 3-Stage charging
  if (vBatt >= BULK_V) {
    // Absorption / Float via simple P control
    float error = vBatt - BULK_V;
    pwmDuty = constrain(pwmDuty - (int)(error * 50), 0, 255);
  } else {
    pwmDuty = 255;  // Bulk: full power
  }
  // Low voltage load disconnect
  if (vBatt < LVD_V) digitalWrite(LOAD_RELAY, LOW);
  else if (vBatt > 12.0) digitalWrite(LOAD_RELAY, HIGH);
  analogWrite(MOSFET_PWM, pwmDuty);
  // Display
  lcd.setCursor(0,0);
  lcd.print("P:" + String(vPanel,1) + "V " + String(vPanel*iCharge,1) + "W");
  lcd.setCursor(0,1);
  lcd.print("B:" + String(vBatt,1) + "V D:" + String(pwmDuty*100/255) + "%");
  delay(1000);
}
