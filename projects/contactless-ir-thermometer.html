Contactless IR Thermometer

Point an MLX90614 infrared sensor at any surface or forehead for an instant non-contact temperature reading on an OLED — with automatic fever threshold alerts and ambient compensation.

📖 Introduction

The MLX90614 is a factory-calibrated thermopile that measures the infrared radiation emitted by any object in its 35° field of view — without touching it. Its on-chip 17-bit ADC and DSP provide a temperature resolution of 0.02°C and accuracy of ±0.5°C, which exceeds many commercial ear thermometers.

Communication is via SMBus (compatible with I2C at 100 kHz). This project builds a handheld thermometer gun using an Arduino Nano, the MLX90614, an SSD1306 OLED, an LED indicator, and a piezo buzzer. The firmware applies ambient temperature compensation and a ±0.3°C emissivity adjustment for skin vs. surface measurements — covering both clinical and industrial uses.

🧩 Components Required

  • 🎛️
    Arduino Nano — 3.3 V tolerant I2C master
  • 🌡️
    MLX90614 IR Sensor — I2C @ 0x5A, 3.3 V or 5 V version
  • 📟
    SSD1306 0.96" OLED — 128×64 I2C display @ 0x3C
  • 💡
    Green + Red LED — Normal / fever indicator LEDs
  • 🔊
    Active Buzzer — Beep pattern on fever detection
  • 🔘
    Push Button (trigger) — Take single-point reading on demand

⚙️ How It Works

  1. 1

    Read Object and Ambient Temperature

    Call mlx.readObjectTempC() for the target surface and mlx.readAmbientTempC() for the internal die temperature. Both values are returned as floats. The sensor pre-compensates for ambient variations internally, but reading ambient is useful for displaying environmental conditions.

  2. 2

    Moving Average Noise Filter

    Raw readings fluctuate ±0.3°C due to electrical noise. Implement a 5-sample circular buffer: keep the last 5 readings, return the average. Discard any reading more than 5°C from the current average (outlier rejection for motion errors).

  3. 3

    Emissivity and Body Compensation

    MLX90614 ships with emissivity = 1.0 (ideal blackbody). Human skin emissivity ≈ 0.98. Add +0.5°C offset to convert wrist/forehead measurement to oral-equivalent temperature (clinical standard). Store mode (BODY / SURFACE) in a toggle variable, applying different correction factors.

  4. 4

    Display and Alert

    Render the temperature in large font on the OLED. If temp ≥ 37.5°C, display "FEVER" in red, light the red LED, and sound 3 short buzzer beeps. Below threshold: "NORMAL" in green. Output readings every 500 ms to Serial for logging via Python script or serial recorder.

💻 Core Code Snippet

#include <Wire.h>
#include <Adafruit_MLX90614.h>
#include <Adafruit_SSD1306.h>

Adafruit_MLX90614 mlx;
Adafruit_SSD1306 oled(128, 64, &Wire, -1);

#define FEVER_THRESH 37.5
#define LED_GREEN 4
#define LED_RED   5
#define BUZZER    6
#define TRIGGER   7
#define AVG_SIZE  5

float readings[AVG_SIZE]; int readIdx = 0;

float filteredTemp() {
  float sum = 0;
  for (int i = 0; i < AVG_SIZE; i++) sum += readings[i];
  return sum / AVG_SIZE;
}

void setup() {
  mlx.begin();
  oled.begin(SSD1306_SWITCHCAPVCC, 0x3C);
  pinMode(LED_GREEN, OUTPUT); pinMode(LED_RED, OUTPUT);
  pinMode(BUZZER, OUTPUT); pinMode(TRIGGER, INPUT_PULLUP);
  // Pre-fill buffer
  for (int i=0; i<AVG_SIZE; i++) readings[i] = 36.5;
}

void takeReading() {
  float raw = mlx.readObjectTempC() + 0.5;  // body correction
  readings[readIdx++ % AVG_SIZE] = raw;
  float temp = filteredTemp();
  float ambient = mlx.readAmbientTempC();
  oled.clearDisplay(); oled.setTextSize(2); oled.setCursor(0,0);
  oled.print(String(temp,1) + " C");
  oled.setTextSize(1); oled.setCursor(0,24);
  if (temp >= FEVER_THRESH) {
    oled.print("!! FEVER DETECTED !!");
    digitalWrite(LED_RED, HIGH); digitalWrite(LED_GREEN, LOW);
    for(int b=0;b<3;b++){digitalWrite(BUZZER,1);delay(150);digitalWrite(BUZZER,0);delay(150);}
  } else {
    oled.print("NORMAL");
    digitalWrite(LED_GREEN, HIGH); digitalWrite(LED_RED, LOW);
  }
  oled.setCursor(0,40); oled.print("Ambient: " + String(ambient,1) + "C");
  oled.display();
}

void loop() {
  if (!digitalRead(TRIGGER)) { takeReading(); delay(500); }
}
