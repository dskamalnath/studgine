Sign Language Translator

Translate ASL hand signs to spoken words in real time using MediaPipe hand landmarks, a TFLite neural network classifier on Raspberry Pi, and pyttsx3 text-to-speech — an assistive technology project with real-world impact.

📖 Introduction

MediaPipe Hands (by Google) detects 21 3D landmarks per hand in real time — even on Raspberry Pi 4 at 14+ FPS. This project uses those landmarks as a 63-feature vector (21 points × x,y,z) to train a lightweight neural network to classify ASL letter hand shapes. The classifier runs as a TFLite model for speed.

This is a complete machine learning pipeline project: data collection → preprocessing → model training → quantisation to INT8 TFLite → edge deployment → speech output. The assistive technology angle makes it an excellent showcasing project for capstone presentations.

🧩 Components Required

  • 🧠
    Raspberry Pi 4 (2GB+) — 2GB sufficient; TFLite inference is fast; HDMI display for live camera preview with landmark overlay
  • 📷
    Pi Camera v2 / USB Webcam — USB webcam (720p) works well; position obliquely from above at 45° for clearest landmark detection
  • 🔊
    USB Speaker or 3.5mm Speaker — pyttsx3 outputs speech via ALSA; USB speaker avoids Pi audio jack noise issues
  • 🖥️
    7" HDMI Touchscreen (optional) — Shows live video with landmark overlay + recognised letter; touchscreen adds word-delete and TTS toggle buttons

⚙️ How It Works

  1. 1

    Data Collection

    Run a capture script: press A–Z keys to label, capture 200 frames per letter. For each frame: extract 21 MediaPipe landmarks, normalise to wrist origin (subtract landmark 0), flatten to 63-feature vector. Save to CSV. Total: 26 × 200 = 5200 samples. Include variation: hand distance, angle, slight rotation. This step takes ~1–2 hours but is essential for accuracy.

  2. 2

    Train the Classifier

    On a more powerful PC (or Colab): load CSV, split 80/20 train/test. Model architecture: Dense(128, relu) → Dropout(0.3) → Dense(64, relu) → Dense(26, softmax). Compile: Adam optimiser, categorical cross-entropy. Train 50 epochs, usually reaches 97%+ accuracy. Convert to TFLite INT8: converter.optimizations = [tf.lite.Optimize.DEFAULT]. Copy the .tflite file to Pi.

  3. 3

    Real-Time Inference

    Load TFLite interpreter: interpreter = tflite.Interpreter(model_path="asl_model.tflite"). On each frame: MediaPipe extracts landmarks → normalise → reshape to (1,63) → interpreter.set_tensor → invoke → get_tensor → argmax → letter. Apply confidence threshold (0.85) and temporal stability: only accept a letter if it appears in 3 consecutive frames to prevent flickering.

  4. 4

    Text Buffer + Text-to-Speech

    Maintain a letter buffer. Stability: a letter must be held (>2 seconds) to add to buffer (prevents adding during sign transitions). Space sign (open flat hand): add space to buffer. Backspace: specific sign (thumbs-down). When a complete word is detected (space after sequence), call engine.say(word); engine.runAndWait() in a background thread so TTS doesn't block video. Display current buffer and recent words on screen.

💻 Core Code Snippet

import cv2, mediapipe as mp, numpy as np
import tflite_runtime.interpreter as tflite
import pyttsx3, threading

mp_hands = mp.solutions.hands
hands = mp_hands.Hands(max_num_hands=1, min_detection_confidence=0.7)
interpreter = tflite.Interpreter("asl_model.tflite")
interpreter.allocate_tensors()
in_idx = interpreter.get_input_details()[0]['index']
out_idx = interpreter.get_output_details()[0]['index']
engine = pyttsx3.init()
LETTERS = list("ABCDEFGHIJKLMNOPQRSTUVWXYZ")

cap = cv2.VideoCapture(0)
buffer = ""; stable_letter = ""; stable_count = 0

while True:
    ret, frame = cap.read()
    results = hands.process(cv2.cvtColor(frame, cv2.COLOR_BGR2RGB))
    if results.multi_hand_landmarks:
        lm = results.multi_hand_landmarks[0].landmark
        origin = [lm[0].x, lm[0].y, lm[0].z]
        feat = np.array([[l.x-origin[0], l.y-origin[1], l.z-origin[2]] for l in lm]).flatten()
        interpreter.set_tensor(in_idx, feat.reshape(1,-1).astype(np.float32))
        interpreter.invoke()
        pred = interpreter.get_tensor(out_idx)[0]
        idx = pred.argmax(); conf = pred[idx]
        letter = LETTERS[idx] if conf > 0.85 else ""
        if letter == stable_letter: stable_count += 1
        else: stable_letter = letter; stable_count = 0
        if stable_count == 30:  # ~2 seconds at 15fps
            buffer += letter; stable_count = 0
            if letter == " ":
                word = buffer.strip().split()[-1] if buffer.strip() else ""
                threading.Thread(target=lambda w: (engine.say(w),engine.runAndWait()), args=(word,)).start()
        cv2.imshow("ASL", frame)
    if cv2.waitKey(1)&0xFF==ord('q'): break
