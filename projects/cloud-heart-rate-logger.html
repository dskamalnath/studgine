Cloud Heart Rate Logger

A MAX30102 PPG sensor on an ESP32 reads BPM and SpO₂ every 5 seconds and uploads readings to Firebase Realtime Database. View a live BPM graph on a web dashboard from anywhere in the world.

📖 Introduction

Wearable health monitoring is one of the fastest-growing areas of IoT. This project puts all the key fundamentals together: a photoplethysmography (PPG) sensor that detects blood volume changes in the fingertip through red and infrared light absorption, an ESP32 that processes the raw AC signal to extract beat-to-beat intervals, and a Firebase backend that stores the time-series data and serves it to a web dashboard.

The MAX30102 replaces the older MAX30100 (which required a custom library workaround for 3.3 V operation) and is a drop-in upgrade — built-in 3.3 V regulator and cleaner I2C implementation. This project teaches photonics-based biometrics, real-time cloud database operations, and basic signal processing without needing an oscilloscope or DSP hardware.

🧩 Components Required

  • ❤️
    MAX30102 PPG Sensor Module — I2C (@ 0x57); red (660 nm) + IR (880 nm) LEDs; built-in temperature sensor
  • 📡
    ESP32 Dev Board — Dual-core, Wi-Fi + BT; runs signal processing + Firebase HTTP calls concurrently
  • 🖥️
    0.96″ SSD1306 OLED — Local BPM and SpO₂ display without needing a phone or laptop nearby
  • 🔋
    LiPo 1000 mAh + TP4056 — Wearable battery pack; USB-C charging; 6–8 h runtime

⚙️ How It Works

  1. 1

    MAX30102 PPG Signal Processing

    The chip outputs raw 18-bit ADC values for Red and IR channels at up to 3200 Hz. Use the SparkFun MAX3010x library's built-in getHeartRate() and getSpO2() functions, which implement a PAN-TOMPKINS-style peak detector and the standard Beer-Lambert SpO₂ formula. Ensure finger is placed flat — even slight motion causes artefacts. Validate BPM is in human physiological range (40–200 BPM) before accepting.

  2. 2

    Firebase Realtime Database Upload

    Use FirebaseESP32 library. Authenticate with database secret or Google service account. Store readings at /heartrate/<unix_timestamp>: {bpm: 72, spo2: 98, temp: 36.5}. Set .write rule to authenticated only. Read back last 60 readings via orderByKey().limitToLast(60) for graph rendering. Firebase REST API works directly over HTTPS without SDK if you prefer lightweight WiFiClientSecure.

  3. 3

    Web Dashboard (HTML + Firebase JS SDK)

    Create a single-page HTML file hosted on Firebase Hosting or GitHub Pages. Use Firebase JS SDK to subscribe to real-time updates: ref.on("value", snap => chart.update(snap.val())). Chart.js renders the live BPM line graph, updating as new data arrives. Add a threshold line at 100 BPM (tachycardia alert). The entire dashboard is one 50-line HTML file — no backend server required.

  4. 4

    Abnormality Detection & Alert

    On-device: if BPM > 100 or SpO₂ < 94% for 3 consecutive readings, trigger a local alarm (OLED warning + buzzer) AND send a Telegram bot alert via HTTP POST. Implement running average over 5 readings to prevent single noisy readings triggering false alarms. Add a "calibration hold" — ignore first 15 seconds after finger placement while the AGC (auto-gain control) settles.

💻 Core Code Snippet

#include "MAX30105.h"
#include "heartRate.h"
#include <FirebaseESP32.h>

MAX30105 sensor; FirebaseData fbData;
FirebaseConfig fbConfig; FirebaseAuth fbAuth;

const byte RATE_SIZE = 4;
byte rates[RATE_SIZE]; byte rateSpot = 0;
long lastBeat = 0; float beatsPerMinute; int beatAvg;

void setup() {
  sensor.begin(Wire, I2C_SPEED_FAST);
  sensor.setup();
  fbConfig.database_url = "https://your-project.firebaseio.com/";
  fbConfig.signer.tokens.legacy_token = "YOUR_SECRET";
  Firebase.begin(&fbConfig, &fbAuth);
}

void loop() {
  long irValue = sensor.getIR();
  if(checkForBeat(irValue)) {
    long delta = millis() - lastBeat; lastBeat = millis();
    beatsPerMinute = 60 / (delta / 1000.0);
    if(beatsPerMinute > 20 && beatsPerMinute < 255) {
      rates[rateSpot++ % RATE_SIZE] = (byte)beatsPerMinute;
      beatAvg = 0; for(byte x : rates) beatAvg += x;
      beatAvg /= RATE_SIZE;
    }
  }
  // Upload to Firebase every 5s
  static unsigned long lastUpload = 0;
  if(millis() - lastUpload > 5000) {
    Firebase.setInt(fbData, "/heartrate/bpm", beatAvg);
    lastUpload = millis();
  }
}
