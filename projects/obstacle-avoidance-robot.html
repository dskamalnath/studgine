Obstacle Avoidance Robot

Build a fully autonomous two-wheel robot that scans for obstacles with an HC-SR04 ultrasonic sensor mounted on a micro servo. When a wall is detected within 20 cm the robot stops, sweeps left and right to find the clearest path, then steers around it — all without human input.

📖 Introduction

The HC-SR04 ultrasonic sensor emits a 40 kHz burst and times the echo return. Distance in centimetres is calculated as duration / 58. Mounted on a 9g micro servo it can rotate from 30° to 150° to sweep the space in front of the robot before making a steering decision.

The L298N dual H-bridge module drives two DC motors independently at variable speed via PWM signals. Setting one motor forward while braking the other turns the robot on the spot. This differential drive approach eliminates the need for a steering servo on the chassis, keeping the mechanical build simple.

🤡 Components Required

  • 🧠
    Arduino Uno R3 — Motor PWM on D5, D6, D9, D10. Servo on D3. Ultrasonic TRIG D7, ECHO D8.
  • 🔌
    HC-SR04 Ultrasonic Sensor — Range 2–400 cm, ±3 mm accuracy. VCC to 5V; TRIG and ECHO to digital pins.
  • 🔧
    9g Micro Servo (SG90) — Mounts HC-SR04 for 120° sweep. PWM signal on D3. Powered from Arduino 5V (servo draws <200 mA stall).
  • ⚡
    L298N Dual H-Bridge Motor Driver — Up to 2A per channel. ENA / ENB to PWM pins for speed control; IN1–IN4 for direction.
  • 🚗
    2 × DC Gearbox Motors + 2WD Chassis Kit — Typical ~200 RPM at 6V. Chassis kits include motors, wheels, and caster (~$8 on AliExpress).
  • 🔋
    2× 18650 Li-ion cells (7.4V) or 6× AA battery holder (9V) — Powers L298N motor supply. The L298N 5V onboard regulator then powers the Arduino via its Vin pin.

⚙ How It Works

  1. 1

    Ultrasonic Distance Measurement

    measureCm() pulls TRIG HIGH for 10 µs, then measures the ECHO pulse width with pulseIn(ECHO, HIGH, 30000). Dividing by 58 gives distance in centimetres. Three readings are averaged every 50 ms to filter noise. If all three fail within the 30 ms timeout, the function returns 0 (treated as “obstacle very close”).

  2. 2

    Free-Drive & Stop Logic

    In the main loop, if the front distance exceeds 20 cm, both motors run forward at 180/255 PWM. When distance drops below 20 cm, both motors stop immediately and the state machine enters SCAN mode.

  3. 3

    Servo Scan & Path Selection

    The servo sweeps to 30° (left) and 150° (right), measuring distance at each. A 300 ms settle delay after each servo move prevents vibration from corrupting the sonar reading. The side with the greater clearance is selected. If both sides are blocked (< 15 cm), the robot backs up 600 ms and scans again.

  4. 4

    Differential Drive Turn

    To turn right: left motor forward at full speed, right motor reversed at full speed for 350 ms. To turn left: right motor forward, left motor reversed for 350 ms. After the turn the servo returns to 90° (centre) and free-drive resumes. Turn duration is tuned to achieve ~90° rotation on typical carpet — adjust if on a slippery surface.

💻 Core Code Snippet

#include <Servo.h>

// ── Pin map ──────────────────────────────────────────
#define TRIG_PIN  7
#define ECHO_PIN  8
#define SERVO_PIN 3
#define ENA 5  // Left motor speed
#define IN1 4  // Left motor dir A
#define IN2 2  // Left motor dir B
#define ENB 6  // Right motor speed
#define IN3 9  // Right motor dir A
#define IN4 10 // Right motor dir B

#define AVOID_CM   20    // Obstacle threshold (cm)
#define DRIVE_SPD  180   // Motor PWM 0-255

Servo scanner;

long measureCm() {
  digitalWrite(TRIG_PIN, LOW);  delayMicroseconds(2);
  digitalWrite(TRIG_PIN, HIGH); delayMicroseconds(10);
  digitalWrite(TRIG_PIN, LOW);
  long d = pulseIn(ECHO_PIN, HIGH, 30000);
  return d ? d / 58 : 0;
}

void motorCmd(int la, int lb, int ra, int rb, int spd) {
  analogWrite(ENA, spd); analogWrite(ENB, spd);
  digitalWrite(IN1, la); digitalWrite(IN2, lb);
  digitalWrite(IN3, ra); digitalWrite(IN4, rb);
}

void stopMotors()           { motorCmd(0,0,0,0,0); }
void driveForward(int spd)  { motorCmd(1,0,1,0,spd); }
void turnLeft(int ms)       { motorCmd(0,1,1,0,DRIVE_SPD); delay(ms); stopMotors(); }
void turnRight(int ms)      { motorCmd(1,0,0,1,DRIVE_SPD); delay(ms); stopMotors(); }

void setup() {
  pinMode(TRIG_PIN, OUTPUT);
  pinMode(ECHO_PIN, INPUT);
  for (int p : {ENA,IN1,IN2,ENB,IN3,IN4}) pinMode(p, OUTPUT);
  scanner.attach(SERVO_PIN);
  scanner.write(90);
  delay(600);
}

void loop() {
  long dist = measureCm();

  if (dist == 0 || dist < AVOID_CM) {
    stopMotors();

    // Scan left and right
    scanner.write(30);  delay(300);
    long leftDist  = measureCm();
    scanner.write(150); delay(300);
    long rightDist = measureCm();
    scanner.write(90);  delay(200);

    if (leftDist < 15 && rightDist < 15) {
      motorCmd(0,1,0,1,DRIVE_SPD); delay(600); // back up
      stopMotors();
    } else if (leftDist > rightDist) {
      turnLeft(350);
    } else {
      turnRight(350);
    }
  } else {
    driveForward(DRIVE_SPD);
  }
  delay(50);
}
