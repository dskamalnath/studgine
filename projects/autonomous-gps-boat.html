Autonomous GPS Boat

A waterproofed Raspberry Pi boat platform autonomously navigates pre-programmed GPS waypoints using bearing-hold PID and differential brushless motor thrust vector control — all in a sealed pelican-case hull.

📖 Introduction

Autonomous surface vehicles (ASVs) combine GPS navigation, motor control, and waterproof engineering. This project builds a twin-motor catamaran platform with a Raspberry Pi as the "autopilot brain". The NEO-6M GPS module provides position at 5 Hz; the Pi computes bearing to next waypoint using the haversine formula and drives a PID loop to keep the true bearing locked, with differential thrust on two brushless motors providing yaw control.

The vessel's electronics live in a sealed IP-rated enclosure with cable glands for motor power and GPS antenna. RC override via a standard RC receiver lets you regain manual control at any time — critical for safe operation near shores and obstacles.

🧩 Components Required

  • 📡
    NEO-6M GPS Module — 5 Hz update rate; 2.5m horizontal accuracy; UART NMEA output; connected to Pi UART (/dev/ttyAMA0)
  • 🧠
    Raspberry Pi 4 (2 GB) + Pelican Case — Python autopilot; IP-67 waterproof Pelican 1120 case; cable glands for motor ESC wires
  • ⚡
    2× 30A Brushless ESCs — 30A BEC ESC; controlled via pigpio PPM signal (1000–2000µs); differential thrust = yaw control
  • 🔩
    2× A2212 1000KV Brushless Motors + Props — Twin-motor catamaran; 70mm propellers; 3S LiPo 2200mAh for ~45 min runtime
  • 🕹️
    FlySky FS-iA6B RC Receiver — RC override channel; autopilot monitors CH5 to switch between AUTO and MANUAL modes

⚙️ How It Works

  1. 1

    GPS Position Parsing

    Parse NMEA GPRMC sentences from NEO-6M on UART: serial.Serial('/dev/ttyAMA0', 9600). Extract latitude/longitude as decimal degrees. Use pynmea2 library: lat = msg.latitude; lon = msg.longitude. Compute bearing to target waypoint using haversine: dlon = lon2-lon1; y = sin(dlon)*cos(lat2); x = cos(lat1)*sin(lat2)-sin(lat1)*cos(lat2)*cos(dlon); bearing = atan2(y,x) in degrees. Compute Euclidean distance: d = sqrt((Δlat*111320)² + (Δlon*111320*cos(lat))²) in metres. Waypoint reached when d < 1.5m.

  2. 2

    Compass Heading + Yaw Error

    A QMC5883L I²C magnetometer provides compass heading. Apply hard-iron calibration: capture readings in a circle, subtract ellipse centre offsets. Compute yaw error: error = target_bearing - compass_heading. Wrap to ±180°: error = ((error+180)%360)-180. This normalized error feeds the PID controller.

  3. 3

    Bearing-Hold PID → Differential Thrust

    PID controller: output = Kp*error + Ki*integral + Kd*(error-prev_error). Tuned values: Kp=0.8, Ki=0.05, Kd=0.15. Convert to motor commands: left_throttle = base_throttle - pid_output, right_throttle = base_throttle + pid_output. Clamp to [0,100]%. Send PPM pulses via pigpio: pi.set_servo_pulsewidth(ESC_L, int(1000+left_throttle*10)). At yaw error > 20°, reduce forward thrust to allow tighter turn radius.

  4. 4

    Waypoint Mission + RC Override

    Load mission from JSON: [{"lat":51.5074,"lon":-0.1278},...]. Advance waypoint index when arrived. Monitor RC CH5 (override channel) via pigpio callback: if pulse > 1600µs, switch to MANUAL and pass RC throttle/yaw directly to ESCs. Log GPS track to CSV for post-flight analysis. Heartbeat LED blinks every second in AUTO mode, solid in MANUAL mode.

💻 Core Code Snippet

import pigpio, serial, pynmea2, json, math, time

pi = pigpio.pi()
ESC_L, ESC_R = 18, 19  # GPIO pins
gps = serial.Serial('/dev/ttyAMA0', 9600, timeout=1)

waypoints = [{"lat":51.5074,"lon":-0.1278},{"lat":51.5076,"lon":-0.1270}]
wp_idx = 0

def haversine_bearing(lat1,lon1,lat2,lon2):
    lat1,lon1,lat2,lon2 = map(math.radians,[lat1,lon1,lat2,lon2])
    dlon = lon2-lon1
    y = math.sin(dlon)*math.cos(lat2)
    x = math.cos(lat1)*math.sin(lat2)-math.sin(lat1)*math.cos(lat2)*math.cos(dlon)
    return math.degrees(math.atan2(y,x))

def set_motor(pin, throttle):  # throttle 0–100
    pw = int(1000 + throttle*10)
    pi.set_servo_pulsewidth(pin, pw)

# PID state
Kp,Ki,Kd = 0.8,0.05,0.15
integral=0.0; prev_error=0.0; BASE_THROTTLE=50

while wp_idx < len(waypoints):
    line = gps.readline().decode('ascii','ignore')
    if line.startswith('$GPRMC'):
        msg = pynmea2.parse(line)
        if msg.status=='A':
            bearing = haversine_bearing(msg.latitude,msg.longitude,
                          waypoints[wp_idx]['lat'],waypoints[wp_idx]['lon'])
            # PID on yaw error (needs compass heading for real heading)
            error = ((bearing - COMPASS_HEADING + 180)%360)-180
            integral += error; derivative = error-prev_error
            pid = Kp*error + Ki*integral + Kd*derivative
            prev_error = error
            set_motor(ESC_L, max(0,min(100, BASE_THROTTLE - pid)))
            set_motor(ESC_R, max(0,min(100, BASE_THROTTLE + pid)))
