Smart Pill Dispenser

A rotating 7-compartment drum dispenses the correct medication at DS3231 RTC-programmed times with buzzer alert, LCD reminder countdown, and EEPROM dose history logging.

📖 Introduction

Non-adherence to medication schedules is one of the most significant challenges in healthcare — especially for elderly patients managing multiple drugs. This dispenser automates that problem: load each week's compartments once, and the machine handles reminders and dispensing automatically.

A 3D-printed drum has 7 or 14 compartments (one per day, or two per day for 7 days). A stepper or high-torque servo indexes the drum to the correct compartment at each programmed alarm time. The DS3231's two built-in alarm outputs can wake the Arduino from sleep, saving significant battery power between doses. A check switch verifies the compartment is empty (pill taken) vs. still full (missed dose) and logs accordingly.

🧩 Components Required

  • 🎛️
    Arduino Uno / Nano — Main controller
  • ⏰
    DS3231 RTC Module — I2C RTC with 2 hardware alarms and coin cell
  • ⚙️
    MG996R Servo Motor — High torque for reliable drum indexing
  • 📟
    16×2 I2C LCD — Next dose time, compartment number, missed count
  • 🔊
    Active Buzzer — Reminder alert until pill is taken
  • 🖨️
    3D Printed Drum (7 compartments) — Print with PLA; hex centre socket for servo horn

⚙️ How It Works

  1. 1

    Set DS3231 Alarms

    Use RTClib to set Alarm 1 for each daily dose time (e.g., 08:00, 13:00, 20:00). DS3231 pulls SQW/INT pin LOW when the alarm fires, waking the Arduino from power-down sleep via INT0. This extends battery life to weeks between charges.

  2. 2

    Index the Drum

    Track current compartment index (0–6) in EEPROM. On alarm fire, call indexDrum(currentCompartment + 1). The servo rotates to a target angle = compartment × (360/7) = 51.4° per step. Use position feedback (potentiometer on servo shaft) or count precise PWM pulses for accuracy.

  3. 3

    Alert and Wait for Acknowledgement

    Sound the buzzer in a pattern (3 beeps every 30 s) and display "TAKE PILL NOW" on the LCD. An IR beam-break or micro-switch detects when the compartment is accessed. If no detection within 10 minutes, mark as MISSED in EEPROM and send alert via serial/GSM.

  4. 4

    EEPROM Dose Log

    Each dose event (taken / missed / skipped) is logged as a struct: timestamp (Unix epoch from RTC), compartment index, status. Use a 256-byte ring buffer in EEPROM for the last 30 days of history. Read the log through the Serial Monitor menu.

💻 Core Code Snippet

#include <RTClib.h>
#include <Servo.h>
#include <LiquidCrystal_I2C.h>
#include <avr/sleep.h>

RTC_DS3231 rtc;
Servo drum;
LiquidCrystal_I2C lcd(0x27,16,2);

const int BUZZER=7, SERVO_PIN=9, INT_PIN=2;
int compartment = 0;  // 0..6 (7 days)
float COMPARTMENT_ANGLE[7] = {0,51.4,102.8,154.3,205.7,257.1,308.6};

void indexDrum(int slot) {
  drum.attach(SERVO_PIN);
  drum.write((int)COMPARTMENT_ANGLE[slot % 7]);
  delay(1200);
  drum.detach();
  compartment = slot % 7;
  EEPROM.write(0, compartment);
}

void setNextAlarm(int hour, int minute) {
  rtc.clearAlarm(1);
  rtc.setAlarm1(DateTime(2025,1,1,hour,minute,0),
                DS3231_A1_Hour);      // match hour:minute:00
}

void IRAM_ATTR alarmISR() { /* wake up */ }

void dispenseDose() {
  indexDrum(compartment + 1);
  lcd.clear(); lcd.print("TAKE PILL NOW");
  lcd.setCursor(0,1); lcd.print("Slot " + String(compartment+1) + "/7");
  for (int i=0; i<6; i++) {
    digitalWrite(BUZZER, HIGH); delay(200);
    digitalWrite(BUZZER, LOW);  delay(800);
  }
}

void setup() {
  rtc.begin();
  drum.attach(SERVO_PIN);
  lcd.init(); lcd.backlight();
  pinMode(INT_PIN, INPUT_PULLUP);
  attachInterrupt(digitalPinToInterrupt(INT_PIN), alarmISR, FALLING);
  setNextAlarm(8, 0);  // 08:00 first dose
}

void loop() {
  if (rtc.alarmFired(1)) {
    dispenseDose();
    rtc.clearAlarm(1);
    setNextAlarm(20, 0);  // next dose 20:00
  }
  // Sleep between doses...
  set_sleep_mode(SLEEP_MODE_PWR_DOWN);
  sleep_enable(); sleep_cpu(); sleep_disable();
}
