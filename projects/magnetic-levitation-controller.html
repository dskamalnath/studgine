Magnetic Levitation Controller

An electromagnet (12V solenoid coil) pulls upward on a permanent magnet below it. A Hall-effect sensor measures the gap with 0.1mm resolution; a PID control loop running at 500Hz on Arduino adjusts coil current via a power MOSFET to maintain perfectly stable levitation — a beautiful demonstration of linear feedback control theory.

📖 Introduction

Magnetic levitation (maglev) is an inherently unstable system — without control, the magnet either crashes into the coil or falls away. A PID controller samples the Hall-effect sensor voltage (which correlates to gap distance) 500 times per second and drives a power MOSFET to modulate the electromagnet current, maintaining the setpoint gap. This is one of the most satisfying control theory demonstrations because you can see the result (a floating magnet) and tune the PID gains to observe under/over-damped and critically-damped responses. Kp=4.0, Ki=0.2, Kd=1.8 for typical 5mm gap with 15g magnet.

🧩 Components Required

  • 🧲
    Electromagnet Coil (12V, 1.5kg hold) — 200-turn solenoid coil; 12V 2A; 50×30mm; mount above; 5mm air gap target with 15–25g neodymium magnet below
  • 🎛️
    Hall-Effect Sensor (A1302 or SS49E) — Linear ratiometric output; 6.25mV/Gauss; 0–5V; mount inside coil aperture to measure floating magnet proximity
  • ⚡
    IRF540N N-MOSFET + Gate Driver — Logic-level MOSFET for 12V coil PWM control; flyback diode 1N5819; gate pull-down 10kΩ; Arduino PWM pin 9
  • 🧩
    Arduino Mega (or Uno) — 500Hz timer interrupt for PID; 10-bit ADC for Hall sensor; 8-bit PWM output; Serial.print for PC tuning

⚙️ How It Works

  1. 1

    Hall Sensor Calibration to Gap Distance

    Hall sensor output voltage V_hall increases as magnet approaches. Calibrate: place magnet at 2mm, 4mm, 6mm, 8mm, 10mm gaps; record ADC value at each. Fit polynomial: gap_mm = polyval(coeff, adc_value). Alternatively: gap = A / (V_hall - V_offset) (inverse relationship fits magnetic field vs distance). Store calibration coefficients in EEPROM. Setpoint: 5.0mm gap. Sensor reads 500 samples/sec via 2kHz Timer2 interrupt.

  2. 2

    PID Controller Implementation

    500Hz timer interrupt: error = gap_setpoint - gap_measured; integral += error * dt; derivative = (error - prev_error) / dt; output = Kp*error + Ki*integral + Kd*derivative; prev_error = error. Clamp output [0, 255]. Anti-windup: clamp integral when output saturated. Send to MOSFET: analogWrite(MOSFET_PIN, (int)output). Kp=4.0, Ki=0.2, Kd=1.8 (tuned by Ziegler-Nichols method). dt = 0.002s (500Hz).

  3. 3

    MOSFET Drive Circuit

    MOSFET gate driven by Arduino PWM + 100Ω gate resistor. Flyback diode (1N5819) across coil to absorb inductive kick. Current limiting: 1Ω sense resistor + op-amp for current feedback (optional — improves linearity by controlling current directly rather than voltage). 12V supply: separate 2A PSU, not Arduino 5V rail. Heat sink on MOSFET (1.5W dissipation at mid-PWM). Ground common: Arduino GND = PSU GND = Hall sensor GND.

  4. 4

    Tuning + Serial Plotter Visualisation

    Arduino Serial.print at 50Hz: gap, setpoint, PWM. Use Arduino Serial Plotter or Python matplotlib live plot to observe step response. Tuning procedure: start Kd=0, Ki=0, increase Kp until oscillation. Reduce Kp 50%. Add Kd until oscillations damped. Add tiny Ki for zero steady-state error. Demonstrate: load a 5g extra weight → controller adapts within 200ms. Change setpoint from 5mm to 7mm → observe transient response. Classic control theory lesson made physical.

💻 Core Code Snippet

// Arduino (C++) — 500Hz PID maglev controller
const int HALL_PIN = A0, MOSFET_PIN = 9;
const float Kp=4.0, Ki=0.2, Kd=1.8, SETPOINT=5.0, dt=0.002;
float integral=0, prev_error=0;

float hall_to_gap(int adc) {
  // Calibrated inverse relationship
  return 25.0 / (float(adc) * 4.88e-3 - 0.2);  // mm
}

ISR(TIMER2_COMPA_vect) {  // 500 Hz interrupt
  int raw = analogRead(HALL_PIN);
  float gap = hall_to_gap(raw);
  float error = SETPOINT - gap;
  integral = constrain(integral + error*dt, -50, 50);  // anti-windup
  float derivative = (error - prev_error) / dt;
  float output = Kp*error + Ki*integral + Kd*derivative;
  output = constrain(output, 0, 255);
  analogWrite(MOSFET_PIN, (int)output);
  prev_error = error;
  // Log at 50Hz
  static int log_tick = 0;
  if(++log_tick >= 10) {
    Serial.print(gap); Serial.print(",");
    Serial.print(SETPOINT); Serial.print(",");
    Serial.println((int)output);
    log_tick = 0;
  }
}
