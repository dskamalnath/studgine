Smart Parking System

Ultrasonic sensors detect occupied/vacant bays and display availability on an LCD board with a servo-operated entry barrier and slot logic.

📖 Introduction

The Smart Parking System mimics how modern malls and airports manage parking. Each bay gets an HC-SR04 ultrasonic sensor mounted at the ceiling. When a car is present (<30 cm), the bay LED turns red on the entrance display; when empty, it stays green. The Arduino counts free slots and shows the total on an LCD panel at the entrance.

A servo-operated barrier only raises if free slots > 0, preventing entry when the lot is full. You'll learn multi-sensor polling, state machine design, servo control, and I2C LCD — skills central to SCADA panels and smart building systems.

🧩 Components Required

  • 🎛️
    Arduino Uno / Mega — Mega recommended for 8+ sensors
  • 📡
    HC-SR04 Ultrasonic Sensors ×8 — One per parking bay
  • 🖥️
    16×2 I2C LCD Module — Entrance slot count display
  • ⚙️
    SG90 Servo Motor — Entry barrier arm (0°–90°)
  • 🔴
    Red + Green LEDs ×8 — Per-bay visual indicator
  • 🔌
    Resistors 330Ω + Breadboard — LED current limiting

⚙️ How It Works

  1. 1

    Poll Each Bay Sensor

    Trigger each HC-SR04 with a 10 µs pulse, then measure echo duration. Distance = duration / 58 cm. If distance < 30 cm, mark bay as occupied. Use a 3-reading debounce buffer (majority vote) to prevent false triggers from vibration.

  2. 2

    Update LED Display

    A shift register (74HC595) drives the 8 red/green LED pairs from 3 Arduino pins. Build a bitmask from the occupancy array and shift it out each loop iteration. This saves 13 Arduino GPIO pins.

  3. 3

    Count Free Slots & Update LCD

    Sum the occupancy array to get occupied count. Free = TOTAL_BAYS − occupied. Print to the I2C LCD every second: "Free: 5/8 Slots". Colour-code the LCD backlight (green if >2 free, amber if 1, red if 0).

  4. 4

    Barrier Servo Logic

    When an IR sensor at the entrance detects a waiting car AND freeSlots > 0, rotate servo to 90° (open). A second IR sensor at the exit triggers servo close after the car passes. If lot is full, barrier stays at 0° and LCD shows "FULL".

💻 Core Code Snippet

#include <LiquidCrystal_I2C.h>
#include <Servo.h>

LiquidCrystal_I2C lcd(0x27, 16, 2);
Servo barrier;

#define TOTAL_BAYS  8
#define CAR_DIST    30   // cm threshold for car detected

int trigPins[8] = {2, 4, 6, 8, 10, 12, A0, A2};
int echoPins[8] = {3, 5, 7, 9, 11, 13, A1, A3};
bool occupied[8];
int freeSlots;

long getDistance(int trig, int echo) {
  digitalWrite(trig, HIGH); delayMicroseconds(10); digitalWrite(trig, LOW);
  return pulseIn(echo, HIGH) / 58;
}

void loop() {
  freeSlots = 0;
  for (int i = 0; i < TOTAL_BAYS; i++) {
    occupied[i] = (getDistance(trigPins[i], echoPins[i]) < CAR_DIST);
    if (!occupied[i]) freeSlots++;
    digitalWrite(greenLED[i], !occupied[i]);
    digitalWrite(redLED[i],    occupied[i]);
  }
  lcd.setCursor(0,0); lcd.print("Free: ");
  lcd.print(freeSlots); lcd.print("/"); lcd.print(TOTAL_BAYS);
  lcd.setCursor(0,1); lcd.print(freeSlots == 0 ? "LOT IS FULL     " : "  Welcome!      ");

  // Barrier: open if free and car waiting at entrance
  if (freeSlots > 0 && digitalRead(IR_ENTRY) == LOW)
    barrier.write(90);  // raise
  if (digitalRead(IR_EXIT) == LOW)
    barrier.write(0);   // lower after car exits
  delay(500);
}
