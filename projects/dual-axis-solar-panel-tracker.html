Dual-Axis Solar Panel Tracker

Four LDR sensors arranged in a quad pattern drive pan-tilt servo gimbals to continuously orient a solar panel toward the sun. Logs watt-hour harvest improvement vs. a fixed-panel baseline over several days — typically 30–50% more energy.

📖 Introduction

A dual-axis solar tracker keeps the panel perpendicular to the sun throughout the day, capturing significantly more irradiance than a fixed-tilt panel. The sensing principle uses four LDRs (Light Dependent Resistors), one in each quadrant of a divided aperture. When the sun is not directly facing the panel, one pair of LDRs reads higher resistance than the opposite pair. A P controller derives pan and tilt correction signals and drives two MG996R servos (pan: 0–270°, tilt: 0–90°). Power from the tracked vs. fixed panel is measured with INA219 current sensors and logged to CSV.

🧩 Components Required

  • 💡
    4× LDR + Voltage Dividers — GL5528 LDRs; 10kΩ dividers; separated by cross-shaped baffle on sensor mount; read via MCP3208 ADC
  • ⚙️
    2× MG996R Servos (Pan + Tilt) — Pan: 180° range; Tilt: 0–90° elevation; both on PCA9685 PWM driver; driven by RPi or Arduino
  • ☀️
    5W Polycrystalline Solar Panel — Tracker panel + identical fixed reference panel; compare output via 2× INA219 current sensors
  • 📊
    INA219 Current/Voltage Sensor — I²C; measures panel voltage + current; compute instantaneous watt output; log to CSV every 60s

⚙️ How It Works

  1. 1

    LDR Quad Sensor Differential

    Read all four LDRs via MCP3208 ADC channels 0–3. Divide into pairs: Top-Left (TL), Top-Right (TR), Bottom-Left (BL), Bottom-Right (BR). Pan error: pan_err = (TL + BL) - (TR + BR). Tilt error: tilt_err = (TL + TR) - (BL + BR). When sun is centred, all four values match and errors → 0. A cross-shaped cardboard baffle casts asymmetric shadows to amplify the differential signal.

  2. 2

    P-Controller Servo Update

    Apply proportional controller: pan_angle += Kp * pan_err, tilt_angle += Kp * tilt_err. Kp = 0.01 (tune to avoid hunting). Clamp: pan to [0°, 180°], tilt to [10°, 80°]. Update PCA9685 servo PWM every 500ms. Add dead-band: skip update if |err| < 50 ADC counts to prevent jitter in bright overcast (uniform diffuse light). Night mode: return to home position (pan=90°, tilt=20°) when irradiance drops below threshold.

  3. 3

    Power Harvesting Measurement

    INA219 (I²C addr 0x40) measures tracked panel: bus_voltage = ina.bus_voltage; current_mA = ina.current; power_mW = bus_voltage * current_mA. INA219 at 0x41 measures fixed reference panel. Log every 60s to CSV: timestamp, tracked_mW, fixed_mW, pan, tilt. Compute cumulative harvest ratio: sum(tracked_Wh) / sum(fixed_Wh). Expected gain: 30–50% more energy capture on clear sunny days.

  4. 4

    Data Visualisation

    Plot logged CSV with matplotlib: two overlaid power curves (tracked vs. fixed) across the day. Area under curve = total Wh harvested. Annotate crossover times. Optional: push data to Grafana via MQTT for live web dashboard. Cloud option: log to Google Sheets via gspread API for long-term multi-day comparison — very compelling project presentation data.

💻 Core Code Snippet

import time, csv
from adafruit_pca9685 import PCA9685
from ina219 import INA219
import spidev

spi = spidev.SpiDev(); spi.open(0,0); spi.max_speed_hz=500000
ina_track = INA219(0.1, address=0x40)
ina_fixed = INA219(0.1, address=0x41)
ina_track.configure(); ina_fixed.configure()

pan, tilt = 90.0, 45.0
Kp = 0.008; DEAD_BAND = 50

def read_ldr(ch):
    r = spi.xfer2([0x06|(ch>>2),(ch&3)<<6,0])
    return ((r[1]&0x0F)<<8)|r[2]

def set_servo(pca, ch, angle):
    pulse = int(1000 + angle*(1000/180))
    duty = int((pulse/20000)*0xFFFF)
    pca.channels[ch].duty_cycle = duty

import busio, board
i2c = busio.I2C(board.SCL, board.SDA)
pca = PCA9685(i2c); pca.frequency = 50

with open('harvest_log.csv','a') as f:
    writer = csv.writer(f)
    while True:
        tl,tr,bl,br = read_ldr(0),read_ldr(1),read_ldr(2),read_ldr(3)
        pan_err = (tl+bl)-(tr+br)
        tilt_err = (tl+tr)-(bl+br)
        if abs(pan_err) > DEAD_BAND:
            pan = max(0, min(180, pan + Kp*pan_err))
        if abs(tilt_err) > DEAD_BAND:
            tilt = max(10, min(80, tilt + Kp*tilt_err))
        set_servo(pca, 0, pan); set_servo(pca, 1, tilt)
        tracked_mW = ina_track.power()
        fixed_mW = ina_fixed.power()
        writer.writerow([time.time(), tracked_mW, fixed_mW, pan, tilt])
        time.sleep(0.5)
