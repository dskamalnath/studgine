Fire Detector Alarm

A flame sensor detects the infrared wavelengths emitted by real fire and immediately sounds a buzzer alarm — your first life-safety electronics project.

📖 Introduction

The KY-026 flame sensor module contains an IR photodiode specifically tuned to detect light in the 760–1100 nm range — the infrared emission spectrum of a candle or open fire. It provides both a digital output (HIGH/LOW with adjustable potentiometer threshold) and an analog output proportional to IR intensity.

Using the analog output with analogRead() gives you a graduated response: you can distinguish between a distant glow and a close flame, making the alarm smarter than the simple digital mode. This is a key concept in sensor systems — analog over digital for nuanced detection.

⚙️ How It Works

  1. 1
    Wire the sensor — KY-026: VCC → 5 V, GND → GND, AO → A0, DO → D2. Buzzer + → D8, – → GND. Red LED anode via 220 Ω → D7, cathode → GND.
  2. 2
    Calibrate threshold — Open Serial Monitor. Move a lit lighter slowly toward the sensor and note where analogRead(A0) drops below ~300 (more IR = lower ADC count on this sensor). Set const int FIRE_THRESHOLD = 300;. Adjust the onboard potentiometer too until the digital output on DO changes reliably.
  3. 3
    Alarm logic — if (analogRead(A0) < FIRE_THRESHOLD) { digitalWrite(BUZZER, HIGH); digitalWrite(LED, HIGH); } else { digitalWrite(BUZZER, LOW); digitalWrite(LED, LOW); }. Add a delay(200) to debounce flicker.
  4. 4
    Add tone variation — Replace digitalWrite(BUZZER, HIGH) with tone(BUZZER_PIN, 2000, 200); to produce a beeping pattern rather than a constant tone. Use millis() to create an alternating 1000 Hz / 2000 Hz siren effect without blocking code.
