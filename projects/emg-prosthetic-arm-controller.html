EMG Prosthetic Arm Controller

Capture forearm surface EMG signals with wet electrodes and an AD8226 instrumentation amp, classify 4 hand gestures with an SVM, and drive a 3D-printed servo hand in real time — a complete bioelectronics + ML project.

📖 Introduction

Electromyography (EMG) measures the electrical signals produced when muscles contract. Surface electrodes (hydrogel snap electrodes) placed over the forearm flexor/extensor muscles capture µV-level signals that, after amplification and filtering, encode hand gestures reliably enough to control a prosthetic hand.

The AD8226 instrumentation amplifier provides the differential gain needed to extract muscle signals from the body's electrical noise floor. You're operating at the intersection of bioelectronics, signal processing, and machine learning — exactly the domain of modern prosthetics and BMI research.

🧩 Components Required

  • ⚗️
    AD8226 Instrumentation Amplifier — Differential amp; set gain with single resistor: Gain = 1 + 200k/R_G. For EMG: gain 500–1000× typical. CMRR>80dB rejects 50/60Hz mains noise
  • 🔬
    Wet Snap Electrodes (3 per channel) — AgCl hydrogel disposable electrodes. IN+, IN−: 2cm apart over muscle belly. REF: elbow bony prominence (electrical reference)
  • 🧠
    Raspberry Pi 4 — No onboard ADC; needs MCP3208 12-bit SPI ADC (3.3V, 1 MSPS) to digitise amplified EMG signal at ≥1 kHz sampling rate
  • 🖐️
    5× MG90S Servos (3D-printed hand) — InMoov or open-source prosthetic hand STL files; 5 servos per finger group; powered from separate 5V 5A supply
  • 🔌
    MCP3208 SPI ADC — 8-channel 12-bit; sample EMG at 1–2 kHz; SPI interface connects to Pi GPIO (MOSI/MISO/SCLK/CS)

⚙️ How It Works

  1. 1

    Signal Acquisition + Filtering

    ADC samples at 1 kHz. Raw EMG: 50–500 Hz signal buried in noise. Digital filter chain in Python: (1) Bandpass filter 20–450 Hz (4th-order Butterworth): scipy.signal.butter(4,[20,450],'bandpass',fs=1000). (2) Notch filter 50 Hz (or 60 Hz) to remove mains noise: scipy.signal.iirnotch(50,Q=30,fs=1000). (3) Full-wave rectify: abs(filtered). (4) Envelope: moving average 50ms window → the "muscle activation envelope".

  2. 2

    Feature Extraction

    Extract features from 200ms sliding windows with 50ms step: RMS: sqrt(mean(x**2)). MAV (Mean Absolute Value): mean(abs(x)). ZCR (Zero-Crossing Rate): count sign changes per window (frequency proxy). Waveform Length: sum(abs(diff(x))) (complexity measure). These 4 features per channel (×2 channels = 8 features) are strong enough for 4-class classification.

  3. 3

    SVM Classifier Training

    Data collection: hold each gesture (fist, open, pinch, wrist flex) for 5 seconds × 20 repetitions. Extract features → CSV. Train on PC: sklearn.svm.SVC(kernel='rbf', C=10, gamma='scale'). Test accuracy: typically 90–95% for 4 classes. Export: pickle.dump(clf, open('emg_classifier.pkl','wb')). Load on Pi and run inference on each new 200ms window. Apply majority vote over last 3 predictions for stability.

  4. 4

    Servo Hand Actuation

    Map gesture class to servo positions: GESTURE_SERVOS = {"fist":[180,180,180,180,180], "open":[0,0,0,0,0], "pinch":[180,0,180,180,180], "wrist_flex":[90,90,90,90,90]}. Use PCA9685 16-channel PWM driver (I2C) to control all 5 servos simultaneously. Ramp servos smoothly: interpolate between current and target positions over 200ms. Battery: 5V 5A USB power brick; never power servos from Pi GPIO (8V peak inrush = instant damage).

💻 Core Code Snippet

import spidev, numpy as np, pickle, time
from scipy.signal import butter, lfilter, iirnotch
import RPi.GPIO as GPIO

# MCP3208 SPI
spi = spidev.SpiDev(); spi.open(0,0); spi.max_speed_hz=1000000
def read_adc(ch): r=spi.xfer2([6|(ch>>2),(ch&3)<<6,0]); return ((r[1]&31)<<8)|r[2]

# Filters
b_bp, a_bp = butter(4, [20,450], btype='bandpass', fs=1000)
b_n, a_n   = iirnotch(50, Q=30, fs=1000)

clf = pickle.load(open("emg_classifier.pkl","rb"))

def extract_features(window):
    rms = np.sqrt(np.mean(window**2))
    mav = np.mean(np.abs(window))
    zcr = np.sum(np.diff(np.sign(window)) != 0)
    wl  = np.sum(np.abs(np.diff(window)))
    return [rms, mav, zcr, wl]

WINDOW = 200; buffer = []
while True:
    sample = read_adc(0) * 3.3 / 4096  # voltage
    buffer.append(sample)
    if len(buffer) >= WINDOW:
        sig = np.array(buffer[-WINDOW:])
        filt = lfilter(b_bp,a_bp, lfilter(b_n,a_n,sig))
        features = extract_features(filt)
        gesture = clf.predict([features])[0]
        print(f"Gesture: {gesture}")
        buffer = buffer[-100:]  # 50ms step
    time.sleep(0.001)
