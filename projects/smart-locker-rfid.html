Smart Locker with RFID

Enroll authorized MIFARE cards with an MFRC522 reader, control a 12 V solenoid lock via relay, and log every access attempt to an SD card with DS3231 timestamps.

📖 Introduction

RFID (Radio Frequency Identification) access control is everywhere — from hotel key cards to office doors. The MFRC522 is a 13.56 MHz SPI RFID reader that can read ISO/IEC 14443 Type A cards (MIFARE Classic 1K) at up to 10 cm range. It outputs a unique 4-byte UID per card, which is all you need to implement a whitelist-based access system.

This project stores up to 10 authorized card UIDs in EEPROM, drives a 12 V solenoid latch bolt via a relay, gives visual/audible feedback (green/red LED + buzzer), and logs every scan attempt to a microSD card with a DS3231 timestamp. An enroll mode (activated by holding a master card for 3 seconds) adds new cards to the EEPROM whitelist without reprogramming.

🧩 Components Required

  • 🎛️
    Arduino Uno / Nano — SPI host for MFRC522 and SD module simultaneously
  • 📡
    MFRC522 RFID Reader Module — 13.56 MHz SPI reader; SS→D10, SCK→D13, MOSI→D11, MISO→D12, RST→D9
  • 🔐
    12 V Solenoid Bolt Lock — NC solenoid; energize to unlock, spring-return to lock
  • 🔌
    5 V Relay Module — Switches 12 V solenoid from Arduino 5 V logic; 1N4007 flyback included
  • 🕐
    DS3231 RTC + MicroSD Module — Timestamp and event logging; SD CS→D4 (separate SPI CS)
  • 💳
    MIFARE Classic 1K Cards (5-pack) — Standard 13.56 MHz ISO cards; key fobs also compatible

⚙️ How It Works

  1. 1

    Enroll Master Card and Whitelist

    On first boot (EEPROM byte 0 ≠ 0xAA), enter setup mode. The first card scanned becomes the master card — its UID stored at EEPROM addr 10–13. To add a user card: hold master card for 3+ seconds → green LED blinks → scan new card → UID written to next slot (EEPROM addr 20, 24, 28...). Up to 10 user cards (40 bytes).

  2. 2

    Scan → UID Comparison → Unlock

    In normal operation, poll RFID every 100 ms with mfrc522.PICC_IsNewCardPresent() and mfrc522.PICC_ReadCardSerial(). Extract the 4-byte UID from mfrc522.uid.uidByte[] and compare against all EEPROM whitelist entries. If match: relay HIGH for 3 seconds, green LED, 1 beep. If no match: relay LOW, red LED, 3 rapid beeps.

  3. 3

    SD Card Event Logging

    After every scan, open ACCESS.CSV on the SD card and append a row: timestamp, UID_hex, GRANTED/DENIED, user_slot. Use SD in SPIFFS mode with CS on D4 (separate from RFID CS on D10). Flush after every write. On SD failure, print to Serial and flash onboard LED but don't block — access control must remain functional even without logging.

  4. 4

    Fail-Safe Design

    Use a normally-closed solenoid so the door stays locked when power fails (security-positive mode). Add a 1000 µF capacitor across the relay coil to handle inrush. An indoor emergency release button bypasses the solenoid relay directly. 3 consecutive DENY scans within 30 seconds triggers a tamper alert (rapid buzzer + Serial message).

💻 Core Code Snippet

#include <SPI.h>
#include <MFRC522.h>
#include <EEPROM.h>

#define SS_PIN  10
#define RST_PIN  9
#define RELAY    6
#define BUZZER   7

MFRC522 rfid(SS_PIN, RST_PIN);

bool uidInWhitelist(byte *uid, byte len) {
  for(int slot = 0; slot < 10; slot++) {
    int addr = 20 + slot * 4;
    bool match = true;
    for(int i = 0; i < 4; i++) {
      if(EEPROM.read(addr + i) != uid[i]) { match = false; break; }
    }
    if(match) return true;
  }
  return false;
}

void addToWhitelist(byte *uid) {
  for(int slot = 0; slot < 10; slot++) {
    int addr = 20 + slot * 4;
    if(EEPROM.read(addr) == 0xFF) {  // empty slot
      for(int i = 0; i < 4; i++) EEPROM.write(addr + i, uid[i]);
      break;
    }
  }
}

void setup() {
  SPI.begin(); rfid.PCD_Init();
  pinMode(RELAY, OUTPUT); pinMode(BUZZER, OUTPUT);
}

void loop() {
  if(!rfid.PICC_IsNewCardPresent() || !rfid.PICC_ReadCardSerial()) return;
  byte *uid = rfid.uid.uidByte;
  if(uidInWhitelist(uid, rfid.uid.size)) {
    digitalWrite(RELAY, HIGH);
    tone(BUZZER, 1000, 200);
    delay(3000);
    digitalWrite(RELAY, LOW);
  } else {
    tone(BUZZER, 300, 100); delay(150);
    tone(BUZZER, 300, 100); delay(150);
    tone(BUZZER, 300, 100);
  }
  rfid.PICC_HaltA();
}
