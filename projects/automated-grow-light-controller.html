Automated Grow Light Controller

Schedule precise light/dark photoperiods using a DS3231 RTC, compensate for ambient daylight with an LDR, and add temperature-driven fan control — giving indoor plants exactly the environment they need to thrive, automatically.

📖 Introduction

Plants respond to photoperiod — the daily ratio of light to darkness — to regulate flowering, vegetative growth, and dormancy. Getting this ratio right is critical for indoor cultivation. Too little light and plants etiolate; too much and flowering is inhibited. This project automates the grow light schedule with exact RTC-based timing, dims the light on bright sunny days (using an LDR to avoid wasting electricity), and keeps the canopy temperature in the optimal 22–28°C range by switching a cooling fan via relay.

Optional PWM dimming allows the controller to simulate dawn/dusk ramps, reducing plant stress from abrupt light transitions — a feature usually only found on commercial grow controllers.

🧩 Components Required

  • 💡
    LED Grow Light Panel (12 V, 20–45 W) — Full-spectrum or red+blue blended; supports PWM dimming via MOSFET
  • ⏰
    DS3231 RTC Module — Battery-backed; schedules light-on and light-off times daily
  • ☀️
    LDR (GL5528) + 10 kΩ Resistor — Ambient light sensing for daylight compensation
  • 🌡️
    DHT22 or DS18B20 Sensor — Canopy temperature and humidity for fan control logic
  • 🔌
    IRF540N MOSFET + 4-channel Relay — MOSFET for LED PWM dimming; relay for fan and water pump
  • 🔧
    Arduino Uno + 20×4 LCD (I2C) — Controller + display showing current time, phase, light status, and temperature

⚙️ How It Works

  1. 1

    RTC-Based Photoperiod Scheduling

    Read current time from DS3231 each second: DateTime now = rtc.now();. Compare current hour+minute against user-configured ON_TIME and OFF_TIME (stored in EEPROM, settable via serial menu or physical buttons). If within light window: activate grow light. Support both long-day (18h/6h for vegetative) and short-day (12h/12h for flowering) modes selectable by a switch.

  2. 2

    Ambient Light Compensation

    Read LDR ADC value. Map to ambient lux estimate. If ambient lux > 5000 lux (bright sunny day): reduce LED PWM proportionally (map(lux, 0, 15000, 255, 80)) — less artificial light needed. If ambient lux > 15000 lux: turn off grow light entirely (natural light sufficient). This prevents over-lighting and saves energy during daylight hours. Override available for cloudy days via button.

  3. 3

    Dawn/Dusk Ramp Simulation

    In the 15 minutes before scheduled light-on: gradually ramp PWM from 0% to 100% in 1% steps every 9 s. Reverse at light-off time: ramp down over 15 minutes. This mimics natural sunrise/sunset, reducing plant stress from abrupt light changes and mimicking the photo-oxidative responses that natural photoperiods trigger. Implement as a state machine: RAMP_UP → FULL → RAMP_DOWN → OFF.

  4. 4

    Temperature & Fan Control

    Read DHT22 every 30 s. If canopy temp > 28°C: activate cooling fan relay. If temp > 32°C: reduce LED brightness by 20% and trigger alarm. If temp < 20°C: turn fan off; optional heating mat relay on. Log temperature history to EEPROM ring buffer (last 24 h, 1 reading/hour) for plant health trend analysis. Display min/max daily temperature on LCD.

💻 Core Code Snippet

#include <RTClib.h>
#include <DHT.h>
#define LED_PWM_PIN 9
#define FAN_RELAY 7
#define LDR_PIN A0

RTC_DS3231 rtc; DHT dht(6, DHT22);
int lightOnHour = 6, lightOffHour = 22;  // 16h vegetative
int targetPWM = 0, currentPWM = 0;

bool isLightWindow() {
  DateTime now = rtc.now();
  return (now.hour() >= lightOnHour && now.hour() < lightOffHour);
}

int ambientCompensation() {
  int ldr = analogRead(LDR_PIN);
  // ldr high = dark; map dark->full brightness
  return map(ldr, 100, 900, 80, 255);
}

void loop() {
  targetPWM = isLightWindow() ? ambientCompensation() : 0;
  // smooth ramp ±2 per cycle
  if(currentPWM < targetPWM) currentPWM = min(currentPWM+2, targetPWM);
  if(currentPWM > targetPWM) currentPWM = max(currentPWM-2, targetPWM);
  analogWrite(LED_PWM_PIN, currentPWM);

  float temp = dht.readTemperature();
  digitalWrite(FAN_RELAY, temp > 28.0 ? HIGH : LOW);
  delay(100);
}
