CV Robotic Sorting System

An overhead Pi Camera classifies coloured objects by HSV colour segmentation and contour shape recognition using OpenCV — then commands a 2-axis servo arm to pick and sort them into labelled bins. A miniaturised, programmable factory automation demonstrator with computer vision at its core.

📖 Introduction

Industrial sorting robots use machine vision to classify and route products at high speed. This project replicates the core concept at maker scale: an overhead camera detects objects on a conveyor (or flat surface), classifies them by colour (HSV masking) and shape (contour circularity/aspect ratio), computes the object centroid position, then drives a 2-DOF servo arm to the calculated X/Y position to nudge or pick the item into the correct bin. Supports 3+ colour classes and 2 shape classes (round vs. rectangular). Process time: ~200ms per object.

🧩 Components Required

  • 📷
    Raspberry Pi Camera Module v2 — 640×480 @ 30fps overhead mount; consistent lighting essential; use diffuse LED ringlight for shadow-free imaging
  • ⚙️

  • 🧠
    Raspberry Pi 4 (2 GB) — OpenCV 4.x; picamera2; adafruit PCA9685; servo IK calculation; real-time video display
  • 🏗️
    Conveyor Belt (optional) / Flat Stage — DIY belt: DC motor + timing belt + HCSR04 trigger sensor; or place items manually on white platform

⚙️ How It Works

  1. 1

    HSV Colour Segmentation

    Convert frame to HSV: hsv = cv2.cvtColor(frame, cv2.COLOR_BGR2HSV). Define colour masks: Red: H=[0-10 ∪ 170-180], S=[100-255], V=[100-255]. Green: H=[40-80]. Blue: H=[100-130]. Apply: mask = cv2.inRange(hsv, lower, upper). Morphological cleanup: cv2.morphologyEx(mask, cv2.MORPH_OPEN, kernel). Find contours: contours, _ = cv2.findContours(mask, cv2.RETR_EXTERNAL, cv2.CHAIN_APPROX_SIMPLE).

  2. 2

    Shape Classification + Centroid

    For each detected contour (area > 500 px²): compute circularity = 4π*area / perimeter². Circularity > 0.8 = circle; <0.6 = rectangle. Bounding rect aspect ratio for square vs. elongated. Compute centroid: M = cv2.moments(cnt); cx, cy = M['m10']/M['m00'], M['m01']/M['m00']. Convert pixel coords to world coords: world_x = (cx - CAM_CX) * SCALE with calibration from checkerboard.

  3. 3

    2DOF Arm Inverse Kinematics

    Given target world position (Wx, Wy), solve 2-link planar IK: c2 = (Wx²+Wy²-L1²-L2²)/(2*L1*L2); s2=√(1-c2²). θ2 = atan2(s2,c2); θ1 = atan2(Wy,Wx)-atan2(L2*s2, L1+L2*c2). Clamp to servo limits [0°–180°]. Smooth motion: interpolate from current to target angles in 10 steps at 20ms each to prevent servo jerk. Drive PCA9685: set_pulse(channel, angle_to_pulse(theta)).

  4. 4

    Bin Assignment + Object Tracking

    Bin map: Red objects → Bin 0 (left), Green → Bin 1 (centre), Blue → Bin 2 (right). Arm state machine: IDLE → DETECT → APPROACH → PICK (activate suction cup or claw servo) → PLACE at bin coords → RETURN. Track sorted items: increment bin counters. Display classification overlay on live video: draw bounding box, class label, confidence. Log each sort event to CSV with timestamp. Sort rate: ~5 objects/minute with 2-second settle time.

💻 Core Code Snippet

import cv2, numpy as np
from picamera2 import Picamera2
from adafruit_pca9685 import PCA9685
import math, busio, board

cam = Picamera2(); cam.start()
L1, L2 = 120, 80  # arm link lengths mm

COLOR_RANGES = {
    'RED':   [(0,100,100),(10,255,255)],
    'GREEN': [(40,80,80),(80,255,255)],
    'BLUE':  [(100,80,80),(130,255,255)]
}

def detect_objects(frame):
    hsv = cv2.cvtColor(frame, cv2.COLOR_BGR2HSV)
    objects = []
    for color, (lower, upper) in COLOR_RANGES.items():
        mask = cv2.inRange(hsv, np.array(lower), np.array(upper))
        kernel = cv2.getStructuringElement(cv2.MORPH_ELLIPSE,(7,7))
        mask = cv2.morphologyEx(mask, cv2.MORPH_OPEN, kernel)
        contours, _ = cv2.findContours(mask, cv2.RETR_EXTERNAL,
                                       cv2.CHAIN_APPROX_SIMPLE)
        for cnt in contours:
            if cv2.contourArea(cnt) < 500: continue
            M = cv2.moments(cnt)
            if M['m00'] == 0: continue
            cx, cy = int(M['m10']/M['m00']), int(M['m01']/M['m00'])
            # Circularity
            peri = cv2.arcLength(cnt, True)
            circ = 4*math.pi*cv2.contourArea(cnt)/(peri**2+1e-6)
            shape = 'circle' if circ > 0.75 else 'rectangle'
            objects.append({'color':color,'shape':shape,'cx':cx,'cy':cy})
    return objects

def solve_ik(wx, wy):
    """2-link planar IK → (theta1, theta2) in radians"""
    c2 = (wx**2+wy**2-L1**2-L2**2)/(2*L1*L2)
    c2 = max(-1, min(1, c2))
    s2 = math.sqrt(1-c2**2)
    t2 = math.atan2(s2, c2)
    t1 = math.atan2(wy, wx) - math.atan2(L2*s2, L1+L2*c2)
    return math.degrees(t1), math.degrees(t2)
