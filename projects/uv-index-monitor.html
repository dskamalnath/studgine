UV Index Monitor

Measure real-time UV-A and UV-B radiation with a VEML6075 sensor, compute the standardised UV index (0–11+), display risk level and safe skin exposure time on an OLED.

📖 Introduction

Ultraviolet radiation is invisible but causes cumulative skin and eye damage. The WHO-standardised UV Index (UVI) compresses the irradiance spectrum into a single 0–11+ number that quantifies erythemally-weighted UV exposure regardless of latitude or season. A UVI of 3 warrants sun protection; above 8, unprotected skin can burn in as little as 10 minutes.

The VEML6075 is a 2-photodiode I2C sensor that separately measures UVA (315–400 nm) and UVB (280–315 nm) and does hardware compensation for visible and IR light leakage. The Arduino calculates UVI from the sensor's datasheet formula, computes safe exposure time for a skin type II person, and alerts with a buzzer above UVI 6. The build is compact enough for a wearable wristband or pocket device.

🧩 Components Required

  • 🎛️
    Arduino Nano / ESP32 — I2C host; ESP32 adds BLE for phone notifications
  • ☀️
    VEML6075 UV Sensor Breakout — I2C @ 0x10; separate UVA + UVB photodiodes with compensation
  • 🖥️
    SSD1306 0.96″ OLED (128×64) — UVI, risk level, and safe exposure display
  • 🔔
    Passive Buzzer + RGB LED — Colour-coded UVI level + audio alert at UVI > 6
  • 🔋
    18650 Li-Ion + TP4056 — Rechargeable power; low quiescent current for wearable use

⚙️ How It Works

  1. 1

    Read UVA and UVB with VEML6075

    Initialize with uv.begin(). Call uv.readUVA() and uv.readUVB() every 500 ms. The library handles the integration time (100 ms) and dynamic setting (1×). Raw counts are hardware-compensated for IR leakage internally — no calibration coefficients needed from user code.

  2. 2

    Calculate UV Index

    Use the Vishay datasheet formula: UVI = (UVA_comp × UVA_resp) + (UVB_comp × UVB_resp) where UVA_resp = 0.001461 and UVB_resp = 0.002424. The VEML6075 Adafruit library wraps this in uv.readUVI(). Clamp to 0–20. Average 5 readings to remove flicker from rapidly-changing cloud cover.

  3. 3

    Determine Risk Level and Safe Time

    Map UVI to WHO risk category: 0–2 Low (green), 3–5 Moderate (yellow), 6–7 High (orange), 8–10 Very High (red), 11+ Extreme (violet). Compute safe sunburn time for skin type II using: minutes = 200 / (MED_factor × UVI) where MED_factor ≈ 0.06 for type II. Display "Safe: XX min — USE SPF50" for High+.

  4. 4

    Alert and Log

    When UVI exceeds 6, pulse the buzzer twice every 30 seconds and set RGB LED to orange. At UVI 11+, sound a 3-beep alarm and lock the display to "EXTREME — SEEK SHADE" until UVI drops below 8. Optional: write UVI + timestamp to SD card every 5 minutes for daily sun exposure tracking.

💻 Core Code Snippet

#include <Adafruit_VEML6075.h>
#include <Adafruit_SSD1306.h>
#include <Wire.h>

Adafruit_VEML6075 uv;
Adafruit_SSD1306 oled(128, 64, &Wire, -1);

const char* riskLabel[] = {"Low","Low","Low","Moderate","Moderate","Moderate",
                            "High","High","VHigh","VHigh","VHigh","Extreme"};
const uint16_t colors[]  = {0x07E0,0x07E0,0x07E0,0xFFE0,0xFFE0,0xFFE0,
                             0xFC00,0xFC00,0xF800,0xF800,0xF800,0xF81F};

void setup() {
  Wire.begin(); uv.begin(); oled.begin(SSD1306_SWITCHCAPVCC, 0x3C);
  oled.clearDisplay(); oled.display();
}

void loop() {
  float uvi = uv.readUVI();
  int idx = constrain((int)uvi, 0, 11);
  // Safe exposure for skin type II (MED = 200, MEF = 0.06)
  int safeMin = (uvi > 0) ? (int)(200.0 / (0.06 * uvi)) : 999;

  oled.clearDisplay();
  oled.setTextSize(2); oled.setCursor(0,0);
  oled.print("UVI: "); oled.println(uvi, 1);
  oled.setTextSize(1); oled.setCursor(0,20);
  oled.print("Risk: "); oled.println(riskLabel[idx]);
  oled.print("Safe: "); oled.print(safeMin); oled.println(" min");
  if(uvi >= 6) { oled.println(">>> USE SPF50 <<<"); }
  oled.display();

  if(uvi >= 6) tone(8, 880, 200);
  delay(500);
}
