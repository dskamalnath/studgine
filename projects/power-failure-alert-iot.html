📖 Introduction

A power failure IoT alerter is deceptively simple in concept — detect that AC power has been cut, then fire off a network notification — but it has one critical constraint: the ESP8266 must successfully send the alert before its backup power runs out. This project uses a 1000 µF bulk capacitor and/or a small Li-Po cell as ~30-second bridging power, giving the module time to form a Wi-Fi connection and execute one HTTPS POST.

AC presence is detected by a 5V rectified sense circuit (miniature transformer-free resistor divider from the 5V USB supply): when the mains adapter is plugged in, GPIO pin reads HIGH; when power fails, GPIO reads LOW immediately. The ESP wakes, connects to saved Wi-Fi, sends a Telegram message (or IFTTT webhook), then goes back to deep sleep if power is still absent — saving the remaining battery for a confirmed reconnection alert.

🧩 Components Required

  • 📡
    ESP8266 NodeMCU v3 — Wi-Fi, HTTPS client, deep-sleep GPIO16 wake
  • ⚡
    HLK-PM01 5V/1A Mains Module — Isolated 230/110V → 5V. DO NOT use home-brew mains circuits
  • 🔋
    500 mAh Li-Po + TP4056 BMS — UPS power for ~25–30 s at full Wi-Fi TX power
  • 🔀
    P-Channel MOSFET (IRF9540N) — Automatic switchover: HLK primary, Li-Po when AC fails
  • 🔌
    1N5819 Schottky Diodes (2×) — OR-ing diodes to prevent backfeed from mains to battery
  • 📱
    Telegram Bot or IFTTT Webhook — Push notification delivery service

⚙️ How It Works

  1. 1

    Detect AC Presence

    The HLK-PM01 output charges a 220 µF capacitor through a 4.7kΩ resistor, holding GPIO14 HIGH via a 10kΩ pull-up. When mains is cut, this node discharges through the pull-down divider (10kΩ to GND) within ≈ 2 ms. The ESP polls this pin every 500 ms — if it reads LOW for 2 consecutive samples, it declares a power failure (debounce).

  2. 2

    Use WiFiManager for Zero-Config Setup

    On first boot, connect to "PowerAlert-Setup" AP (192.168.4.1), enter your Wi-Fi SSID/password and Telegram chat ID via a web form. WiFiManager saves credentials to EEPROM. On all subsequent boots, ESP connects silently in <3 seconds to saved network — essential as the UPS power window is only ~30 s.

  3. 3

    Send Telegram Notification via HTTPS

    Use WiFiClientSecure + the Telegram Bot API: GET https://api.telegram.org/bot{TOKEN}/sendMessage?chat_id={CHAT_ID}&text=⚠️POWER+FAILURE. Use fingerprint verification OR client.setInsecure() for simplicity in a local network. Include timestamp from millis() and a configurable device name. Send reconnection notice on the same path when AC sense pin returns HIGH.

  4. 4

    Deep Sleep When AC Is Present

    When AC is live, run ESP.deepSleep(30e6) (30 s), wake via GPIO16-RST jumper, re-check pin, sleep again. This reduces average current from 70 mA (active) to <20 µA (deep sleep) — extending Li-Po standby life to weeks. On failure detection, skip the sleep, connect Wi-Fi, alert, then enter indefinite sleep with a 60 s watchdog for reconnect retry.

💻 Core Code Snippet

#include <ESP8266WiFi.h>
#include <WiFiClientSecure.h>
#include <EEPROM.h>

#define AC_SENSE    14  // GPIO14 (D5)
#define TOKEN       "YOUR_BOT_TOKEN"
#define CHAT_ID     "YOUR_CHAT_ID"

bool acWasPresent = true;

void sendTelegram(String msg) {
  WiFiClientSecure client;
  client.setInsecure();
  if (client.connect("api.telegram.org", 443)) {
    String url = "/bot" + String(TOKEN) + "/sendMessage?chat_id=" + CHAT_ID + "&text=" + msg;
    client.print("GET " + url + " HTTP/1.1\r\nHost: api.telegram.org\r\nConnection: close\r\n\r\n");
    delay(500);
  }
}

void setup() {
  pinMode(AC_SENSE, INPUT_PULLUP);
  bool acNow = digitalRead(AC_SENSE);

  WiFi.begin("YOUR_SSID", "YOUR_PASS");
  int t = 0;
  while (WiFi.status() != WL_CONNECTED && t++ < 20) delay(500);

  if (WiFi.status() == WL_CONNECTED) {
    if (!acNow) {
      sendTelegram("⚠️+POWER+FAILURE+detected");
    } else {
      sendTelegram("✅+Power+RESTORED");
    }
  }
  // Sleep and retry in 60 s regardless
  ESP.deepSleep(!acNow ? 60e6 : 30e6);
}

void loop() {} // never reached — deep sleep restarts setup()
