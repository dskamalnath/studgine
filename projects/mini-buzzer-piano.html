📖 Introduction

The Arduino tone() function generates a square wave on any digital pin at a specified frequency. A passive buzzer (which needs a driving signal, unlike an active buzzer that beeps on its own) converts that electrical oscillation into sound. Middle C is 262 Hz, A4 is 440 Hz — standard equal temperament frequencies are fixed constants you'll learn to work with across many audio projects.

With 8 buttons wired to 8 digital pins, you create a playable one-octave piano. Each button press calls tone(buzzerPin, frequency) and releasing it calls noTone(buzzerPin). You can then encode songs as frequency + duration arrays and play them back automatically.

⚙️ How It Works

  1. 1
    Define frequencies — Create an integer array: int notes[] = {262, 294, 330, 349, 392, 440, 494, 523}; — that's C4, D4, E4, F4, G4, A4, B4, C5. Wire each button to pins 2–9 with 10 kΩ pull-down resistors.
  2. 2
    Read buttons in a loop — Use an array of pin numbers: int btnPins[] = {2,3,4,5,6,7,8,9};. In loop(), iterate with a for loop, check digitalRead(btnPins[i]), and if any are HIGH call tone(10, notes[i]); (buzzer on pin 10) then break.
  3. 3
    Handle release silence — After the loop, if no button was pressed in this iteration, call noTone(10);. This stops the sound immediately when all buttons are released, giving clean note boundaries.
  4. 4
    Play a song — Add a secondary function playSong() triggered by a 9th button. Encode a melody as a 2D array of {note, duration} pairs (e.g. Happy Birthday) and iterate through it with tone(pin, freq, dur) and a brief gap between notes.
