RGB Color Mixer

Three potentiometers independently dial in the red, green, and blue PWM channels of an RGB LED — mix any colour you want in real time.

Home › Projects › RGB Color Mixer
→

3× analogRead (0–1023) → map() → 3× analogWrite (0–255) PWM

▶ Watch on YouTube

📖 Introduction

An RGB LED contains three separate dies (Red, Green, Blue) in one package. By varying the brightness of each channel via PWM (Pulse Width Modulation) — where analogWrite(pin, 0–255) sets duty cycle — you can mix 16.7 million colours.

A 10 kΩ potentiometer connected as a voltage divider gives an analog value of 0–1023 on the ADC. The Arduino map() function rescales this to 0–255 for the PWM output. This one-line relationship is the foundation of every digital dimmer, audio volume control, and motor speed controller.

⚙️ How It Works

  1. 1
    Wire the pots — Outer legs to 5 V and GND; wiper to A0 (Red), A1 (Green), A2 (Blue). Wire RGB LED: common cathode to GND; Red pin via 220 Ω to D9 (~); Green to D10 (~); Blue to D11 (~). All three are PWM-capable pins on the Uno.
  2. 2
    Read and map — In loop(): int rVal = map(analogRead(A0), 0, 1023, 0, 255); — repeat for G and B. The map() function does linear interpolation and handles the 10-bit → 8-bit conversion in one call.
  3. 3
    Write PWM — analogWrite(9, rVal); analogWrite(10, gVal); analogWrite(11, bVal);. For a common-anode RGB LED, invert: analogWrite(9, 255 - rVal);. Add a short delay(10); to smooth readings.
  4. 4
    Add Serial output — Print the hex colour code to Serial Monitor: Serial.printf("#%02X%02X%02X\n", rVal, gVal, bVal); so you can note colours as CSS hex values. A great way to build your own colour palette.
