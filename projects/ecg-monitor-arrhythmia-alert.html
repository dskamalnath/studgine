ECG Monitor with Arrhythmia Alert

AD8232 ECG front-end digitises Lead-I cardiac signal at 360Hz via Arduino Mega. The Pan-Tompkins algorithm detects R-peaks in real-time. RR interval statistics classify three arrhythmia types: bradycardia (<50 BPM), tachycardia (>150 BPM), and irregular rhythm (RR SD >0.1s). Waveform and diagnosis log to micro-SD with timestamps; a buzzer fires on alert.

📖 Introduction

The 12-lead clinical ECG machine costs thousands of dollars, but the fundamental electrocardiography problem — detecting R-peaks and classifying rate/rhythm — can be solved with a $8 AD8232 module. This project implements the Pan-Tompkins QRS detection algorithm (1985, still gold-standard) on an Arduino Mega: bandpass filter → derivative → squaring → moving integration → adaptive threshold. Once R-peaks are located, RR interval statistics reveal the three most common arrhythmias detectable from single-lead ECG. This is an educational project — not a medical device — but the signal processing is production-grade.

⚙️ How It Works

  1. 1

    Pan-Tompkins Stage 1: Bandpass Filter

    Remove baseline wander (<0.5Hz) and muscle noise (>40Hz). Two-stage IIR filter: low-pass then high-pass. Low-pass: y[n] = 2*y[n-1] - y[n-2] + x[n] - 2*x[n-6] + x[n-12] (cutoff ~11Hz). High-pass: y[n] = 32*x[n-16] - (y[n-1] + x[n] - x[n-32]) / 32. These integer-coefficient filters run efficiently on 8-bit AVR without float. Result: clean QRS complex isolated from noise.

  2. 2

    Pan-Tompkins Stage 2: Derivative + Square + Integrate

    Derivative accentuates QRS slope: y[n] = (1/8)*(-x[n-2] - 2*x[n-1] + 2*x[n+1] + x[n+2]). Squaring: y[n] = x[n]^2 — makes all values positive and amplifies large slopes. Moving window integration over 150ms (54 samples at 360Hz): y[n] = (1/W)*sum(x[n-W+1..n]). This produces a smooth envelope with a peak at each QRS complex, used to threshold R-peak detection robustly.

  3. 3

    Adaptive Threshold + R-Peak Detection

    Pan-Tompkins uses two adaptive thresholds tracking noise level and signal level: SPK = 0.875*SPK + 0.125*peak_val (signal); NPK = 0.875*NPK + 0.125*peak_val (noise); threshold1 = NPK + 0.25*(SPK-NPK). Peak detection: find local maximum above threshold1 after 200ms refractory period. Store R-peak location, compute RR interval: RR = (current_peak - last_peak) / 360.0 // seconds. HR = 60/mean(RR_last8). Missed beat search-back algorithm for weak beats.

  4. 4

    Rhythm Classification + Alert

    Maintain rolling buffer of last 8 RR intervals. Classification rules: Tachycardia: HR > 150; Bradycardia: HR < 50; Irregular: std(RR) > 0.1. SD calculation: float rr_mean = avg(rr_buf, 8); float sd = sqrt(avg_sq_diff); if(sd > 0.1) rhythm = IRREGULAR. On arrhythmia: tone buzzer (3× beeps), flash LED, scroll LCD message. Log to SD: file.printf("%lu,%.3f,%d,%s\n", millis(), rr_ms, hr, diagnosis).

💻 Core Code Snippet

// Arduino Mega — Pan-Tompkins R-peak detection
// Timer1 ISR fires at 360Hz to read AD8232 ADC

volatile int ecg_sample = 0;
volatile bool new_sample = false;
int bp_buf[12] = {0}; int sq_buf[54] = {0};
float rr_buf[8] = {0}; int rr_idx = 0;
long last_peak = 0; float SPK = 0, NPK = 0;

ISR(TIMER1_COMPA_vect) {
  ecg_sample = analogRead(A0); new_sample = true;
}

float pan_tompkins(int x) {
  // Simplified low-pass
  static int lp[6]={0}; float lp_out = 2*lp[0]-lp[1]+x-2*bp_buf[5]+bp_buf[11];
  memmove(lp+1,lp,5*sizeof(int)); lp[0]=lp_out/4;
  // Derivative (approximation)
  static int prev[2]={0}; float deriv = (x - prev[1]) / 2.0;
  prev[1]=prev[0]; prev[0]=x;
  // Square
  float sq = deriv * deriv;
  // Moving integration
  static float mwi = 0; mwi += sq/54.0 - sq_buf[53]/54.0;
  memmove(sq_buf+1,sq_buf,53*sizeof(int)); sq_buf[0]=(int)sq;
  return mwi;
}

void loop() {
  if (!new_sample) return; new_sample = false;
  float env = pan_tompkins(ecg_sample);
  static long sample_count = 0; sample_count++;
  float thresh = NPK + 0.25*(SPK-NPK);
  if (env > thresh && sample_count - last_peak > 72) { // >200ms refractory
    SPK = 0.875*SPK + 0.125*env;
    rr_buf[rr_idx%8] = (sample_count - last_peak)/360.0f;
    rr_idx++; last_peak = sample_count;
    float hr = 60.0 / avg(rr_buf, 8);
    Serial.printf("HR:%.0f SD:%.3f\n", hr, std_dev(rr_buf, 8));
  } else NPK = 0.875*NPK + 0.125*env;
}
