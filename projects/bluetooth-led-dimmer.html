🟢 Beginner Project

Bluetooth LED Dimmer

Send brightness values from a free phone serial app over HC-05 Bluetooth to smoothly dim an LED — your first wireless Arduino control project.

Home › Projects › Bluetooth LED Dimmer
BT

Phone → HC-05 UART → Arduino SoftwareSerial → analogWrite() PWM

▶ Watch on YouTube

📖 Introduction

The HC-05 Bluetooth module acts as a wireless UART bridge — it transmits whatever your phone sends, byte-by-byte, over a serial connection to the Arduino. Using SoftwareSerial on pins 10/11 frees the hardware serial port for debugging, while the received byte value (0–255) feeds directly into analogWrite() as a brightness level.

This project teaches the most fundamental wireless pattern in IoT: send a value, receive a value, do something proportional. Once you understand it, ESP8266 Wi-Fi, LoRa, and ZigBee all follow the same principle with different transport layers.

⚙️ How It Works

  1. 1
    Wire the HC-05 — HC-05 VCC → 5 V (module reg), GND → GND, TXD → D10 (Arduino RX), RXD → D11 via a voltage divider (10 kΩ + 20 kΩ) to step 5 V TX down to 3.3 V for the module. LED anode via 220 Ω → D9 (~), cathode → GND.
  2. 2
    Set up SoftwareSerial — #include <SoftwareSerial.h>. Declare SoftwareSerial bt(10, 11); (RX, TX). In setup(): bt.begin(9600); — HC-05 default baud rate. The hardware Serial.begin(9600); remains available for debug prints.
  3. 3
    Receive and act — In loop(): if (bt.available()) { int bright = bt.read(); analogWrite(9, bright); bt.print("Brightness: "); bt.println(bright); }. The phone app sends a single byte (0–255); the Arduino echoes the value back for confirmation.
  4. 4
    Pair and test — Power on HC-05 (LED flashes every 2 s = waiting to pair). On phone: pair to "HC-05" with pin "1234" or "0000". In the Serial Bluetooth Terminal app, type a number 0–255 and hit send. The LED changes brightness instantly. Create quick-send buttons in the app for 0, 64, 128, 192, 255.
