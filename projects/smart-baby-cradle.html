Smart Baby Cradle

A sound sensor detects infant crying and automatically starts rocking the cradle with an MG996R servo motor while a DFPlayer Mini plays soothing lullabies. Stops rocking after 10 minutes of quiet — saving parents a trip and a night's sleep.

📖 Introduction

New parents average 4–5 hours of sleep per night in the first year. An automated cradle that responds to infant crying can restore precious rest by handling the soothing reflex automatically. This project connects a sound sensor to an Arduino that runs a state machine managing rocking motion and lullaby playback — and automatically returns to standby once the baby has been quiet for 10 minutes.

The design is safe-first: the servo uses a crank-and-connecting-rod mechanism that physically limits the rocking angle to ±15°, and a photo-interrupter optionally confirms cradle motion (so a motor stall is detected immediately). Temperature and humidity sensors monitor the sleeping environment for caregiver peace of mind.

🧩 Components Required

  • 🎤
    KY-037 / KY-038 Sound Sensor — Adjustable threshold potentiometer; digital out for cry detection trigger
  • 🤖
    MG996R High-Torque Servo (or DC Motor + Crank) — 9 kg·cm torque; crank-rod mechanism converts rotation to swing motion
  • 🎵
    DFPlayer Mini MP3 Module — Plays 128 kbps MP3 lullabies from microSD card on a small 3 W speaker
  • 🌡️
    DHT11 or DHT22 Sensor — Monitors crib environment temperature and humidity; alerts if overheating
  • 🔧
    Arduino Uno / Nano — Primary logic controller; UART to DFPlayer, PWM to servo
  • 🖥️
    16×2 LCD + 3 Push Buttons — Manual override, song select, rocking speed adjust, sleep timer

⚙️ How It Works

  1. 1

    Cry Detection with Sound Sensor

    The KY-037 digital output triggers a hardware interrupt on Arduino pin 2 when sound level exceeds the threshold set by the on-board potentiometer. Calibrate the threshold to distinguish normal environmental noise from crying: tune so that a hand clap 50 cm away triggers it reliably. Count continuous triggers above threshold for 500 ms before treating it as a cry event (not a single noise spike). Software debounce prevents one trigger restarting the 10-minute quiet timer prematurely.

  2. 2

    Servo Rocking State Machine

    State IDLE: servo at 90° (neutral). State ROCKING: use sin(t) function to smoothly sweep servo between 75° and 105° (±15°) with a period of 2 s — producing a natural, pendulum-like motion. Angle = 90 + 15 × sin(2π × t / rockPeriod). Do NOT use stepwise motion — the smooth sinusoidal sweep is important for infant safety (no jerking). Speed configurable via button: 1.5 s (fast), 2 s (normal), 3 s (slow).

  3. 3

    DFPlayer MP3 Lullaby Control

    Communicate with DFPlayer Mini via SoftwareSerial (any two pins). On cry detection: myDFPlayer.play(1);. Implement shuffle: myDFPlayer.playMp3Folder(random(1, numSongs+1));. Set volume to 20 (out of 30) for safe hearing levels. On returning to IDLE: myDFPlayer.stop(); after a 30-second fade-out (reduce volume by 1 every 2 s). Store last-played track in EEPROM to avoid repeating same song immediately.

  4. 4

    Safety Monitoring

    Read DHT22 every 60 s. If crib temperature > 30°C: display warning on LCD and optionally send notification via ESP8266 add-on. Add a 5-minute auto-stop safety timer: if rocking >5 minutes without further cry trigger, slow and stop (baby may have fallen asleep). Manual override button starts/stops rocking regardless of sensor state. Photo-interrupter on rocker confirms mechanical motion — if encoder doesn't show motion when servo commanded, set alarm (servo stall or mechanical jam).

💻 Core Code Snippet

#include <Servo.h>
#include <DFRobotDFPlayerMini.h>
#include <SoftwareSerial.h>

Servo cradle;
SoftwareSerial dfSerial(10, 11);
DFRobotDFPlayerMini player;

bool rocking = false;
unsigned long cryTime = 0, quietStart = 0;
float t = 0;

void cryISR() {
  if(!rocking) { rocking = true; player.play(1); }
  cryTime = millis();   // reset quiet timer
}

void setup() {
  cradle.attach(9);
  dfSerial.begin(9600); player.begin(dfSerial);
  player.volume(20);
  attachInterrupt(digitalPinToInterrupt(2), cryISR, RISING);
}

void loop() {
  if(rocking) {
    float angle = 90 + 15 * sin(2 * PI * t / 2000.0);
    cradle.write((int)angle);
    t += 20;
    if(t > 2000) t = 0;
    // stop after 10 min of quiet
    if(millis() - cryTime > 600000) {
      rocking = false; player.stop(); cradle.write(90);
    }
  }
  delay(20);
}
