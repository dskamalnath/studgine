RF Spectrum Analyser

A $25 RTL2832U RTL-SDR USB dongle feeds raw 8-bit IQ samples at up to 3.2 Msps to a Raspberry Pi running GNU Radio or pyrtlsdr to visualise the 24–1766 MHz RF spectrum in real time — a powerful tool for RF engineering, SDR exploration, and signal identification.

📖 Introduction

Software Defined Radio (SDR) replaces analogue radio hardware with digital signal processing. The RTL2832U dongle (originally designed for DVB-T TV reception) exposes raw IQ samples via USB, enabling any frequency from 24 MHz to 1766 MHz to be received and processed in software. GNU Radio Companion provides a visual flow-graph IDE; pyrtlsdr enables direct Python control. This project builds a real-time spectrum display, FM radio demodulator, NOAA weather satellite decoder, and aircraft ADS-B receiver — all from one $25 dongle.

⚙️ How It Works

  1. 1

    RTL-SDR Setup + IQ Capture

    Install: sudo apt install rtl-sdr python3-rtlsdr gnuradio. Test: rtl_test -t. Python capture: from rtlsdr import RtlSdr; sdr = RtlSdr(). Set tuner: sdr.sample_rate = 2.4e6; sdr.center_freq = 98e6; sdr.gain = 'auto'. Capture IQ: samples = sdr.read_samples(256*1024) → complex64 numpy array. IQ = in-phase + quadrature components representing the analogue RF signal's amplitude and phase.

  2. 2

    FFT Spectrum Display

    Compute Power Spectral Density: psd = np.abs(np.fft.fftshift(np.fft.fft(samples))) ** 2. Convert to dBm: psd_db = 10*np.log10(psd + 1e-10). Generate frequency axis: freqs = np.fft.fftshift(np.fft.fftfreq(len(samples), 1/sdr.sample_rate)) + sdr.center_freq. Plot with matplotlib: real-time update with plt.ion(); plt.pause(0.01). For GNU Radio: use QT GUI Frequency Sink block for zero-code real-time waterfall + spectrum.

  3. 3

    FM Radio Demodulation

    WFM demodulation: low-pass filter IQ at ±75kHz bandwidth, downsample to 200kHz. Frequency discriminator: demod = np.angle(samples[1:] * np.conj(samples[:-1])). De-emphasis filter (75µs): scipy.signal.lfilter. Resample to 48kHz audio. Play with sounddevice or pipe to aplay. GNU Radio equivalent: FM Receiver flowgraph (~6 blocks). Tune to local FM station for instant audio proof-of-concept.

  4. 4

    Advanced Applications

    NOAA Weather Satellite: tune to 137.1/137.9 MHz, 56kHz bandwidth FM, decode APT audio to greyscale images with WXtoImg or apt-decoder. ADS-B aircraft: tune 1090MHz, run dump1090 — decodes aircraft transponder positions, display on map in browser. ADS-B on Pi 4 decodes 1000+ aircraft/hour. DAB Radio: GNU Radio + gr-dab plugin for digital radio. RTL-SDR is the entry point to the entire SDR ecosystem.

💻 Core Code Snippet

import numpy as np, matplotlib.pyplot as plt
from rtlsdr import RtlSdr

sdr = RtlSdr()
sdr.sample_rate = 2.4e6   # 2.4 Msps
sdr.center_freq = 98.0e6  # 98 MHz FM band centre
sdr.gain = 'auto'

plt.ion()
fig, ax = plt.subplots()
N = 4096

while True:
    samples = sdr.read_samples(N)
    # Apply Hann window to reduce spectral leakage
    window = np.hanning(N)
    windowed = samples * window
    
    # FFT → Power Spectral Density in dB
    fft_vals = np.fft.fftshift(np.fft.fft(windowed, N))
    psd = 10 * np.log10(np.abs(fft_vals)**2 + 1e-10)
    
    # Frequency axis in MHz
    freqs = np.fft.fftshift(
        np.fft.fftfreq(N, 1/sdr.sample_rate)) + sdr.center_freq
    
    ax.cla()
    ax.plot(freqs/1e6, psd, color='#aa44ff', linewidth=0.8)
    ax.set_xlabel('Frequency (MHz)')
    ax.set_ylabel('Power (dB)')
    ax.set_title(f'Spectrum @ {sdr.center_freq/1e6:.1f} MHz')
    plt.pause(0.05)

sdr.close()
