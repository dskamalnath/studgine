3D Structured-Light Scanner

Project Gray-code binary stripe patterns from a pico projector onto an object. A Pi Camera captures how the patterns deform over the surface. OpenCV triangulation converts those per-pixel phase maps into a dense 3D point cloud — exportable as an STL mesh for 3D printing — without any rotating parts or laser hazards.

📖 Introduction

Structured-light scanning is the same technology used in Apple Face ID and industrial quality-control systems. It works by projecting known patterns onto a surface and analysing how those patterns deform due to surface geometry. Gray code patterns are a sequence of binary-encoded stripe images where each pixel column gets a unique binary code across N images — this encodes absolute position without ambiguity even at discontinuities. Combined with sinusoidal phase-shifting for sub-pixel accuracy, the result is a dense, accurate 3D point cloud from just a projector + camera pair. Achievable resolution: 0.3mm at 30cm range.

🧩 Components Required

  • 💡
    Pico DLP Projector (AAXA P2 or similar) — HDMI input; 854×480 resolution; 50+ lumens; connect to Pi HDMI; project in dark room for best depth SNR
  • 📷
    Pi Camera v2 or HQ Camera — Fixed focus at scanning distance (30cm); global shutter preferred; triggered capture sync with projector via GPIO
  • 🧠
    Raspberry Pi 4 (4 GB) — Generates pattern images; controls projector via HDMI framebuffer; captures images; runs OpenCV triangulation
  • 📐
    Calibration Board + Rigid Rig — Printed A4 chessboard 9×6 corners; rigid frame to fix camera-projector baseline angle (15–20cm separation, 30° angle)

⚙️ How It Works

  1. 1

    Gray Code Pattern Generation

    Generate N=log2(W) binary stripe images where each image halves the stripe width. For 16 strip codes: 4 images (8, 4, 2, 1 pixel stripes). Plus 4-step phase-shifted sinusoids for sub-pixel precision. PIL/Pygame render patterns as HDMI framebuffer: for bit in range(N): img[y, x] = 255 if (x >> (N-1-bit)) & 1 else 0. Project each image, capture with Pi Camera: camera.capture(f"frame_{bit}.jpg"); time.sleep(0.2). Also capture uniformly-white and uniformly-black frames for normalisation.

  2. 2

    Gray Code Decoding to Column Index

    Per-pixel, decode the bit sequence across N binary images to get projector column index: for each pixel (py,px): binary = [frames[b][py,px] > threshold for b in range(N)]; col = gray_to_binary(binary). Gray-to-binary conversion: b[0]=g[0]; for i in range(1,N): b[i]=b[i-1]^g[i]. Add phase shift sub-pixel refinement: phi = arctan2(I3-I1, I0-I2); col_refined = col + phi/(2*pi). Each camera pixel now maps to a projector column.

  3. 3

    Stereo Triangulation to 3D Points

    Pre-calibrate camera (intrinsics K, distortion) + projector (treat as inverse camera with its own K). Compute fundamental matrix from chessboard calibration. Per-pixel correspondence: camera pixel (u_c, v_c) ↔ projector column u_p. Triangulation: x_world = cam_origin + t * ray_direction. Using OpenCV: pts3d = cv2.triangulatePoints(P1, P2, pts_camera.T, pts_projector.T). Homogenise: pts3d = pts3d[:3] / pts3d[3]. Filter by reprojection error <1px. Result: 50k–200k 3D points per scan.

  4. 4

    Point Cloud to Mesh Export

    Save point cloud as PLY: import open3d as o3d; pcd = o3d.geometry.PointCloud(); pcd.points = o3d.utility.Vector3dVector(pts3d.T); o3d.io.write_point_cloud("scan.ply", pcd). Visualise in Open3D viewer. Mesh reconstruction via Poisson surface: mesh, _ = o3d.geometry.TriangleMesh.create_from_point_cloud_poisson(pcd, depth=9). Export STL for 3D printing. Multiple scan angles: ICP registration to fuse front + side scans into complete 360° model.

💻 Core Code Snippet

import numpy as np, cv2
from picamera2 import Picamera2

camera = Picamera2(); camera.start()
N = 4  # Gray code bits → 16 column regions

def gen_gray_pattern(bit, width=854, height=480):
    img = np.zeros((height, width), dtype=np.uint8)
    for x in range(width):
        img[:, x] = 255 if (x >> (N-1-bit)) & 1 else 0
    return img

def gray_to_int(gray_bits):
    """Convert Gray code bits to integer."""
    n = 0;  mask = gray_bits[0]
    n |= mask
    for i in range(1, len(gray_bits)):
        mask ^= gray_bits[i]; n = (n << 1) | mask
    return n

frames = []
for bit in range(N):
    pattern = gen_gray_pattern(bit)
    # Display on HDMI projector (framebuffer), capture image
    # display_pattern(pattern)  # via pygame/HDMI
    img = camera.capture_array()
    frames.append(cv2.cvtColor(img, cv2.COLOR_RGB2GRAY))

# Decode per-pixel projector column
h, w = frames[0].shape
decoded = np.zeros((h, w), dtype=np.float32)
for y in range(h):
    for x in range(w):
        bits = [(frames[b][y, x] > 128) for b in range(N)]
        decoded[y, x] = gray_to_int(bits)

print(f"Decoded range: {decoded.min():.0f} – {decoded.max():.0f}")
