Smart Shopping Cart

Scan items with an RC522 RFID reader as they're added to the cart. A running total updates on an LCD and a HX711 load cell verifies weight — flagging mismatches before checkout.

📖 Introduction

Self-checkout kiosks in modern supermarkets use the same two principles this project teaches: RFID tag scanning for item identification and a weight scale as an anti-theft verification layer. By building a scaled version on Arduino, students learn how product databases, running totals, and sensor fusion work together in retail IoT systems.

Each RFID tag stores a UID that maps to a product name, price, and expected weight in an Arduino array (acting as a simple in-memory database). The HX711 continuously weighs the basket. When the Expected Weight (sum of all scanned items) doesn't match the Measured Weight within ±50 g, an alert sounds.

🧩 Components Required

  • 📡
    RC522 RFID Reader Module — SPI-based 13.56 MHz reader; comes with 2 RFID cards + 2 key fobs
  • ⚖️
    HX711 + 5 kg Load Cell — 24-bit ADC amplifier; measures cart weight for anti-theft verification
  • 🔧
    Arduino Uno / Mega — Mega preferred for multiple I2C devices + more program storage for item DB
  • 🖥️
    20×4 I2C LCD — Shows item name, unit price, cart total, and weight status on 4 rows
  • 🔔
    Buzzer + Red/Green LEDs — Beep on scan; green for weight match, red for mismatch
  • 🛒
    Miniature Cart / Basket + Platform — Load cell mounted under basket platform; RFID reader at entry point

⚙️ How It Works

  1. 1

    RFID Tag Scanning & Product Lookup

    Continuously poll MFRC522 with mfrc522.PICC_IsNewCardPresent(). On new card detected: read UID bytes and format as hex string. Search the product array for matching UID. If found: add item to cart, update running total, play short beep. If not found: display "Unknown Item" — can be used to verify the card isn't a fraudulent tag. Debounce: prevent same UID registering twice in 3 s (same item held too long at reader).

  2. 2

    HX711 Load Cell Weight Reading

    After each scan, wait 1 s for item to settle in basket, then read load cell: scale.get_units(10) averages 10 readings to reduce noise. Compare measured weight against expected weight (sum of registered item weights). Tolerance band ±50 g accounts for scale drift and sensor noise. Display status: "WEIGHT OK ✓" or "WEIGHT MISMATCH! Check cart". Recalibrate with known reference weight on startup: scale.set_scale(calibration_factor);

  3. 3

    Remove Item Feature

    A dedicated "Remove Mode" button toggles into removal state. Next RFID scan in remove mode subtracts item from cart and expected weight. LCD shows "REMOVED:" + item name. Subtotal updates immediately. This mirrors supermarket-style "remove from bag" functionality. Prevents negating items that were never added (check: don't allow removal if item not in cart list).

  4. 4

    Checkout & Receipt Print

    Checkout button finalises the session. Print receipt to serial monitor (line-by-line: item, qty, subtotal) — can redirect to a Bluetooth thermal printer for real receipts. Display final total on LCD for 10 s. Reset all cart arrays and weight expected value. Optionally send total to Blynk dashboard for remote checkout verification. Log transaction to SD card for store records.

💻 Core Code Snippet

#include <MFRC522.h>
#include <HX711.h>
#include <LiquidCrystal_I2C.h>

MFRC522 rfid(10, 9);
HX711 scale;
LiquidCrystal_I2C lcd(0x27, 20, 4);

struct Product { char uid[12]; char name[20]; float price; int weight_g; };
Product catalog[] = {
  {"A1B2C3D4", "Apple 500g",   45.00, 500},
  {"E5F6A7B8", "Milk Packet",  60.00, 485},
  {"C9D0E1F2", "Bread Loaf",   35.00, 400},
};

float cartTotal = 0; int expectedWeight = 0;

void loop() {
  if(!rfid.PICC_IsNewCardPresent()) return;
  rfid.PICC_ReadCardSerial();
  String uid = "";
  for(byte i=0;i<rfid.uid.size;i++) uid += String(rfid.uid.uidByte[i], HEX);
  uid.toUpperCase();

  for(auto& p : catalog) {
    if(uid == String(p.uid)) {
      cartTotal += p.price;
      expectedWeight += p.weight_g;
      lcd.print(p.name); lcd.print(" "); lcd.print(p.price);
      float measured = scale.get_units(10);
      bool ok = abs(measured - expectedWeight) < 50;
      lcd.print(ok ? " OK" : " MISMATCH!");
      break;
    }
  }
  rfid.PICC_HaltA();
}
