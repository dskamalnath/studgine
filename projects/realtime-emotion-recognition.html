Real-Time Emotion Recognition

A face-detection pipeline using MediaPipe Face Detection crops faces from a live Pi Camera frame, then feeds them into a MobileNetV2 emotion classifier trained on FER-2013 to display the dominant emotion — happy, sad, angry, surprised, neutral, or disgusted — overlaid on the live video stream at ~15fps on Raspberry Pi 4.

📖 Introduction

Facial emotion recognition has applications in human-computer interaction, mental health monitoring, and customer experience analysis. The FER-2013 dataset contains 35,887 greyscale 48×48 face images labelled with 7 emotions. Training a MobileNetV2 (transfer learning from ImageNet) on FER-2013 achieves ~65–70% test accuracy (comparable to human performance on the dataset, which has significant noise). The Raspberry Pi pipeline: MediaPipe Face Detection → crop + preprocess → TFLite inference → draw emotion label with colour-coded bounding box. Processes at ~15fps on Pi 4 without a Coral TPU.

🧩 Components Required

  • 📷
    Raspberry Pi Camera Module v3 — 1080p capable; picamera2; run at 640×480 for 25fps; autofocus v3 helps in variable lighting
  • 🧠
    Raspberry Pi 4 (4 GB) — TFLite MobileNetV2 INT8 (1.3MB); MediaPipe Face Detection (~6MB); ~15fps end-to-end pipeline
  • 🖥️
    HDMI Monitor — 1080p for live video overlay display; emotion label + bounding box + probability bar chart rendered with cv2.imshow
  • 💡
    Consistent Lighting — Softbox or LED panel at 45°; avoid backlit windows; proper lighting is critical for high model accuracy

⚙️ How It Works

  1. 1

    Face Detection with MediaPipe

    MediaPipe Face Detection (BlazeFace): runs at >100fps on mobile CPU, locates face bounding boxes. import mediapipe as mp; fd = mp.solutions.face_detection.FaceDetection(min_detection_confidence=0.6). Process: results = fd.process(rgb_frame). Extract bounding box: bbox = detection.location_data.relative_bounding_box. Convert to pixel coords: xmin=int(bbox.xmin*W), ymin=int(bbox.ymin*H), w=int(bbox.width*W), h=int(bbox.height*H). Crop face with 20px padding.

  2. 2

    MobileNetV2 Training on FER-2013

    Download FER-2013 from Kaggle (35k images, 7 classes). Training on PC: base = tf.keras.applications.MobileNetV2(include_top=False, input_shape=(48,48,3)); model.add(GlobalAvgPooling2D()); model.add(Dense(7, activation='softmax')). Fine-tune with ImageDataGenerator augmentation. Test accuracy: ~65%. Export: converter = tf.lite.TFLiteConverter.from_keras_model(model); converter.optimizations=[tf.lite.Optimize.DEFAULT] → INT8 quantised TFLite model.

  3. 3

    Real-Time Inference Pipeline

    For each face crop: face_rgb = cv2.resize(crop, (48,48)); face_norm = face_rgb / 255.0. TFLite inference: interp.set_tensor(in_d['index'], face_norm[np.newaxis,...].astype(np.float32)); interp.invoke(); probs = interp.get_tensor(out_d['index'])[0]. EMOTIONS=['Angry','Disgust','Fear','Happy','Sad','Surprise','Neutral']. Dominant: label = EMOTIONS[np.argmax(probs)]. Temporal smoothing: exponential moving average of probabilities over 5 frames to prevent flickering.

  4. 4

    Display + Applications

    Draw on frame: cv2.rectangle(frame, (xmin,ymin), (xmin+w,ymin+h), EMOTION_COLOURS[label], 2). Emotion emoji + label above box. Draw 7-bar probability chart in corner. Display: cv2.imshow('Emotion', frame). Log session: emotion timeline CSV + histogram. Applications: audience engagement analyser (lecture/presentation), mood journal app, adaptive music player (play happy music when user is sad), retail shelf reaction analysis.

💻 Core Code Snippet

import cv2, numpy as np, mediapipe as mp
import tflite_runtime.interpreter as tflite
from picamera2 import Picamera2

cam = Picamera2(); cam.start()
fd = mp.solutions.face_detection.FaceDetection(min_detection_confidence=0.6)
interp = tflite.Interpreter('emotion_mobilenet_int8.tflite')
interp.allocate_tensors()
in_d = interp.get_input_details(); out_d = interp.get_output_details()

EMOTIONS = ['Angry','Disgust','Fear','Happy','Sad','Surprise','Neutral']
EMO_EMOJI = {'Happy':'😊','Sad':'😢','Angry':'😠','Surprise':'😲','Neutral':'😐'}
COLORS = {'Happy':(0,220,255),'Sad':(255,80,0),'Angry':(0,0,255),'Neutral':(150,150,150)}

smooth_probs = np.ones(7)/7
ALPHA = 0.3

while True:
    frame = cam.capture_array()
    H, W = frame.shape[:2]
    rgb = cv2.cvtColor(frame, cv2.COLOR_BGR2RGB)
    results = fd.process(rgb)
    if results.detections:
        d = results.detections[0]
        b = d.location_data.relative_bounding_box
        x,y,w,h = int(b.xmin*W),int(b.ymin*H),int(b.width*W),int(b.height*H)
        face = frame[max(0,y-10):y+h+10, max(0,x-10):x+w+10]
        if face.size:
            inp = cv2.resize(face,(48,48)).astype(np.float32)[np.newaxis,...]/255.0
            interp.set_tensor(in_d[0]['index'], inp)
            interp.invoke()
            probs = interp.get_tensor(out_d[0]['index'])[0]
            smooth_probs = ALPHA*probs + (1-ALPHA)*smooth_probs
            label = EMOTIONS[np.argmax(smooth_probs)]
            color = COLORS.get(label,(255,255,255))
            cv2.rectangle(frame,(x,y),(x+w,y+h),color,2)
            cv2.putText(frame,label,(x,y-8),cv2.FONT_HERSHEY_SIMPLEX,0.7,color,2)
    cv2.imshow('Emotion Recognition',frame)
