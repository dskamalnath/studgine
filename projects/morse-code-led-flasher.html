Morse Code LED Flasher

📖 Introduction

Morse code uses short flashes (dots) and long flashes (dashes) to represent letters and numbers. The ITU standard defines timing ratios: a dot = 1 unit, a dash = 3 units, intra-character gap = 1 unit, inter-character gap = 3 units, inter-word gap = 7 units. Using a single base time unit (e.g. 200 ms), all other durations follow.

This project creates a full lookup table mapping A–Z and 0–9 to their Morse patterns. You then iterate a string, look up each character, and toggle an LED accordingly. The result is a transmitter that can flash any message you encode in the sketch.

⚙️ How It Works

  1. 1
    Build the lookup table — Create a String morseTable[] = {".-", "-...", ...} array of 36 entries (A–Z then 0–9). Index 0 = 'A', index 25 = 'Z', index 26 = '0', etc. Define const int DOT = 200; as your base unit in milliseconds.
  2. 2
    Flash a symbol — Write a flashSymbol(char s) function: if s == '.' → LED on for DOT, off for DOT; if s == '-' → LED on for 3*DOT, off for DOT.
  3. 3
    Flash a letter — Write flashLetter(char c): convert c to uppercase, find its index in the table, loop through each character in the Morse string calling flashSymbol(). After the letter, wait 3*DOT (inter-character gap).
  4. 4
    Flash a message — In loop(), iterate over your message string. For each character: call flashLetter(); if the character is a space, wait 7*DOT. After the full message, pause 2 seconds and repeat. Optionally add tone(8, 700) / noTone(8) in the flash functions for an audible signal.
