Smart Wi-Fi Power Strip

Control four AC outlets independently from any web browser using an ESP8266 and relay module — building skills in HTML web servers, URL routing, and safe relay wiring.

📖 Introduction

The Smart Wi-Fi Power Strip turns a standard extension board into a browser-controlled smart home device. The ESP8266 serves a mobile-friendly HTML dashboard over your home Wi-Fi. Clicking a button sends an HTTP GET request (/relay1/on) and the ESP8266 toggles the corresponding relay output — no app download required.

You'll learn how to build an embedded HTTP server, parse URL routes, generate dynamic HTML with relay state, and safely wire a relay module to 230/110 VAC power — one of the most demanded IoT skills in product development.

🧩 Components Required

  • 🎛️
    NodeMCU ESP8266 (v3) — Built-in Wi-Fi web server
  • 🔌
    4-Channel 5 V Relay Module — Opto-isolated, active LOW trigger
  • 🔌
    AC Extension Box / Power Strip — Modified to interrupt live wire per outlet
  • 🔴
    Status LEDs ×4 — Visual ON/OFF indicator per relay
  • 🔋
    5 V / 2 A Hi-Link AC-DC Module — Powers ESP8266 from mains safely
  • ⚠️
    Heat-shrink Tubing + Enclosure — Safe mains AC isolation — mandatory!

⚙️ How It Works

  1. 1

    Connect to Home Wi-Fi

    Use WiFi.begin(ssid, pass) and wait for IP assignment. Print the IP to Serial Monitor — use it as the browser address. Use MDNS.begin("powerstrip") so users can navigate to powerstrip.local instead of the raw IP.

  2. 2

    Handle Route Requests

    Register handlers: server.on("/relay1/on", [](){relay1=LOW; redirect();}) and server.on("/relay1/off", [](){relay1=HIGH; redirect();}). Use a redirect response (HTTP 303) back to "/" so the page reloads with updated state.

  3. 3

    Serve Dynamic HTML Dashboard

    The root handler builds an HTML string containing a toggle button per relay. If relayState[i] == LOW, the button shows "Turn OFF" in red; otherwise "Turn ON" in green. This dynamic HTML pattern avoids WebSockets for a simpler beginner-friendly design.

  4. 4

    ⚠️ Safe AC Wiring

    The relay interrupts only the LIVE (brown/black) wire — never the neutral. Use a commercially-rated relay module with opto-isolation. Enclose all exposed AC connections in heat-shrink. The Hi-Link module eliminates exposed transformer windings. Test at 12 V DC first, then AC last.

💻 Core Code Snippet

#include <ESP8266WiFi.h>
#include <ESP8266WebServer.h>

ESP8266WebServer server(80);
const int relayPins[4] = {D1, D2, D3, D4};
bool relayState[4] = {HIGH, HIGH, HIGH, HIGH}; // HIGH = relay OFF (active LOW)

void handleRoot() {
  String html = "<!DOCTYPE html><html><head>"
    "<meta name='viewport' content='width=device-width,initial-scale=1'>"
    "<style>body{font-family:sans-serif;text-align:center;background:#111;color:#fff}"
    "a.on{background:#e07070;padding:8px 20px;border-radius:8px;text-decoration:none;color:#fff}"
    "a.off{background:#70c070;padding:8px 20px;border-radius:8px;text-decoration:none;color:#fff}</style></head><body>"
    "<h2>Smart Power Strip</h2>";
  String names[4] = {"Fan","Lamp","Charger","Monitor"};
  for (int i = 0; i < 4; i++) {
    bool on = (relayState[i] == LOW);
    html += "<p>" + names[i] + ": <a href='/r" + i + "/" + (on?"off":"on") +
            "' class='" + (on?"on":"off") + "'>" + (on?"Turn OFF":"Turn ON") + "</a></p>";
  }
  html += "</body></html>";
  server.send(200, "text/html", html);
}

void setup() {
  for (int i = 0; i < 4; i++) { pinMode(relayPins[i], OUTPUT); digitalWrite(relayPins[i], HIGH); }
  WiFi.begin("YourSSID", "YourPass");
  while (WiFi.status() != WL_CONNECTED) delay(100);
  server.on("/", handleRoot);
  // Register relay toggle routes
  for (int i = 0; i < 4; i++) {
    int idx = i;
    server.on("/r" + String(i) + "/on",  [idx](){ relayState[idx]=LOW;  digitalWrite(relayPins[idx],LOW);  server.sendHeader("Location","/"); server.send(303); });
    server.on("/r" + String(i) + "/off", [idx](){ relayState[idx]=HIGH; digitalWrite(relayPins[idx],HIGH); server.sendHeader("Location","/"); server.send(303); });
  }
  server.begin();
}
