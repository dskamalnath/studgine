Binary Clock Display

Display the current time in binary using a 4×4 LED grid kept accurate by a DS3231 RTC module. A nerdy, eye-catching desk project and a great lesson in numeral systems.

Home › Projects › Binary Clock Display

DS3231 time → hours / minutes / seconds → binary LED columns

▶ Watch on YouTube

📖 Introduction

A binary clock shows time in base-2 instead of base-10. Each column of LEDs represents one decimal digit of the time. For HH:MM:SS, that's six columns: H₁ H₂ : M₁ M₂ : S₁ S₂. Each column uses up to 4 LEDs representing bit values 1, 2, 4, 8 (from bottom to top). So the seconds digit "5" would light the "1" and "4" rows (since 1+4=5).

The DS3231 is a temperature-compensated RTC accurate to ±2 ppm (about ±1 minute per year). It communicates via I2C, making it easy to integrate alongside your LED matrix without consuming many Arduino pins.

⚙️ How It Works

  1. 1
    Read time from DS3231 — Wire SDA → A4, SCL → A5, 3.3V → 3.3V, GND → GND. Include #include <RTClib.h>. In setup: RTC.begin(); if (!RTC.isrunning()) RTC.adjust(DateTime(F(__DATE__), F(__TIME__)));. In loop: DateTime now = RTC.now(); int h=now.hour(), m=now.minute(), s=now.second();
  2. 2
    Extract Binary-Coded Decimal (BCD) digits — Split each time unit into tens and units digits: int h1=h/10, h2=h%10, m1=m/10, m2=m%10, s1=s/10, s2=s%10;. Each value 0–9 maps to a 4-bit binary number. For example, 5 = 0101₂ → LEDs for bits 0 and 2 are ON.
  3. 3
    Drive LEDs with shift registers — Two chained 74HC595 give 16 outputs. Clock in a 16-bit value that encodes all 6 BCD columns (6×4=24 bits, but H1 only needs 2 LEDs ≤ 2, so some bits are unused). Use shiftOut(DATA_PIN, CLK_PIN, MSBFIRST, highByte(ledState)); then the low byte, then latch.
  4. 4
    Set time via Serial — Add a simple serial command parser: when "T20250101120000" is received, call RTC.adjust(DateTime(year, month, day, hour, min, sec));. Alternatively add 3 buttons for hour/minute increment — the DS3231 retains the set time indefinitely with its coin cell backup.
